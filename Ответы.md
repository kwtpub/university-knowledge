Теорема ролля - 
если функция:
- диффиринцируема на (a, b)
- Непрерына на отрезке [a, b]
- f(a) = f(b)
То есть такая точка, что 
f'(x) = 0 

Теорема лагранжа:
если функция: 
- диффиринцируема на отрезке (a, b)
- непрерына на отрезке [a, b]
То существует такая точка, что: 
f(b) - f(a) / b - a = f'(x0)
(  По теореме лагранжа легко доказать теорему ролля)

Теорема коши: 
Если существуют такие функции f(x) и g(x) которые: 
- непрерыны на отрезке [a,b]
- диффиринцируемы на отрезке (a, b)
- g'(x) != 0 
то существует такая точка f'(x0) и g'(x0) что
f(b) - f(a) / g(b) - g(a) = f'(x0) / g'(x0)

Формула коненых приращений: 
	f(b) - f(a) = f'(x0) * (b - a)
