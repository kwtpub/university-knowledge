---

## 1. Что такое «форма»

**Форма** — это функция, которая принимает один или несколько векторов и возвращает **число**.

| Название | Сколько векторов на входе | Пример |
|----------|--------------------------|--------|
| Линейная форма | 1 вектор | $f(\vec{x}) = 2x_1 + 3x_2$ |
| Билинейная форма | 2 вектора | $f(\vec{x}, \vec{y}) = x_1 y_1 + x_2 y_2$ (скалярное произведение) |
| Квадратичная форма | 1 вектор | $f(\vec{x}) = x_1^2 + 2x_1 x_2 + x_2^2$ |

**Словами:** форма — это «машина», на вход подаём вектор(ы), на выходе получаем число. Название говорит, сколько векторов и какая зависимость.

---

## 2. Определение линейной формы

**Определение.** Линейная форма (линейный функционал) — это линейное отображение из пространства $V$ в числа $\mathbb{R}$:

$$f: V \to \mathbb{R}$$

**Словами:** линейная форма — это форма от **одного** вектора, которая линейно зависит от его координат. Подаём вектор — получаем число. При этом она подчиняется двум правилам:

1. **Аддитивность:** $f(\vec{x} + \vec{y}) = f(\vec{x}) + f(\vec{y})$
2. **Однородность:** $f(\lambda \vec{x}) = \lambda \cdot f(\vec{x})$

**Словами проще:** линейная форма — это «взвешенная сумма» координат вектора. Подаём вектор на вход — получаем одно число на выходе.

---

## 2. Примеры линейных форм

### Пример 1. Простая взвешенная сумма

$$f(x, y) = 2x + 3y$$

Вектор $(1, 4)$ → $f(1, 4) = 2 \cdot 1 + 3 \cdot 4 = 14$

Вектор $(0, 0)$ → $f(0, 0) = 0$ (линейная форма всегда даёт 0 на нулевом векторе)

Проверим линейность:
- $f((1,4) + (2,1)) = f(3, 5) = 6 + 15 = 21$
- $f(1,4) + f(2,1) = 14 + 7 = 21$ ✓

### Пример 2. Проекция на ось

$$f(x, y, z) = x$$

Берём только первую координату — это линейная форма. Возвращает проекцию вектора на ось $Ox$.

### Пример 3. Скалярное произведение с фиксированным вектором

Зафиксируем вектор $\vec{a} = (2, -1, 3)$. Тогда:

$$f(\vec{x}) = (\vec{a}, \vec{x}) = 2x_1 - x_2 + 3x_3$$

Скалярное произведение с фиксированным вектором — всегда линейная форма.

### Пример 4. НЕ линейная форма

$$g(x, y) = x^2 + y$$

Проверим: $g(2, 0) = 4$, $g(1, 0) = 1$, $g(1+1, 0) = g(2, 0) = 4 \neq g(1,0) + g(1,0) = 2$ ✗

Квадрат нарушает линейность.

## 3. Координатное представление

**Теорема.** В базисе $e_1, e_2, \ldots, e_n$ любая линейная форма записывается как:

$$f(\vec{x}) = a_1 x_1 + a_2 x_2 + \ldots + a_n x_n$$

где $x_1, x_2, \ldots, x_n$ — координаты вектора $\vec{x}$ в этом базисе, а $a_i = f(e_i)$ — значения формы на базисных векторах.

**В матричной записи:**

$$f(\vec{x}) = a^T x = \begin{pmatrix} a_1 & a_2 & \ldots & a_n \end{pmatrix} \begin{pmatrix} x_1 \\ x_2 \\ \vdots \\ x_n \end{pmatrix}$$

**Словами:** линейная форма полностью определяется своими коэффициентами $(a_1, a_2, \ldots, a_n)$. Зная их, мы можем вычислить значение формы на любом векторе — просто скалярное произведение.

### Откуда берутся коэффициенты

Коэффициент $a_i$ — это значение формы на $i$-м базисном векторе:

$$a_i = f(e_i)$$

**Числовой пример.** Пусть $f(x, y, z) = 2x - 3y + z$.

Стандартный базис: $e_1 = (1,0,0)$, $e_2 = (0,1,0)$, $e_3 = (0,0,1)$.

- $a_1 = f(e_1) = f(1,0,0) = 2$
- $a_2 = f(e_2) = f(0,1,0) = -3$
- $a_3 = f(e_3) = f(0,0,1) = 1$

Коэффициенты: $(2, -3, 1)$ — они же стоят при $x, y, z$.

## 4. Преобразование коэффициентов при смене базиса

**Задача:** у нас есть линейная форма с коэффициентами $a = (a_1, \ldots, a_n)$ в старом базисе. Переходим в новый базис с матрицей перехода $C$. Какие будут коэффициенты $a'$ в новом базисе?

**Формула:**

$$a' = C^T a$$

**Словами:** коэффициенты линейной формы преобразуются через **транспонированную** матрицу перехода. Не через $C$ и не через $C^{-1}$, а именно через $C^T$.

### Почему так

Значение формы не зависит от базиса (число остаётся тем же):

$$f(\vec{x}) = a^T x = (a')^T x'$$

Координаты вектора преобразуются: $x = C x'$ (старые через новые). Подставляем:

$$a^T (C x') = (a^T C) x' = (C^T a)^T x'$$

Значит $(a')^T = (C^T a)^T$, то есть $a' = C^T a$.

### Числовой пример

Форма $f(x, y) = 3x + y$ в стандартном базисе. Коэффициенты: $a = \begin{pmatrix} 3 \\ 1 \end{pmatrix}$.

Матрица перехода к новому базису $e_1' = (1, 1)$, $e_2' = (1, -1)$:

$$C = \begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}$$

Новые коэффициенты:

$$a' = C^T a = \begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix} \begin{pmatrix} 3 \\ 1 \end{pmatrix} = \begin{pmatrix} 3 + 1 \\ 3 - 1 \end{pmatrix} = \begin{pmatrix} 4 \\ 2 \end{pmatrix}$$

В новом базисе: $f = 4x_1' + 2x_2'$.

**Проверка.** Вектор $\vec{v} = (2, 1)$ в старом базисе: $f = 3 \cdot 2 + 1 \cdot 1 = 7$.

Найдём координаты $\vec{v}$ в новом базисе: $(2, 1) = x_1'(1,1) + x_2'(1,-1)$, решаем систему → $x_1' = 3/2$, $x_2' = 1/2$.

$f = 4 \cdot 3/2 + 2 \cdot 1/2 = 6 + 1 = 7$ ✓ Значение то же.

## Итог

| | Формула |
|---|---------|
| Определение | $f: V \to \mathbb{R}$, линейное отображение |
| Координатная запись | $f(\vec{x}) = a_1 x_1 + a_2 x_2 + \ldots + a_n x_n$ |
| Матричная запись | $f(\vec{x}) = a^T x$ |
| Коэффициенты | $a_i = f(e_i)$ |
| Смена базиса | $a' = C^T a$ |
