# Билет 48. Классификация поверхностей второго порядка. Эллипсоид, одно- и двуполостный гиперболоиды, коническая поверхность.

---

## 1. Классификация поверхностей второго порядка

Все поверхности второго порядка (уравнение степени 2 от $x, y, z$) после приведения к каноническому виду дают один из стандартных типов.

**Словами:** любое уравнение вида $Ax^2 + By^2 + Cz^2 + Dxy + \ldots = 0$ можно поворотом и сдвигом привести к простому виду. По этому виду сразу понятно, что за поверхность.

### Как определить тип

Смотрим на **знаки** при квадратах и **правую часть**:

| Знаки при $x^2, y^2, z^2$ | Правая часть | Тип |
|---------------------------|-------------|-----|
| $+, +, +$ | $= 1$ | Эллипсоид |
| $+, +, -$ | $= 1$ | Однополостный гиперболоид |
| $+, +, -$ | $= -1$ (или $-, -, +$ и $= 1$) | Двуполостный гиперболоид |
| $+, +, -$ | $= 0$ | Конус |
| $+, +$ и линейный $z$ | — | Эллиптический параболоид |
| $+, -$ и линейный $z$ | — | Гиперболический параболоид |

---

## 2. Эллипсоид

**Каноническое уравнение:**

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1$$

**Словами:** «мяч» или «яйцо». Замкнутая поверхность, все точки внутри ограниченной области. Три полуоси $a, b, c$ определяют размеры по каждой оси.

### Элементы

| Элемент | Значение |
|---------|----------|
| Полуоси | $a$ (по $x$), $b$ (по $y$), $c$ (по $z$) |
| Вершины | $(\pm a, 0, 0)$, $(0, \pm b, 0)$, $(0, 0, \pm c)$ |
| Центр | $(0, 0, 0)$ |

### Сечения

Сечение плоскостью $z = h$ (горизонтальный «срез»):

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 - \frac{h^2}{c^2}$$

- При $|h| < c$ — **эллипс** (сечение непустое)
- При $|h| = c$ — **точка** (вершина)
- При $|h| > c$ — **пустое множество** (срез выше/ниже эллипсоида)

**Словами:** режем эллипсоид горизонтально — получаем эллипсы. Чем ближе к вершине, тем меньше эллипс. Выше/ниже вершин — пусто.

### Числовой пример

$$\frac{x^2}{9} + \frac{y^2}{4} + \frac{z^2}{1} = 1$$

$a = 3$, $b = 2$, $c = 1$. Сплюснутый эллипсоид (как дыня).

Сечение $z = 0$: $\frac{x^2}{9} + \frac{y^2}{4} = 1$ — эллипс, самый большой.

Сечение $z = 0.5$: $\frac{x^2}{9} + \frac{y^2}{4} = 1 - 0.25 = 0.75$ — эллипс поменьше.

Точка $(3, 0, 0)$: $\frac{9}{9} + 0 + 0 = 1$ ✓ — лежит на поверхности.

### Частные случаи

- $a = b = c$ → **сфера** $x^2 + y^2 + z^2 = a^2$
- $a = b \neq c$ → **эллипсоид вращения** (сечения по $z$ — окружности)

## 3. Однополостный гиперболоид

**Каноническое уравнение:**

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$$

**Словами:** «ваза» или «талия». Поверхность сужается к центру и расширяется вверх и вниз. Не замкнутая — уходит в бесконечность.

### Элементы

| Элемент | Значение |
|---------|----------|
| Полуоси | $a$ (по $x$), $b$ (по $y$), $c$ (по $z$) |
| Центр | $(0, 0, 0)$ |
| «Горло» | При $z = 0$ — самое узкое место (эллипс $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$) |

### Сечения

Сечение $z = h$:

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 + \frac{h^2}{c^2}$$

- Правая часть **всегда > 0** → сечение всегда **эллипс**
- При $h = 0$ — самый маленький эллипс (горло)
- Чем больше $|h|$, тем больше эллипс

**Словами:** режем горизонтально — всегда получаем эллипс. Посередине самый узкий, вверх/вниз — расширяется. Пустых сечений нет.

### Числовой пример

$$\frac{x^2}{4} + \frac{y^2}{4} - \frac{z^2}{9} = 1$$

$a = b = 2$, $c = 3$. Сечения — **окружности** (потому что $a = b$).

Горло ($z = 0$): $x^2 + y^2 = 4$ — окружность радиуса 2.

Сечение $z = 3$: $\frac{x^2}{4} + \frac{y^2}{4} = 1 + 1 = 2$ → $x^2 + y^2 = 8$ — окружность радиуса $2\sqrt{2} \approx 2.83$.

### Линейчатая поверхность

Однополостный гиперболоид — **линейчатая поверхность**: через каждую его точку проходят **две прямые**, целиком лежащие на поверхности. Это свойство используется в архитектуре (Шуховская башня, градирни).

---

## 4. Двуполостный гиперболоид

**Каноническое уравнение:**

$$\frac{z^2}{c^2} - \frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$$

Или эквивалентно: $\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = -1$

**Словами:** «две чаши» — верхняя и нижняя, разделённые пустым пространством. В отличие от однополостного, здесь посередине ничего нет.

### Сечения

Сечение $z = h$:

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = \frac{h^2}{c^2} - 1$$

- При $|h| > c$ — **эллипс** (правая часть > 0)
- При $|h| = c$ — **точка** (вершина полости)
- При $|h| < c$ — **пустое множество** (между чашами)

**Словами:** между $z = -c$ и $z = c$ — пустота. Поверхность существует только при $|z| \geq c$.

### Числовой пример

$$\frac{z^2}{4} - \frac{x^2}{9} - \frac{y^2}{9} = 1$$

$a = b = 3$, $c = 2$.

Вершины полостей: $(0, 0, 2)$ и $(0, 0, -2)$.

Сечение $z = 4$: $\frac{x^2}{9} + \frac{y^2}{9} = \frac{16}{4} - 1 = 3$ → $x^2 + y^2 = 27$ — окружность радиуса $3\sqrt{3}$.

Сечение $z = 0$: $\frac{x^2}{9} + \frac{y^2}{9} = -1$ — **пусто** (отрицательная правая часть).

## 5. Коническая поверхность (конус)

**Каноническое уравнение:**

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 0$$

**Словами:** «воронка» с вершиной в начале координат. Если в уравнении однополостного гиперболоида правую часть заменить на $0$ — получим конус. Конус — это **граница** между однополостным и двуполостным гиперболоидами.

### Сечения

Сечение $z = h$:

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = \frac{h^2}{c^2}$$

- При $h \neq 0$ — **эллипс**, растущий с удалением от вершины
- При $h = 0$ — **точка** (вершина конуса)

**Словами:** режем горизонтально — получаем эллипсы. Чем дальше от вершины, тем больше. В самой вершине — точка.

### Числовой пример

$$\frac{x^2}{4} + \frac{y^2}{4} - \frac{z^2}{9} = 0$$

$a = b = 2$, $c = 3$. Сечения — окружности.

Сечение $z = 3$: $\frac{x^2}{4} + \frac{y^2}{4} = 1$ → $x^2 + y^2 = 4$ — окружность радиуса 2.

Сечение $z = 6$: $\frac{x^2}{4} + \frac{y^2}{4} = 4$ → $x^2 + y^2 = 16$ — окружность радиуса 4.

### Частный случай

Если $a = b$ — **круговой конус** (сечения — окружности, а не эллипсы).
