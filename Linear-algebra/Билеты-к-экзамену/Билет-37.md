# Билет 37. Уравнение прямой в пространстве. Параметрическая форма. Прямая как пересечение двух плоскостей.

---

## 1. Что нужно, чтобы задать прямую в пространстве

Прямая в пространстве определяется:
- **Точкой** $M_0(x_0, y_0, z_0)$, через которую она проходит
- **Направляющим вектором** $\vec{s} = (l, m, n)$, который показывает, куда она направлена

**Словами:** знаем одну точку на прямой и направление — этого достаточно.

---

## 2. Каноническое уравнение прямой

$$\frac{x - x_0}{l} = \frac{y - y_0}{m} = \frac{z - z_0}{n}$$

где:
- $(x_0, y_0, z_0)$ — известная точка на прямой
- $(l, m, n)$ — направляющий вектор

**Словами:** берём произвольную точку $(x, y, z)$. Вектор от $M_0$ до неё: $(x - x_0,\; y - y_0,\; z - z_0)$. Этот вектор должен быть **коллинеарен** направляющему вектору $(l, m, n)$. Коллинеарность значит пропорциональность координат — отсюда три равные дроби.

### Числовой пример

Точка $M_0(1, 2, 3)$, направляющий вектор $\vec{s} = (2, -1, 4)$:

$$\frac{x - 1}{2} = \frac{y - 2}{-1} = \frac{z - 3}{4}$$

Проверка: точка $(3, 1, 7)$ — подставляем:

$$\frac{3 - 1}{2} = \frac{1 - 2}{-1} = \frac{7 - 3}{4} \;\Rightarrow\; \frac{2}{2} = \frac{-1}{-1} = \frac{4}{4} \;\Rightarrow\; 1 = 1 = 1 \;\checkmark$$

### Если одна из координат направляющего вектора = 0

Например, $\vec{s} = (2, 0, 3)$, точка $(1, 5, -1)$:

$$\frac{x - 1}{2} = \frac{z + 1}{3}, \quad y = 5$$

**Словами:** если $m = 0$, значит прямая «не двигается» по $y$ — координата $y$ постоянна. Записываем её отдельно.

---

## 3. Параметрическое уравнение прямой

$$\begin{cases} x = x_0 + l \cdot t \\ y = y_0 + m \cdot t \\ z = z_0 + n \cdot t \end{cases}, \quad t \in \mathbb{R}$$

где $t$ — параметр (любое вещественное число).

**Словами:** стоим в точке $M_0$, идём в направлении $\vec{s}$. Параметр $t$ — «сколько шагов» мы прошли:
- $t = 0$ → стоим в $M_0$
- $t = 1$ → прошли один вектор $\vec{s}$ вперёд
- $t = -1$ → прошли один вектор назад
- $t = 0.5$ → прошли полвектора

### Числовой пример

Точка $M_0(1, 2, 3)$, вектор $\vec{s} = (2, -1, 4)$:

$$\begin{cases} x = 1 + 2t \\ y = 2 - t \\ z = 3 + 4t \end{cases}$$

| $t$ | $x$ | $y$ | $z$ | Точка |
|-----|-----|-----|-----|-------|
| $0$ | $1$ | $2$ | $3$ | $M_0$ (начальная) |
| $1$ | $3$ | $1$ | $7$ | Прошли один шаг |
| $-1$ | $-1$ | $3$ | $-1$ | Один шаг назад |
| $0.5$ | $2$ | $1.5$ | $5$ | Полшага |

### Связь с каноническим уравнением

Каноническое уравнение — это то же самое, только без параметра. Обозначим каждую дробь за $t$:

$$\frac{x - x_0}{l} = \frac{y - y_0}{m} = \frac{z - z_0}{n} = t$$

Из каждой дроби выражаем координату:
- $x - x_0 = lt$ → $x = x_0 + lt$
- $y - y_0 = mt$ → $y = y_0 + mt$
- $z - z_0 = nt$ → $z = z_0 + nt$

Получили параметрическую форму.

## 4. Прямая как пересечение двух плоскостей

Прямую можно задать **системой двух уравнений плоскостей**:

$$\begin{cases} A_1 x + B_1 y + C_1 z + D_1 = 0 \\ A_2 x + B_2 y + C_2 z + D_2 = 0 \end{cases}$$

**Словами:** каждое уравнение задаёт плоскость. Две непараллельные плоскости пересекаются по прямой. Точки, лежащие одновременно на обеих плоскостях — это и есть прямая.

### Числовой пример

$$\begin{cases} x + y + z - 3 = 0 \\ x - y + z - 1 = 0 \end{cases}$$

Найдём точку на прямой: пусть $z = 0$:

$$\begin{cases} x + y = 3 \\ x - y = 1 \end{cases}$$

Складываем: $2x = 4$ → $x = 2$, $y = 1$. Точка: $(2, 1, 0)$.

### Как найти направляющий вектор

Направляющий вектор прямой пересечения **перпендикулярен обеим нормалям** → это их **векторное произведение**:

$$\vec{s} = \vec{n_1} \times \vec{n_2}$$

В нашем примере:
- $\vec{n_1} = (1, 1, 1)$
- $\vec{n_2} = (1, -1, 1)$

$$\vec{s} = \vec{n_1} \times \vec{n_2} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 1 & 1 & 1 \\ 1 & -1 & 1 \end{vmatrix}$$

$$= \vec{i}(1 \cdot 1 - 1 \cdot (-1)) - \vec{j}(1 \cdot 1 - 1 \cdot 1) + \vec{k}(1 \cdot (-1) - 1 \cdot 1)$$

$$= \vec{i}(2) - \vec{j}(0) + \vec{k}(-2) = (2, 0, -2)$$

Теперь запишем каноническое уравнение (точка $(2, 1, 0)$, вектор $(2, 0, -2)$):

$$\frac{x - 2}{2} = \frac{z}{-2}, \quad y = 1$$

## 5. Три формы — одна прямая

| Форма | Запись | Что нужно знать |
|-------|--------|-----------------|
| Каноническая | $\dfrac{x - x_0}{l} = \dfrac{y - y_0}{m} = \dfrac{z - z_0}{n}$ | Точка + направляющий вектор |
| Параметрическая | $x = x_0 + lt$, $y = y_0 + mt$, $z = z_0 + nt$ | Точка + направляющий вектор |
| Пересечение плоскостей | Система из двух уравнений плоскостей | Два уравнения плоскостей |

### Как переходить между формами

- **Каноническая → параметрическая:** обозначить дроби за $t$, выразить $x, y, z$
- **Параметрическая → каноническая:** выразить $t$ из каждого уравнения, приравнять
- **Пересечение плоскостей → каноническая:** найти точку (подставить $z = 0$ или другое) + направляющий вектор ($\vec{n_1} \times \vec{n_2}$)
