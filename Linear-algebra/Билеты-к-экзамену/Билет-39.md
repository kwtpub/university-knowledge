## 1. Определение эллипса

**Определение.** Эллипс — это множество всех точек на плоскости, для которых **сумма расстояний** до двух фиксированных точек (фокусов) постоянна и равна $2a$.

$$r_1 + r_2 = 2a$$

где $r_1$ и $r_2$ — расстояния от точки до фокусов $F_1$ и $F_2$.

**Словами:** вбиваем два гвоздя (фокусы), натягиваем между ними нитку длиной $2a$. Берём карандаш, натягиваем нитку и рисуем — получится эллипс. Сумма расстояний от карандаша до двух гвоздей всегда = длина нитки.

**Частный случай:** если фокусы совпадают ($c = 0$), получается **окружность** — эллипс с равными полуосями ($a = b$).

---

## 2. Элементы эллипса

| Элемент | Обозначение | Что это |
|---------|-------------|---------|
| Большая полуось | $a$ | Половина длины эллипса по горизонтали (самая длинная) |
| Малая полуось | $b$ | Половина длины эллипса по вертикали (самая короткая) |
| Фокусное расстояние | $c$ | Расстояние от центра до фокуса |
| Фокусы | $F_1(-c, 0)$, $F_2(c, 0)$ | Две «особые» точки внутри эллипса |
| Центр | $O(0, 0)$ | Середина между фокусами |
| Вершины | $(\pm a, 0)$, $(0, \pm b)$ | 4 точки, где эллипс пересекает оси |

**Связь между $a$, $b$, $c$:**

$$c^2 = a^2 - b^2, \quad a \geq b > 0$$

**Словами:** $a$ — самая большая, $b$ — поменьше, $c$ — находится из них. Фокусы всегда внутри эллипса, ближе к центру, чем вершины.

**Числовой пример:** $a = 5$, $b = 3$ → $c = \sqrt{25 - 9} = \sqrt{16} = 4$.

Фокусы: $F_1(-4, 0)$ и $F_2(4, 0)$.

## 3. Каноническое уравнение эллипса

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1, \quad a \geq b > 0$$

**Словами:** подставляем координаты точки — если получается 1, точка лежит на эллипсе. Если меньше 1 — внутри, больше 1 — снаружи.

### Числовой пример

Эллипс с $a = 5$, $b = 3$:

$$\frac{x^2}{25} + \frac{y^2}{9} = 1$$

Проверим вершины:
- $(5, 0)$: $\frac{25}{25} + \frac{0}{9} = 1$ ✓
- $(0, 3)$: $\frac{0}{25} + \frac{9}{9} = 1$ ✓
- $(3, 2.4)$: $\frac{9}{25} + \frac{5.76}{9} = 0.36 + 0.64 = 1$ ✓

Точка $(1, 1)$: $\frac{1}{25} + \frac{1}{9} = 0.04 + 0.11 = 0.15 < 1$ — **внутри** эллипса.

## 4. Эксцентриситет

**Определение:**

$$e = \frac{c}{a}, \quad 0 \leq e < 1$$

**Словами:** эксцентриситет показывает, насколько эллипс «вытянут»:

| $e$ | Форма | Пояснение |
|-----|-------|-----------|
| $e = 0$ | Окружность | $c = 0$, фокусы совпадают, $a = b$ |
| $e$ близок к $0$ | Почти круглый | Фокусы рядом с центром |
| $e$ близок к $1$ | Сильно вытянутый | Фокусы далеко от центра, почти у вершин |

**Числовой пример:** $a = 5$, $b = 3$, $c = 4$:

$$e = \frac{4}{5} = 0.8$$

Эллипс довольно вытянутый.

Другой пример: $a = 5$, $b = 4$, $c = \sqrt{25 - 16} = 3$:

$$e = \frac{3}{5} = 0.6$$

Менее вытянутый — ближе к окружности.

## 5. Директрисы

**Определение.** Директрисы эллипса — две вертикальные прямые:

$$x = \pm \frac{a}{e} = \pm \frac{a^2}{c}$$

Директрисы расположены **за пределами** эллипса (дальше от центра, чем вершины), потому что $\frac{a}{e} > a$ при $e < 1$.

**Фокально-директрисное свойство:** для любой точки эллипса отношение расстояния до фокуса к расстоянию до соответствующей директрисы равно эксцентриситету:

$$\frac{r}{d} = e$$

где $r$ — расстояние до фокуса, $d$ — расстояние до директрисы.

**Словами:** эллипс можно определить не только через два фокуса, но и через фокус + директрису. Точка лежит на эллипсе, если расстояние до фокуса в $e$ раз меньше, чем расстояние до директрисы.

### Числовой пример

$a = 5$, $c = 4$, $e = 0.8$:

$$x = \pm \frac{a^2}{c} = \pm \frac{25}{4} = \pm 6.25$$

Директрисы: $x = -6.25$ и $x = 6.25$.

Вершина $(5, 0)$:
- Расстояние до фокуса $F_2(4, 0)$: $r = 5 - 4 = 1$
- Расстояние до директрисы $x = 6.25$: $d = 6.25 - 5 = 1.25$
- Отношение: $\frac{r}{d} = \frac{1}{1.25} = 0.8 = e$ ✓

---

## 6. Вывод уравнения эллипса (для понимания)

Фокусы $F_1(-c, 0)$ и $F_2(c, 0)$. Точка $M(x, y)$ на эллипсе: $r_1 + r_2 = 2a$.

Расстояния:
- $r_1 = \sqrt{(x + c)^2 + y^2}$
- $r_2 = \sqrt{(x - c)^2 + y^2}$

После алгебраических преобразований (возведение в квадрат, раскрытие, упрощение с учётом $b^2 = a^2 - c^2$) получаем:

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$$

## Итог — что запомнить

| | Формула | Значение |
|---|---------|----------|
| Уравнение | $\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1$ | $a \geq b > 0$ |
| Связь | $c^2 = a^2 - b^2$ | $c$ — расстояние до фокуса |
| Фокусы | $(\pm c, 0)$ | Внутри эллипса |
| Вершины | $(\pm a, 0)$, $(0, \pm b)$ | На осях |
| Эксцентриситет | $e = c / a$ | $0 \leq e < 1$ |
| Директрисы | $x = \pm a^2 / c$ | За пределами эллипса |
| Фокальное свойство | $r_1 + r_2 = 2a$ | Сумма расстояний до фокусов |
| Директрисное свойство | $r / d = e$ | Отношение к директрисе |

## Сравнение: эллипс vs гипербола vs парабола

| | Эллипс | Гипербола | Парабола |
|---|--------|-----------|----------|
| Уравнение | $\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1$ | $\dfrac{x^2}{a^2} - \dfrac{y^2}{b^2} = 1$ | $y^2 = 2px$ |
| Отличие | «$+$» между дробями | «$-$» между дробями | Квадрат только у одной переменной |
| Форма | Замкнутый овал | Две ветви | Одна ветвь, разомкнутая |
| Связь $a, b, c$ | $c^2 = a^2 - b^2$ | $c^2 = a^2 + b^2$ | Только параметр $p$ |
| Эксцентриситет | $0 \leq e < 1$ | $e > 1$ | $e = 1$ |
| Фокальное свойство | $r_1 + r_2 = 2a$ | $\|r_1 - r_2\| = 2a$ | $r = d$ |
