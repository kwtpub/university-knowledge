# Билет 17. Определение линейного отображения и преобразования. Примеры линейных преобразований: тождественное преобразование, сдвиг, растяжение и поворот плоскости. Преобразование подобия.

## Определения

**Линейное отображение (исчерпывающее определение)**:  
пусть `V` и `W` — векторные пространства над одним и тем же полем `F`
(обычно `R` или `C`). Отображение `f: V → W` называется линейным, если для
любых `x, y ∈ V` и любых `α, β ∈ F` выполняется

`f(αx + βy) = αf(x) + βf(y)`.

Эквивалентная запись (через два свойства):
- аддитивность: `f(x + y) = f(x) + f(y)`;
- однородность: `f(αx) = αf(x)`.


То есть линейное отображение полностью сохраняет линейную структуру:
суммы, умножение на скаляр и любые линейные связи между векторами.

Немедленные следствия:
- `f(0) = 0`;
- `f(-x) = -f(x)`;


**Смысл записи `f: V -> W` и связь со свойствами**:
- `f: V -> W` означает, что каждому вектору `x` из `V` ставится в соответствие
  ровно один вектор `f(x)` из `W`.
- Само по себе это просто отображение (функция между пространствами), без
  дополнительных гарантий.
- Свойства линейности (`f(x+y)=f(x)+f(y)`, `f(αx)=αf(x)`) задают класс
  «хороших» отображений, которые сохраняют структуру пространства.
- Поэтому линейное отображение сохраняет суммы, умножение на скаляр, линейные
  связи и может быть задано матрицей после выбора базисов.

**Линейное преобразование** — это отображение `f: V -> V`, которое сохраняет
сумму векторов и умножение на скаляр:
`f(x + y) = f(x) + f(y)`, `f(αx) = αf(x)`.


Тождественное преобразование: каждый вектор остается тем же
  самым. Пример: (2, -1) переходит в (2, -1).

  Сдвиг (сдвиговый срез): вектор «съезжает» вдоль одной оси
  пропорционально другой координате. Пример: при правиле x' =
  x + 2y, y' = y точка (1, 3) переходит в (7, 3).
  (Обычный перенос на постоянный вектор, типа x -> x + a,
  линейным не является.)

  Растяжение: изменение масштаба. Пример равномерного
  растяжения в 2 раза: (1, -3) -> (2, -6). Пример
  неравномерного: по x в 2 раза, по y в 0.5 раза: (2, 4) ->
  (4, 2).

  Поворот плоскости: поворачивает каждый вектор вокруг начала
  координат на один и тот же угол. Пример: поворот на 90
  градусов против часовой стрелки переводит (1, 0) в (0, 1),
  а (0, 1) в (-1, 0).

  Преобразование подобия — это идея «то же самое
  преобразование, но записанное в другом базисе». То есть
  геометрически действие одно и то же, меняется только
  система координат, в которой мы его описываем.
