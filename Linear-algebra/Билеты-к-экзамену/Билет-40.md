# Билет 40. Гипербола. Определение, уравнение, эксцентриситет, директрисы, асимптоты.

---

## 1. Определение гиперболы

**Определение.** Гипербола — это множество всех точек на плоскости, для которых **модуль разности расстояний** до двух фиксированных точек (фокусов) постоянен и равен $2a$.

$$|r_1 - r_2| = 2a$$

где $r_1$ и $r_2$ — расстояния от точки $M(x, y)$ до фокусов:
- $r_1 = \sqrt{(x + c)^2 + y^2}$ — расстояние до $F_1(-c, 0)$
- $r_2 = \sqrt{(x - c)^2 + y^2}$ — расстояние до $F_2(c, 0)$

**Словами:** у эллипса сумма расстояний до фокусов постоянна, а у гиперболы — **разность**. Точка на гиперболе всегда ближе к одному фокусу, чем к другому, и эта разница одинакова для всех точек.

**Числовой пример.** Гипербола $\frac{x^2}{9} - \frac{y^2}{16} = 1$: $a = 3$, $b = 4$, $c = 5$. Фокусы: $F_1(-5, 0)$, $F_2(5, 0)$.

Вершина $(3, 0)$:
- $r_1 = \sqrt{(3+5)^2 + 0} = 8$
- $r_2 = \sqrt{(3-5)^2 + 0} = 2$
- $|r_1 - r_2| = |8 - 2| = 6 = 2 \cdot 3 = 2a$ ✓

**Сравнение с эллипсом:**

| | Эллипс | Гипербола |
|---|--------|-----------|
| Условие | $r_1 + r_2 = 2a$ | $|r_1 - r_2| = 2a$ |
| Форма | Замкнутая (овал) | Две ветви (разомкнутые) |
| Связь $a, b, c$ | $c^2 = a^2 - b^2$ | $c^2 = a^2 + b^2$ |
| Эксцентриситет | $e < 1$ | $e > 1$ |

---

## 2. Элементы гиперболы

| Элемент | Обозначение | Что это |
|---------|-------------|---------|
| Действительная полуось | $a$ | Расстояние от центра до вершины (по горизонтали) |
| Мнимая полуось | $b$ | Определяет «раствор» гиперболы (по вертикали) |
| Фокусное расстояние | $c$ | Расстояние от центра до фокуса |
| Фокусы | $F_1(-c, 0)$, $F_2(c, 0)$ | Две точки, через которые определяется гипербола |
| Вершины | $(-a, 0)$, $(a, 0)$ | Ближайшие к центру точки гиперболы |
| Центр | $O(0, 0)$ | Середина между фокусами |

**Связь между $a$, $b$, $c$:**

$$c^2 = a^2 + b^2$$

**Словами:** у эллипса $c^2 = a^2 - b^2$ (фокусы внутри), а у гиперболы $c^2 = a^2 + b^2$ (фокусы дальше от центра, чем вершины).

**Числовой пример:** $a = 3$, $b = 4$ → $c = \sqrt{9 + 16} = \sqrt{25} = 5$.

Фокусы: $F_1(-5, 0)$ и $F_2(5, 0)$. Вершины: $(-3, 0)$ и $(3, 0)$.

## 3. Каноническое уравнение гиперболы

$$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$$

**Словами:** похоже на эллипс, но вместо «$+$» стоит «$-$». Из-за минуса гипербола не замкнутая — она расходится в стороны.

### Числовой пример

$a = 3$, $b = 4$:

$$\frac{x^2}{9} - \frac{y^2}{16} = 1$$

Проверим вершины:
- $(3, 0)$: $\frac{9}{9} - \frac{0}{16} = 1$ ✓
- $(-3, 0)$: $\frac{9}{9} - \frac{0}{16} = 1$ ✓

Точка $(5, \frac{16}{3})$: $\frac{25}{9} - \frac{256/9}{16} = \frac{25}{9} - \frac{16}{9} = \frac{9}{9} = 1$ ✓

Точка $(0, 0)$: $0 - 0 = 0 \neq 1$ — центр **не лежит** на гиперболе.

### Сопряжённая гипербола

Если поменять знаки:

$$\frac{y^2}{b^2} - \frac{x^2}{a^2} = 1$$

Ветви идут **вверх и вниз** (вместо влево-вправо). Фокусы на оси $Oy$: $(0, \pm c)$.

## 4. Эксцентриситет

**Определение:**

$$e = \frac{c}{a}, \quad e > 1$$

**Словами:** эксцентриситет показывает, насколько «широко» раскрыты ветви гиперболы:

| $e$ | Форма |
|-----|-------|
| $e$ чуть больше $1$ | Ветви узкие, почти параллельные |
| $e$ большой | Ветви широко раскрыты |

**Почему $e > 1$:** потому что $c > a$ (из $c^2 = a^2 + b^2$ следует $c > a$).

**Числовой пример:** $a = 3$, $c = 5$:

$$e = \frac{5}{3} \approx 1.67$$

## 5. Асимптоты

**Определение.** Асимптоты — прямые, к которым ветви гиперболы приближаются, но никогда не касаются:

$$y = \pm \frac{b}{a} x$$

где:
- $y, x$ — координаты точек на асимптотах
- $b$ — мнимая полуось гиперболы (по $y$)
- $a$ — действительная полуось гиперболы (по $x$)
- $\pm$ — две асимптоты, одна с наклоном вверх ($+$), другая вниз ($-$)

**Словами:** далеко от центра ветви гиперболы почти сливаются с этими прямыми. Асимптоты — это «направляющие» для ветвей.

**Как запомнить:** приравниваем уравнение гиперболы к нулю вместо единицы:

$$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 0 \quad \Rightarrow \quad y^2 = \frac{b^2}{a^2} x^2 \quad \Rightarrow \quad y = \pm \frac{b}{a} x$$

### Числовой пример

$a = 3$, $b = 4$:

$$y = \pm \frac{4}{3} x$$

Две прямые через начало координат. Гипербола «вписана» между ними.

---

## 6. Директрисы

**Определение.** Директрисы — две вертикальные прямые:

$$x = \pm \frac{a}{e} = \pm \frac{a^2}{c}$$

Директрисы расположены **между вершинами** (ближе к центру, чем вершины), потому что $\frac{a}{e} < a$ при $e > 1$.

**Фокально-директрисное свойство:** для любой точки гиперболы:

$$\frac{r}{d} = e$$

где $r$ — расстояние до фокуса, $d$ — расстояние до соответствующей директрисы.

**Словами:** то же свойство, что у эллипса, но $e > 1$, поэтому расстояние до фокуса **больше**, чем до директрисы.

### Числовой пример

$a = 3$, $c = 5$, $e = 5/3$:

$$x = \pm \frac{9}{5} = \pm 1.8$$

Директрисы: $x = -1.8$ и $x = 1.8$ — между вершинами $(-3, 0)$ и $(3, 0)$.

Проверим для вершины $(3, 0)$:
- Расстояние до фокуса $F_2(5, 0)$: $r = 5 - 3 = 2$
- Расстояние до директрисы $x = 1.8$: $d = 3 - 1.8 = 1.2$
- Отношение: $\frac{r}{d} = \frac{2}{1.2} = \frac{5}{3} = e$ ✓

## 7. Сравнение: эллипс vs гипербола vs парабола

| | Эллипс | Гипербола | Парабола |
|---|--------|-----------|----------|
| Уравнение | $\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1$ | $\dfrac{x^2}{a^2} - \dfrac{y^2}{b^2} = 1$ | $y^2 = 2px$ |
| Отличие | «$+$» между дробями | «$-$» между дробями | Квадрат только у одной переменной |
| Форма | Замкнутый овал | Две ветви | Одна ветвь, разомкнутая |
| Связь $a, b, c$ | $c^2 = a^2 - b^2$ | $c^2 = a^2 + b^2$ | Только параметр $p$ |
| Эксцентриситет | $0 \leq e < 1$ | $e > 1$ | $e = 1$ |
| Директрисы | За пределами | Между вершинами | Одна, слева от вершины |
| Асимптоты | Нет | $y = \pm \frac{b}{a} x$ | Нет |
| Фокальное свойство | $r_1 + r_2 = 2a$ | $\|r_1 - r_2\| = 2a$ | $r = d$ |
| Директрисное свойство | $r/d = e < 1$ | $r/d = e > 1$ | $r/d = 1$ |
