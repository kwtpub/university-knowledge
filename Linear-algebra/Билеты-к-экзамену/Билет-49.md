# Билет 49. Классификация поверхностей второго порядка. Эллиптический и гиперболический параболоиды.

---

## 1. Место параболоидов в классификации

В билете 48 мы рассмотрели поверхности, у которых **все три** переменные входят в квадрате (эллипсоид, гиперболоиды, конус). Параболоиды отличаются тем, что **одна переменная входит линейно** (в первой степени), а не в квадрате.

| Тип                        | Уравнение                                | Особенность           |
| -------------------------- | ---------------------------------------- | --------------------- |
| Эллиптический параболоид   | $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 2z$ | $+, +$ и линейный $z$ |
| Гиперболический параболоид | $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 2z$ | $+, -$ и линейный $z$ |

**Словами:** параболоиды — это «объёмные параболы». У них нет центра симметрии (в отличие от эллипсоида и гиперболоидов), они уходят в бесконечность только в одну сторону.

---

## 2. Эллиптический параболоид

**Каноническое уравнение:**

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 2z$$

**Словами:** «чаша» или «спутниковая тарелка». Открыта вверх (в сторону $z > 0$). Вершина — в начале координат.

### Элементы

| Элемент | Значение |
|---------|----------|
| Вершина | $(0, 0, 0)$ — самая нижняя точка |
| Ось | Ось $z$ — ось симметрии |
| Параметры | $a, b$ — определяют «ширину» чаши |

### Сечения

**Горизонтальные сечения** ($z = h$):

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 2h$$

| $h$ | Сечение | Пояснение |
|-----|---------|-----------|
| $h > 0$ | **Эллипс** | Растёт с высотой |
| $h = 0$ | **Точка** $(0, 0, 0)$ | Вершина |
| $h < 0$ | **Пусто** | Чаша не опускается ниже вершины |

**Вертикальные сечения:**

- $y = 0$: $\frac{x^2}{a^2} = 2z$ → **парабола** в плоскости $xOz$
- $x = 0$: $\frac{y^2}{b^2} = 2z$ → **парабола** в плоскости $yOz$

**Словами:** режем горизонтально — эллипсы (растут вверх). Режем вертикально через ось — параболы.

### Числовой пример

$$\frac{x^2}{4} + \frac{y^2}{9} = 2z$$

$a = 2$, $b = 3$.

Сечение $z = 2$: $\frac{x^2}{4} + \frac{y^2}{9} = 4$ → $\frac{x^2}{16} + \frac{y^2}{36} = 1$ — эллипс с полуосями $4$ и $6$.

Сечение $z = 0$: $\frac{x^2}{4} + \frac{y^2}{9} = 0$ → точка $(0, 0, 0)$.

Сечение $y = 0$: $\frac{x^2}{4} = 2z$ → $z = \frac{x^2}{8}$ — парабола, раскрытая вверх.

### Частный случай

Если $a = b$ — **параболоид вращения**: $x^2 + y^2 = 2a^2 z$. Сечения — окружности.

## 3. Гиперболический параболоид

**Каноническое уравнение:**

$$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 2z$$

**Словами:** «седло» — поверхность, которая в одном направлении изгибается вверх, а в другом — вниз. Как седло лошади или чипс Pringles.

### Элементы

| Элемент | Значение |
|---------|----------|
| Седловая точка | $(0, 0, 0)$ — ни максимум, ни минимум |
| Ось | Ось $z$ |
| Параметры | $a, b$ — определяют форму седла |

### Сечения

**Горизонтальные сечения** ($z = h$):

$$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 2h$$

| $h$ | Сечение | Пояснение |
|-----|---------|-----------|
| $h > 0$ | **Гипербола** (ветви вдоль $x$) | Правая часть > 0 |
| $h = 0$ | **Пара пересекающихся прямых** | $\frac{x^2}{a^2} = \frac{y^2}{b^2}$ → $y = \pm \frac{b}{a}x$ |
| $h < 0$ | **Гипербола** (ветви вдоль $y$) | Правая часть < 0, гипербола «повёрнута» |

**Вертикальные сечения:**

- $y = 0$: $\frac{x^2}{a^2} = 2z$ → **парабола вверх** ↑
- $x = 0$: $-\frac{y^2}{b^2} = 2z$ → **парабола вниз** ↓

**Словами:** режем горизонтально — гиперболы (при $z > 0$ одна ориентация, при $z < 0$ — другая). Режем вертикально — параболы, но в разных направлениях! По $x$ — вверх, по $y$ — вниз. Отсюда форма седла.

### Числовой пример

$$\frac{x^2}{4} - \frac{y^2}{9} = 2z$$

$a = 2$, $b = 3$.

Сечение $z = 1$: $\frac{x^2}{4} - \frac{y^2}{9} = 2$ → $\frac{x^2}{8} - \frac{y^2}{18} = 1$ — гипербола (ветви вдоль $x$).

Сечение $z = -1$: $\frac{x^2}{4} - \frac{y^2}{9} = -2$ → $\frac{y^2}{18} - \frac{x^2}{8} = 1$ — гипербола (ветви вдоль $y$).

Сечение $z = 0$: $\frac{x^2}{4} - \frac{y^2}{9} = 0$ → $y = \pm \frac{3}{2}x$ — две прямые.

Сечение $y = 0$: $z = \frac{x^2}{8}$ — парабола вверх.

Сечение $x = 0$: $z = -\frac{y^2}{18}$ — парабола вниз.

### Линейчатая поверхность

Гиперболический параболоид — **линейчатая поверхность**: через каждую его точку проходят **две прямые**, целиком лежащие на поверхности (как и у однополостного гиперболоида).

## 4. Сравнение двух параболоидов

| | Эллиптический | Гиперболический |
|---|---------------|-----------------|
| Уравнение | $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 2z$ | $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 2z$ |
| Отличие | «$+$» между дробями | «$-$» между дробями |
| Форма | Чаша | Седло |
| Горизонтальные сечения | Эллипсы | Гиперболы |
| Вертикальные сечения | Параболы вверх | Парабола вверх ($y=0$) и вниз ($x=0$) |
| Вершина | Минимум | Седловая точка |
| Ограничена снизу? | Да ($z \geq 0$) | Нет (уходит и вверх, и вниз) |
| Линейчатая? | Нет | Да |

## 5. Полная классификация поверхностей второго порядка

| № | Поверхность | Уравнение | Форма |
|---|-------------|-----------|-------|
| 1 | Эллипсоид | $\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1$ | Мяч |

| 3 | Двуполостный гиперболоид | $\frac{z^2}{c^2} - \frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ | Две чаши |
| 4 | Конус | $\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 0$ | Воронка |
| 5 | Эллиптический параболоид | $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 2z$ | Чаша / тарелка |
| 6 | Гиперболический параболоид | $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 2z$ | Седло |
| 7 | Эллиптический цилиндр | $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ | Труба (овальная) |
