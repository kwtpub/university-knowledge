# Формулы: плоскости и прямые в пространстве (билеты 34–38)

---

## Билет 34. Плоскость — общее уравнение

**Общее уравнение:**

$$Ax + By + Cz + D = 0$$

**Нормальный вектор:** $\vec{n} = (A,\; B,\; C)$ — перпендикулярен плоскости

### Частные случаи

| Условие | Значение |
|---|---|
| $D = 0$ | Проходит через начало координат |
| $A = 0$ | Параллельна оси $Ox$ |
| $B = 0$ | Параллельна оси $Oy$ |
| $C = 0$ | Параллельна оси $Oz$ |
| $A = B = 0$ | Параллельна плоскости $xOy$ (т.е. $z = \text{const}$) |
| $A = C = 0$ | Параллельна плоскости $xOz$ (т.е. $y = \text{const}$) |
| $B = C = 0$ | Параллельна плоскости $yOz$ (т.е. $x = \text{const}$) |

**Правило:** нет переменной → параллельна этой оси. Нет двух переменных → параллельна координатной плоскости.

## Билет 35. Способы задания плоскости

### 1) По точке $M_0(x_0, y_0, z_0)$ и нормали $\vec{n} = (A, B, C)$

$$A(x - x_0) + B(y - y_0) + C(z - z_0) = 0$$

### 2) Векторное уравнение

$$(\vec{r} - \vec{r_0},\; \vec{n}) = 0$$

Это то же самое, только через радиус-векторы.

### 3) Через три точки $M_1, M_2, M_3$

$$\begin{vmatrix} x - x_1 & y - y_1 & z - z_1 \\ x_2 - x_1 & y_2 - y_1 & z_2 - z_1 \\ x_3 - x_1 & y_3 - y_1 & z_3 - z_1 \end{vmatrix} = 0$$

Смешанное произведение $(\overrightarrow{M_1M},\; \overrightarrow{M_1M_2},\; \overrightarrow{M_1M_3}) = 0$

### 4) В отрезках

$$\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$$

$a, b, c$ — отрезки, которые плоскость отсекает на осях $Ox, Oy, Oz$.

Не работает, если $D = 0$ или плоскость параллельна какой-то оси.

## Билет 36. Расстояние, угол, параллельность, перпендикулярность плоскостей

### Расстояние от точки $M_0(x_0, y_0, z_0)$ до плоскости $Ax + By + Cz + D = 0$

$$\boxed{d = \frac{|Ax_0 + By_0 + Cz_0 + D|}{\sqrt{A^2 + B^2 + C^2}}}$$

Числитель — подставили точку, взяли модуль. Знаменатель — длина нормали.

### Угол между двумя плоскостями

Нормали: $\vec{n_1} = (A_1, B_1, C_1)$, $\vec{n_2} = (A_2, B_2, C_2)$

$$\boxed{\cos\varphi = \frac{|A_1A_2 + B_1B_2 + C_1C_2|}{\sqrt{A_1^2+B_1^2+C_1^2} \cdot \sqrt{A_2^2+B_2^2+C_2^2}}}$$

### Параллельность плоскостей

$$\frac{A_1}{A_2} = \frac{B_1}{B_2} = \frac{C_1}{C_2}$$

Нормали коллинеарны → плоскости параллельны.

Если ещё и $\frac{D_1}{D_2}$ = то же число → плоскости **совпадают**.

### Перпендикулярность плоскостей

$$A_1A_2 + B_1B_2 + C_1C_2 = 0$$

Скалярное произведение нормалей = 0.

## Билет 37. Прямая в пространстве

Прямая задаётся **точкой** $M_0(x_0, y_0, z_0)$ и **направляющим вектором** $\vec{s} = (l, m, n)$.

### Каноническое уравнение

$$\boxed{\frac{x - x_0}{l} = \frac{y - y_0}{m} = \frac{z - z_0}{n}}$$

Если $m = 0$: пишем $y = y_0$ отдельно, остальные дроби приравниваем.

### Параметрическое уравнение

$$\boxed{\begin{cases} x = x_0 + lt \\ y = y_0 + mt \\ z = z_0 + nt \end{cases}, \quad t \in \mathbb{R}}$$

### Связь: каноническое $\to$ параметрическое

Каждую дробь обозначаем за $t$, выражаем $x, y, z$.

### Прямая как пересечение двух плоскостей

$$\begin{cases} A_1x + B_1y + C_1z + D_1 = 0 \\ A_2x + B_2y + C_2z + D_2 = 0 \end{cases}$$

**Направляющий вектор** прямой пересечения:

$$\vec{s} = \vec{n_1} \times \vec{n_2} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ A_1 & B_1 & C_1 \\ A_2 & B_2 & C_2 \end{vmatrix}$$

**Точку** на прямой находим: положим, например, $z = 0$ и решим систему двух уравнений с двумя неизвестными.

## Билет 38. Углы, параллельность, перпендикулярность

### Угол между двумя прямыми

Направляющие: $\vec{s_1} = (l_1, m_1, n_1)$, $\vec{s_2} = (l_2, m_2, n_2)$

$$\boxed{\cos\varphi = \frac{|l_1l_2 + m_1m_2 + n_1n_2|}{\sqrt{l_1^2+m_1^2+n_1^2} \cdot \sqrt{l_2^2+m_2^2+n_2^2}}}$$

### Угол между прямой и плоскостью

Направляющий $\vec{s} = (l, m, n)$, нормаль $\vec{n} = (A, B, C)$

$$\boxed{\sin\varphi = \frac{|Al + Bm + Cn|}{\sqrt{A^2+B^2+C^2} \cdot \sqrt{l^2+m^2+n^2}}}$$

> Тут **синус**, не косинус! Потому что нормаль $\perp$ плоскости, и угол «дополняется» до $90°$.

### Сводная таблица условий

|  | Параллельность | Перпендикулярность |
|---|---|---|
| **Прямая ↔ прямая** | $\dfrac{l_1}{l_2} = \dfrac{m_1}{m_2} = \dfrac{n_1}{n_2}$ | $l_1l_2 + m_1m_2 + n_1n_2 = 0$ |
| **Плоскость ↔ плоскость** | $\dfrac{A_1}{A_2} = \dfrac{B_1}{B_2} = \dfrac{C_1}{C_2}$ | $A_1A_2 + B_1B_2 + C_1C_2 = 0$ |
| **Прямая ↔ плоскость** | $Al + Bm + Cn = 0$ | $\dfrac{A}{l} = \dfrac{B}{m} = \dfrac{C}{n}$ |

> **Прямая ↔ плоскость — наоборот:**
> - Прямая **параллельна** плоскости → $\vec{s} \perp \vec{n}$ (скалярное произведение = 0)
> - Прямая **перпендикулярна** плоскости → $\vec{s} \parallel \vec{n}$ (пропорциональны)

## Шпаргалка: какой вектор откуда читать

| Объект | Уравнение | Какой вектор |
|---|---|---|
| Плоскость $Ax + By + Cz + D = 0$ | Коэффициенты при $x, y, z$ | **Нормаль** $\vec{n} = (A, B, C)$ |
| Прямая $\frac{x-x_0}{l} = \frac{y-y_0}{m} = \frac{z-z_0}{n}$ | Знаменатели дробей | **Направляющий** $\vec{s} = (l, m, n)$ |

---

## Шпаргалка: cos или sin?

| Задача | Формула | Почему |
|---|---|---|
| Угол прямая–прямая | $\cos\varphi$ | Оба вектора — направляющие, считаем угол между ними напрямую |
| Угол плоскость–плоскость | $\cos\varphi$ | Оба вектора — нормали, считаем угол между ними напрямую |
| Угол прямая–плоскость | $\sin\varphi$ | Один — направляющий, другой — нормаль. Нормаль $\perp$ плоскости, поэтому $\varphi = 90° - \alpha$, и $\cos\alpha = \sin\varphi$ |
