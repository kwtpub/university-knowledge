# Билет 34. Общее уравнение плоскости в пространстве. Частные случаи.

---

## 1. Общее уравнение плоскости

**Определение.** Любая плоскость в пространстве задаётся уравнением первой степени:

$$Ax + By + Cz + D = 0$$

где $A, B, C, D$ — числа, причём $A, B, C$ **не все равны нулю** одновременно.

**Словами:** плоскость — это «бесконечный плоский лист» в пространстве. Каждая точка $(x, y, z)$, которая при подстановке даёт $0$, лежит на этой плоскости.

### Нормальный вектор

Вектор $\vec{n} = (A, B, C)$ называется **нормальным вектором** плоскости.

**Словами:** нормаль — это вектор, **перпендикулярный** плоскости. Он «торчит» из плоскости под прямым углом. Коэффициенты при $x, y, z$ в уравнении — это координаты этого вектора.

**Числовой пример.** Плоскость $2x - 3y + z - 6 = 0$:
- $A = 2$, $B = -3$, $C = 1$, $D = -6$
- Нормальный вектор: $\vec{n} = (2, -3, 1)$

Проверим точку $(3, 0, 0)$: $2 \cdot 3 - 3 \cdot 0 + 1 \cdot 0 - 6 = 0$ ✓ — лежит на плоскости.

Точка $(0, 0, 0)$: $0 - 0 + 0 - 6 = -6 \neq 0$ ✗ — не лежит.

### Что определяет $D$?

Коэффициент $D$ отвечает за **сдвиг** плоскости от начала координат.

- $D = 0$ → плоскость проходит через начало координат
- $D \neq 0$ → плоскость не проходит через начало координат

---

## 2. Частные случаи

### 2.1. Плоскость, проходящая через начало координат

$$Ax + By + Cz = 0 \quad (D = 0)$$

**Словами:** если $D = 0$, то точка $(0, 0, 0)$ подходит: $A \cdot 0 + B \cdot 0 + C \cdot 0 = 0$ ✓. Плоскость проходит через начало координат.

**Пример:** $x + 2y - z = 0$

Проверка: $(0, 0, 0)$ → $0 + 0 - 0 = 0$ ✓ — начало координат лежит на плоскости.

---

### 2.2. Плоскость, параллельная координатной оси

Если в уравнении **отсутствует одна переменная** — плоскость параллельна соответствующей оси.

| Отсутствует | Уравнение | Параллельна оси | Почему |
|-------------|-----------|-----------------|--------|
| $x$ | $By + Cz + D = 0$ | $Ox$ | Положение точки по $x$ не влияет — плоскость «тянется» вдоль $x$ |
| $y$ | $Ax + Cz + D = 0$ | $Oy$ | Положение точки по $y$ не влияет — плоскость «тянется» вдоль $y$ |
| $z$ | $Ax + By + D = 0$ | $Oz$ | Положение точки по $z$ не влияет — плоскость «тянется» вдоль $z$ |

**Словами:** если переменной нет в уравнении, значит от неё ничего не зависит. Можно менять эту координату как угодно — точка всё равно лежит на плоскости. Поэтому плоскость бесконечно тянется вдоль этой оси → параллельна ей.

**Пример:** $3y - 2z + 1 = 0$

Нет $x$ → параллельна оси $Ox$. При любом $x$ точка $(x, y, z)$ лежит на плоскости, если $3y - 2z + 1 = 0$.

---

### 2.3. Плоскость, параллельная координатной плоскости

Если отсутствуют **две переменные** — плоскость параллельна координатной плоскости.

| Уравнение | Параллельна плоскости | Вид |
|-----------|----------------------|-----|
| $Cz + D = 0$ → $z = -D/C$ | $xOy$ | Горизонтальная плоскость на высоте $z = \text{const}$ |
| $By + D = 0$ → $y = -D/B$ | $xOz$ | Вертикальная плоскость при $y = \text{const}$ |
| $Ax + D = 0$ → $x = -D/A$ | $yOz$ | Вертикальная плоскость при $x = \text{const}$ |

**Словами:** плоскость $z = 5$ — это горизонтальный «пол» на высоте 5. Все точки с $z = 5$ и любыми $x, y$ на ней лежат. Она параллельна полу $xOy$ (где $z = 0$).

**Примеры:**
- $z = 3$ — плоскость параллельна $xOy$, на высоте 3
- $x = -2$ — плоскость параллельна $yOz$, слева от начала координат
- $y = 0$ — это сама координатная плоскость $xOz$

---

### 2.4. Координатные плоскости (частный случай)

Три координатные плоскости — это тоже частные случаи:

| Координатная плоскость | Уравнение | Какие точки лежат |
|------------------------|-----------|-------------------|
| $xOy$ | $z = 0$ | Все точки с $z = 0$ |
| $xOz$ | $y = 0$ | Все точки с $y = 0$ |
| $yOz$ | $x = 0$ | Все точки с $x = 0$ |

---

## 3. Сводная таблица всех частных случаев

| Условие | Что это значит | Пример |
|---------|---------------|--------|
| $D = 0$ | Проходит через начало координат | $x + y + z = 0$ |
| $A = 0$ | Параллельна оси $Ox$ | $2y + 3z - 1 = 0$ |
| $B = 0$ | Параллельна оси $Oy$ | $x - z + 4 = 0$ |
| $C = 0$ | Параллельна оси $Oz$ | $x + 2y - 5 = 0$ |
| $A = B = 0$ | Параллельна плоскости $xOy$ | $z = 7$ |
| $A = C = 0$ | Параллельна плоскости $xOz$ | $y = -3$ |
| $B = C = 0$ | Параллельна плоскости $yOz$ | $x = 2$ |
| $A = 0, D = 0$ | Через начало координат, параллельна $Ox$ | $y + z = 0$ |

---

## 4. Как быстро запомнить

**Правило:** переменная отсутствует → плоскость параллельна этой оси.

- Нет одной переменной → параллельна одной оси
- Нет двух переменных → параллельна координатной плоскости (которая образована этими двумя осями)
- $D = 0$ → проходит через начало координат

**Нормальный вектор** $\vec{n} = (A, B, C)$ — всегда перпендикулярен плоскости. Его координаты — это коэффициенты при $x, y, z$.
