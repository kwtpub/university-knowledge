# Билет 35. Уравнение плоскости через три точки. По точке и нормали. Векторное уравнение. Уравнение в отрезках.

---

## 1. Уравнение плоскости по точке и нормальному вектору

**Дано:** точка $M_0(x_0, y_0, z_0)$ на плоскости и нормальный вектор $\vec{n} = (A, B, C)$.

**Уравнение:**

$$A(x - x_0) + B(y - y_0) + C(z - z_0) = 0$$

**Словами:** берём любую точку $M(x, y, z)$ на плоскости. Вектор $\overrightarrow{M_0 M} = (x - x_0,\; y - y_0,\; z - z_0)$ лежит в плоскости. А нормаль $\vec{n}$ перпендикулярна плоскости. Значит, их скалярное произведение равно нулю.

**Числовой пример.** Точка $M_0(1, 2, 3)$, нормаль $\vec{n} = (2, -1, 4)$:

$$2(x - 1) + (-1)(y - 2) + 4(z - 3) = 0$$

Раскрываем:

$$2x - 2 - y + 2 + 4z - 12 = 0$$

$$2x - y + 4z - 12 = 0$$

Проверка: точка $(1, 2, 3)$: $2 - 2 + 12 - 12 = 0$ ✓

## 2. Векторное уравнение плоскости

**Уравнение:**

$$(\vec{r} - \vec{r_0},\; \vec{n}) = 0$$

где:
- $\vec{r} = (x, y, z)$ — радиус-вектор произвольной точки на плоскости
- $\vec{r_0} = (x_0, y_0, z_0)$ — радиус-вектор известной точки на плоскости
- $\vec{n} = (A, B, C)$ — нормальный вектор
- $(\;,\;)$ — скалярное произведение

**Словами:** это то же самое, что уравнение по точке и нормали, только записанное через векторы. Вектор от $M_0$ до $M$ перпендикулярен нормали → скалярное произведение = 0.

Если раскрыть скалярное произведение, получим:

$$A(x - x_0) + B(y - y_0) + C(z - z_0) = 0$$

То есть это та же формула из пункта 1.

## 3. Уравнение плоскости через три точки

**Дано:** три точки $M_1(x_1, y_1, z_1)$, $M_2(x_2, y_2, z_2)$, $M_3(x_3, y_3, z_3)$, не лежащие на одной прямой.

**Уравнение:**

$$\begin{vmatrix} x - x_1 & y - y_1 & z - z_1 \\ x_2 - x_1 & y_2 - y_1 & z_2 - z_1 \\ x_3 - x_1 & y_3 - y_1 & z_3 - z_1 \end{vmatrix} = 0$$

**Словами:** строим два вектора, лежащих в плоскости:
- $\overrightarrow{M_1 M_2} = (x_2 - x_1,\; y_2 - y_1,\; z_2 - z_1)$
- $\overrightarrow{M_1 M_3} = (x_3 - x_1,\; y_3 - y_1,\; z_3 - z_1)$

Берём произвольную точку $M(x, y, z)$ и вектор $\overrightarrow{M_1 M} = (x - x_1,\; y - y_1,\; z - z_1)$.

Три вектора $\overrightarrow{M_1 M}$, $\overrightarrow{M_1 M_2}$, $\overrightarrow{M_1 M_3}$ лежат в одной плоскости → их смешанное произведение (определитель) = 0.

### Числовой пример

Три точки: $M_1(1, 0, 0)$, $M_2(0, 2, 0)$, $M_3(0, 0, 3)$.

Векторы:
- $\overrightarrow{M_1 M_2} = (-1, 2, 0)$
- $\overrightarrow{M_1 M_3} = (-1, 0, 3)$

Определитель:

$$\begin{vmatrix} x - 1 & y - 0 & z - 0 \\ -1 & 2 & 0 \\ -1 & 0 & 3 \end{vmatrix} = 0$$

Раскладываем по первой строке:

$$(x-1) \begin{vmatrix} 2 & 0 \\ 0 & 3 \end{vmatrix} - y \begin{vmatrix} -1 & 0 \\ -1 & 3 \end{vmatrix} + z \begin{vmatrix} -1 & 2 \\ -1 & 0 \end{vmatrix} = 0$$

$$(x-1)(6 - 0) - y(-3 - 0) + z(0 + 2) = 0$$

$$6(x-1) + 3y + 2z = 0$$

$$6x + 3y + 2z - 6 = 0$$

Проверка всех трёх точек:
- $M_1(1,0,0)$: $6 + 0 + 0 - 6 = 0$ ✓
- $M_2(0,2,0)$: $0 + 6 + 0 - 6 = 0$ ✓
- $M_3(0,0,3)$: $0 + 0 + 6 - 6 = 0$ ✓

## 4. Уравнение плоскости в отрезках

**Определение.** Если плоскость пересекает ось $Ox$ в точке $(a, 0, 0)$, ось $Oy$ в точке $(0, b, 0)$, ось $Oz$ в точке $(0, 0, c)$, то уравнение:

$$\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$$

**Словами:** $a$, $b$, $c$ — длины отрезков, которые плоскость «отсекает» на координатных осях. Подставляешь точку пересечения с любой осью — получаешь 1.

### Как получить из общего уравнения

Из $6x + 3y + 2z - 6 = 0$ (пример выше):

$$6x + 3y + 2z = 6$$

Делим обе части на 6:

$$\frac{x}{1} + \frac{y}{2} + \frac{z}{3} = 1$$

Значит: $a = 1$, $b = 2$, $c = 3$ — плоскость отсекает на осях отрезки 1, 2 и 3.

Проверка:
- На оси $Ox$: точка $(1, 0, 0)$ → $\frac{1}{1} + 0 + 0 = 1$ ✓
- На оси $Oy$: точка $(0, 2, 0)$ → $0 + \frac{2}{2} + 0 = 1$ ✓
- На оси $Oz$: точка $(0, 0, 3)$ → $0 + 0 + \frac{3}{3} = 1$ ✓

### Когда нельзя использовать

Уравнение в отрезках **не работает**, если плоскость:
- Проходит через начало координат ($D = 0$) — тогда хотя бы один «отрезок» = 0, деление на 0
- Параллельна какой-то оси — тогда она не пересекает эту ось, отрезок = $\infty$

## 5. Связь между всеми формами

Все четыре формы задают **одну и ту же** плоскость, просто записанную по-разному:

| Форма | Что нужно знать | Уравнение |
|-------|----------------|-----------|
| По точке и нормали | Точка $M_0$ + нормаль $\vec{n}$ | $A(x-x_0) + B(y-y_0) + C(z-z_0) = 0$ |
| Векторное | Точка $\vec{r_0}$ + нормаль $\vec{n}$ | $(\vec{r} - \vec{r_0}, \vec{n}) = 0$ |
| Через три точки | Три точки $M_1, M_2, M_3$ | Определитель $3 \times 3 = 0$ |
| В отрезках | Отсечки $a, b, c$ на осях | $\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$ |
| Общее | Коэффициенты $A, B, C, D$ | $Ax + By + Cz + D = 0$ |

Любую из этих форм можно привести к общему виду $Ax + By + Cz + D = 0$, раскрыв скобки или определитель.
