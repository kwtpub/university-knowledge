# I. Матрицы и определители

## Матрицы

**Матрица:**  
A = (a_ij), размер m×n

**Виды матриц:**

- квадратная, прямоугольная
    
- нулевая: 0
    
- единичная: E
    
- диагональная
    
- треугольная
    
- симметрическая: Aᵀ = A
    

**Операции:**

- (A + B)_ij = a_ij + b_ij
    
- (λA)_ij = λa_ij
    
- (AB)_ij = Σ a_ik b_kj
    

**Свойства умножения:**

- A(BC) = (AB)C
    
- A(B + C) = AB + AC
    
- generally AB ≠ BA
    

---

## Элементарные преобразования

- перестановка строк
    
- умножение строки на λ ≠ 0
    
- прибавление к строке другой строки
    

**Ступенчатый вид:**

- нулевые строки внизу
    
- ведущие элементы сдвигаются вправо
    

**Главный ступенчатый вид:**

- ведущие элементы = 1
    
- в столбце ведущего элемента остальные нули
    

---

## Ранг матрицы

**rank A** = число ненулевых строк в ступенчатом виде  
= максимальный порядок ненулевого минора

**Теорема о базисном миноре:**  
Любой минор максимального порядка является базисным

---

## Обратная матрица

A⁻¹ существует ⇔ det A ≠ 0

**Через союзную:**
A⁻¹ = (1/det A) · adj A

**Метод Гаусса:**
(A|E) → (E|A⁻¹)

**Свойства:**

- (AB)⁻¹ = B⁻¹A⁻¹
- (Aᵀ)⁻¹ = (A⁻¹)ᵀ
- (A⁻¹)⁻¹ = A
    

---

## Определители

**Определение (2×2):**
|a b|
|c d| = ad − bc

**Определение (3×3) — правило Саррюса:**
det A = a₁₁a₂₂a₃₃ + a₁₂a₂₃a₃₁ + a₁₃a₂₁a₃₂ − a₁₃a₂₂a₃₁ − a₁₁a₂₃a₃₂ − a₁₂a₂₁a₃₃

**Разложение по строке:**
det A = Σ a_ij A_ij

**Алгебраическое дополнение:**  
A_ij = (−1)^{i+j} M_ij

**Свойства:**

- det Aᵀ = det A
    
- перестановка строк → смена знака
    
- умножение строки на λ → det·λ
    
- det = 0 ⇔ строки линейно зависимы

# II. Системы линейных уравнений

Ax = b

**Матричный метод:**
x = A⁻¹b

**Крамер:**
x_i = det A_i / det A

**Кронекер–Капелли:**

- совместна ⇔ rank A = rank(A|b)
    

**Однородная:**  
Ax = 0

- всегда имеет нулевое решение
    
- ненулевое ⇔ rank A < n
    

**Общее решение:**
x = x_частн + x_одн

**Фундаментальная система решений:**
dim ФСР = n − rank A

# III. Линейные пространства

**Линейная оболочка:**  
Lin{v₁,…,v_k} = {Σ α_i v_i}

**Базис:**

- линейно независим
    
- порождает пространство
    

**Размерность:**  
dim V = число векторов базиса

**Смена базиса:**  
x = C x′

---

## Евклидово пространство

**Скалярное произведение:**  
(x, y) = Σ x_i y_i

**Матрица Грама:**  
G = ((v_i, v_j))

**Длина:**  
|x| = √(x, x)

**Угол:**
cos φ = (x, y) / (|x||y|)

**Неравенство Коши-Буняковского-Шварца:**
|(x, y)| ≤ |x| · |y|

**Неравенство треугольника:**
|x + y| ≤ |x| + |y|

**Парсеваль:**
|x|² = Σ x_i² (в ортонорм. базисе)

---

## Ортогонализация Грама–Шмидта

u₁ = v₁
u_k = v_k − Σ ( (v_k,u_i)/(u_i,u_i) ) u_i

**Ортогональная проекция:**
proj_u(v) = ((v,u)/(u,u)) · u

**Ортогональное дополнение:**
dim V + dim V⊥ = dim пространства

---

# IV. Линейные отображения

**Линейное отображение:**  
T(ax + by) = aT(x) + bT(y)

**Матрица отображения:**  
T(x) = Ax

**Смена базиса:**
A′ = C⁻¹ A C

**Матрицы преобразований плоскости:**

- Поворот на угол θ:
  R = |cos θ  −sin θ|
      |sin θ   cos θ|

- Растяжение:
  D = |k₁  0 |
      |0   k₂|

- Сдвиг вдоль x:
  S = |1  k|
      |0  1|

- Отражение относительно оси x:
  |1   0|
  |0  −1|

---

## Собственные значения

Ax = λx

**Характеристическое уравнение:**  
det(A − λE) = 0

**Свойства:**

- сумма λ = tr A
- произведение λ = det A
- собственные векторы разных λ линейно независимы

**Диагональный вид:**
Если A имеет n линейно независимых собственных векторов:
A = P · D · P⁻¹, где D = diag(λ₁, ..., λₙ)

# V. Векторы и координаты

**Вектор:**  
AB = (x₂ − x₁, y₂ − y₁, z₂ − z₁)

**Расстояние:**  
|AB| = √((x₂−x₁)² + ...)

**Деление отрезка:**
M = ( (mx₂+nx₁)/(m+n), (my₂+ny₁)/(m+n), (mz₂+nz₁)/(m+n) )

**Середина отрезка:**
M = ( (x₁+x₂)/2, (y₁+y₂)/2, (z₁+z₂)/2 )

---

## Системы координат

**Полярные (2D):**
x = r cos φ
y = r sin φ

**Цилиндрические:**
x = r cos φ
y = r sin φ
z = z

**Сферические:**
x = r sin θ cos φ
y = r sin θ sin φ
z = r cos θ

---

## Произведения векторов

**Скалярное:**
a·b = |a||b|cos φ = a₁b₁ + a₂b₂ + a₃b₃

**Признак ортогональности:** a·b = 0

**Проекция вектора:**
proj_b(a) = (a·b)/|b|

---

**Векторное:**
|a×b| = |a||b|sin φ

**Координатная форма:**
a×b = |i   j   k |
      |a₁  a₂  a₃|
      |b₁  b₂  b₃|

= (a₂b₃ − a₃b₂, a₃b₁ − a₁b₃, a₁b₂ − a₂b₁)

**Признак коллинеарности:** a×b = 0

**Площадь параллелограмма:** S = |a×b|
**Площадь треугольника:** S = ½|a×b|

---

**Смешанное:**
[a,b,c] = |a₁  a₂  a₃|
          |b₁  b₂  b₃|
          |c₁  c₂  c₃|

**Признак компланарности:** [a,b,c] = 0

**Объём параллелепипеда:** V = |[a,b,c]|
**Объём пирамиды:** V = ⅙|[a,b,c]|

# VI. Аналитическая геометрия

## Плоскость

**Общее уравнение:**
Ax + By + Cz + D = 0

**Нормальный вектор:** n = (A, B, C)

**Плоскость через точку M₀(x₀,y₀,z₀) с нормалью (A,B,C):**
A(x−x₀) + B(y−y₀) + C(z−z₀) = 0

**Плоскость через 3 точки:**
|x−x₁   y−y₁   z−z₁|
|x₂−x₁  y₂−y₁  z₂−z₁| = 0
|x₃−x₁  y₃−y₁  z₃−z₁|

**Плоскость в отрезках:**
x/a + y/b + z/c = 1

**Расстояние от точки до плоскости:**
d = |Ax₀+By₀+Cz₀+D| / √(A²+B²+C²)

**Угол между плоскостями:**
cos φ = |A₁A₂+B₁B₂+C₁C₂| / (√(A₁²+B₁²+C₁²)·√(A₂²+B₂²+C₂²))

**Условие параллельности:** A₁/A₂ = B₁/B₂ = C₁/C₂
**Условие перпендикулярности:** A₁A₂ + B₁B₂ + C₁C₂ = 0

---

## Прямая в пространстве

**Каноническое уравнение:**
(x−x₀)/l = (y−y₀)/m = (z−z₀)/n

**Направляющий вектор:** s = (l, m, n)

**Параметрическое уравнение:**
x = x₀ + lt
y = y₀ + mt
z = z₀ + nt

**Прямая как пересечение плоскостей:**
A₁x + B₁y + C₁z + D₁ = 0
A₂x + B₂y + C₂z + D₂ = 0

**Угол между прямыми:**
cos φ = |l₁l₂+m₁m₂+n₁n₂| / (√(l₁²+m₁²+n₁²)·√(l₂²+m₂²+n₂²))

**Угол между прямой и плоскостью:**
sin φ = |Al+Bm+Cn| / (√(A²+B²+C²)·√(l²+m²+n²))

---

## Кривые второго порядка

**Эллипс:**
x²/a² + y²/b² = 1 (a ≥ b)

- Полуоси: a, b
- Фокусы: (±c, 0), где c² = a² − b²
- Эксцентриситет: e = c/a < 1
- Директрисы: x = ±a/e

**Гипербола:**
x²/a² − y²/b² = 1

- Полуоси: a, b
- Фокусы: (±c, 0), где c² = a² + b²
- Эксцентриситет: e = c/a > 1
- Директрисы: x = ±a/e
- Асимптоты: y = ±(b/a)x

**Парабола:**
y² = 2px

- Фокус: (p/2, 0)
- Директриса: x = −p/2
- Параметр: p

# VII. Формы

**Линейная форма:**  
f(x) = Σ a_i x_i

**Билинейная:**  
B(x,y) = xᵀAy

**Квадратичная:**  
Q(x) = xᵀAx

**Критерий Сильвестра:**
- Положительно определена ⇔ все главные миноры > 0
- Отрицательно определена ⇔ главные миноры чередуются: Δ₁ < 0, Δ₂ > 0, Δ₃ < 0, ...

**Преобразование при смене базиса:**
- Линейная форма: a' = Cᵀa
- Билинейная/квадратичная: A' = CᵀAC

**Симметричная билинейная форма:** B(x,y) = B(y,x), т.е. A = Aᵀ
**Кососимметричная:** B(x,y) = −B(y,x), т.е. A = −Aᵀ

**Разложение:** B = B_сим + B_кос
- B_сим(x,y) = ½(B(x,y) + B(y,x))
- B_кос(x,y) = ½(B(x,y) − B(y,x))

# VIII. Поверхности второго порядка

**Общее уравнение:**  
Ax²+By²+Cz²+2Dxy+2Exz+2Fyz+2Gx+2Hy+2Iz+J=0

**Канонические виды:**

**Эллипсоид:**
x²/a² + y²/b² + z²/c² = 1

**Однополостный гиперболоид:**
x²/a² + y²/b² − z²/c² = 1

**Двуполостный гиперболоид:**
−x²/a² − y²/b² + z²/c² = 1

**Коническая поверхность:**
x²/a² + y²/b² − z²/c² = 0

**Эллиптический параболоид:**
x²/a² + y²/b² = 2z

**Гиперболический параболоид (седло):**
x²/a² − y²/b² = 2z

---

## Цилиндрические поверхности

**Эллиптический цилиндр:**
x²/a² + y²/b² = 1

**Гиперболический цилиндр:**
x²/a² − y²/b² = 1

**Параболический цилиндр:**
y² = 2px

---

## Приведение к каноническому виду

1. Записать матрицу квадратичной формы A
2. Найти собственные значения: det(A − λE) = 0
3. Найти собственные векторы
4. Составить ортогональную матрицу перехода P
5. Новые координаты: x' = Pᵀx
6. Каноническая форма: λ₁x'² + λ₂y'² + λ₃z'² + ... = const
