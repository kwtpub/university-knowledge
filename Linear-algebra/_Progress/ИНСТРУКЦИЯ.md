# 🎓 Инструкция по использованию CLI для подготовки к экзамену

## 📦 Установка и запуск

### Первый запуск
```bash
cd Linear-algebra/_Progress
npm install
npm run build
npm start
```

### Последующие запуски
```bash
cd Linear-algebra/_Progress
npm start
```

## 🎯 Режимы работы

### 1. Обычный режим
- Случайный выбор билетов из базы
- Отслеживание прогресса ответов
- Статистика по всем билетам

### 2. Режим обучения по теме
- Выбор конкретной темы для изучения
- Последовательное изучение всех билетов темы
- Автоматическое создание файлов прогресса в Markdown
- Отдельная база данных для отслеживания обучения

## 📚 Темы билетов

1. **Матрицы и определители** (Билеты 1-6)
2. **Системы линейных уравнений** (Билеты 7-10)
3. **Линейные пространства** (Билеты 11-16)
4. **Линейные отображения и преобразования** (Билеты 17-24)
5. **Векторы и системы координат** (Билеты 25-31)
6. **Аналитическая геометрия** (Билеты 32-41)
7. **Формы** (Билеты 42-45)
8. **Поверхности второго порядка** (Билеты 46-50)

## 📊 Оценка понимания

После каждого билета оцените своё понимание:
- ✅ **ХОРОШО** - материал полностью понятен
- ⚠️ **СРЕДНЕ** - есть пробелы в понимании
- ❌ **ПЛОХО** - материал не понятен
- ❓ **НИКАК** - впервые столкнулись с темой

```
Linear-algebra/_Progress/

└── ИНСТРУКЦИЯ.md
```

## 🔄 Обновление кода

После внесения изменений в TypeScript код:
```bash
npm run build
```

## 💡 Рекомендации по подготовке

1. **Начните с режима обучения** - пройдите все темы последовательно
2. **Используйте файлы прогресса** - они покажут ваши слабые места
3. **Повторяйте билеты с оценкой "ПЛОХО" и "СРЕДНЕ"**
4. **Используйте обычный режим** для финальной проверки знаний
