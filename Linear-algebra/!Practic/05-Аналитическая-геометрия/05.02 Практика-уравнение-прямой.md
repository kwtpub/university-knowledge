# Практика 05.02 — уравнение прямой в пространстве

**Коротко**
**Каноническое уравнение прямой**:
$$\frac{x - x_0}{l} = \frac{y - y_0}{m} = \frac{z - z_0}{n}$$
где $M_0(x_0, y_0, z_0)$ — точка на прямой, $\vec{s} = (l, m, n)$ — направляющий вектор.

**Параметрическое уравнение прямой**:
$$\begin{cases}
x = x_0 + lt \\
y = y_0 + mt \\
z = z_0 + nt
\end{cases}$$
где $t \in \mathbb{R}$ — параметр.

**Векторное уравнение прямой**:
$$\vec{r} = \vec{r}_0 + t\vec{s}$$

**Прямая как пересечение двух плоскостей**:
$$\begin{cases}
A_1 x + B_1 y + C_1 z + D_1 = 0 \\
A_2 x + B_2 y + C_2 z + D_2 = 0
\end{cases}$$

**Направляющий вектор** прямой — пересечения плоскостей:
$$\vec{s} = \vec{n}_1 \times \vec{n}_2$$

**Важные формулы**:
- **Расстояние от точки** $M_1$ до прямой, проходящей через $M_0$ с направляющим вектором $\vec{s}$:
  $$\rho = \frac{|\vec{M_0M_1} \times \vec{s}|}{|\vec{s}|}$$

- **Угол между прямыми**:
  $$\cos \varphi = \frac{|l_1 l_2 + m_1 m_2 + n_1 n_2|}{\sqrt{l_1^2 + m_1^2 + n_1^2} \sqrt{l_2^2 + m_2^2 + n_2^2}}$$

- **Угол между прямой и плоскостью**:
  $$\sin \varphi = \frac{|Al + Bm + Cn|}{\sqrt{A^2 + B^2 + C^2} \sqrt{l^2 + m^2 + n^2}}$$

- **Условие параллельности прямых**: $\frac{l_1}{l_2} = \frac{m_1}{m_2} = \frac{n_1}{n_2}$

- **Условие перпендикулярности прямых**: $l_1 l_2 + m_1 m_2 + n_1 n_2 = 0$

**Примеры с разбором**

Пример 1.
Составить уравнение прямой, проходящей через точку $M(1, 2, 3)$ параллельно вектору $\vec{s} = (2, -1, 3)$.

Решение:
Каноническое уравнение:
$$\frac{x - 1}{2} = \frac{y - 2}{-1} = \frac{z - 3}{3}$$

Параметрическое:
$$\begin{cases}
x = 1 + 2t \\
y = 2 - t \\
z = 3 + 3t
\end{cases}$$

Пример 2.
Составить уравнение прямой, проходящей через точки $A(1, 2, 3)$ и $B(4, 5, 6)$.

Решение:
1. Направляющий вектор: $\vec{AB} = (4-1, 5-2, 6-3) = (3, 3, 3)$ или упрощённо $(1, 1, 1)$

2. Каноническое уравнение:
   $$\frac{x - 1}{1} = \frac{y - 2}{1} = \frac{z - 3}{1}$$
   или
   $$\frac{x - 1}{3} = \frac{y - 2}{3} = \frac{z - 3}{3}$$

Пример 3.
Найти точку пересечения прямой $\frac{x}{1} = \frac{y}{2} = \frac{z}{3}$ с плоскостью $x + y + z = 6$.

Решение:
1. Параметрическое уравнение прямой:
   $$x = t, \quad y = 2t, \quad z = 3t$$

2. Подставляем в уравнение плоскости:
   $$t + 2t + 3t = 6$$
   $$6t = 6$$
   $$t = 1$$

3. Точка пересечения:
   $$M(1, 2, 3)$$

Пример 4.
Найти расстояние от точки $M(1, 0, 0)$ до прямой $\frac{x}{1} = \frac{y}{0} = \frac{z}{0}$ (ось $Ox$).

Решение:
1. Точка на прямой: $M_0(0, 0, 0)$
2. Направляющий вектор: $\vec{s} = (1, 0, 0)$
3. $\vec{M_0M} = (1, 0, 0)$
4. $\vec{M_0M} \times \vec{s} = (1, 0, 0) \times (1, 0, 0) = (0, 0, 0)$
5. $\rho = \frac{|(0, 0, 0)|}{|(1, 0, 0)|} = 0$

Точка лежит на прямой!

Пример 5.
Найти угол между прямой $\frac{x}{1} = \frac{y}{0} = \frac{z}{0}$ и плоскостью $z = 0$.

Решение:
1. Направляющий вектор прямой: $\vec{s} = (1, 0, 0)$
2. Нормаль плоскости: $\vec{n} = (0, 0, 1)$
3. $\sin \varphi = \frac{|1 \cdot 0 + 0 \cdot 0 + 0 \cdot 1|}{\sqrt{1^2 + 0^2 + 0^2} \sqrt{0^2 + 0^2 + 1^2}} = 0$
4. $\varphi = 0°$

Прямая лежит в плоскости (или параллельна ей).
