## Как использовать практики

1. **Изучите теорию** в соответствующем разделе билетов к экзамену
2. **Прочитайте "Коротко"** — краткую выжимку формул и определений
3. **Разберите примеры** — каждый пример содержит подробное решение
4. **Решайте задания** — отмечайте чекбоксами выполненные задачи
5. **Проверяйте себя** — сверяйтесь с примерами и теорией

## Условные обозначения

- ✓ — свойство или важная формула
- ⟹ — следствие
- ⟺ — эквивалентность
- $\vec{a}$ — вектор
- $|\vec{a}|$ — длина (модуль) вектора
- $(\vec{a}, \vec{b})$ — скалярное произведение
- $\vec{a} \times \vec{b}$ — векторное произведение
- $(\vec{a}, \vec{b}, \vec{c})$ — смешанное произведение
