# Практика 06.02 — Приведение общего уравнения поверхности второго порядка к простейшему виду

---

## Алгоритм (напоминание)

1. Выделить полные квадраты по каждой переменной
2. Сделать замену переменных (сдвиг)
3. Привести к каноническому виду
4. Определить тип поверхности

Если есть произведения $xy, xz, yz$ — сначала поворот через собственные значения матрицы.

---

## Задание 1

Привести к каноническому виду и определить тип поверхности:

$$x^2 + y^2 + z^2 - 4x + 6y - 2z + 5 = 0$$

<details>
<summary>Решение</summary>

**Шаг 1.** Произведений $xy, xz, yz$ нет → поворот не нужен. Выделяем полные квадраты:

По $x$:
$$x^2 - 4x = (x - 2)^2 - 4$$

По $y$:
$$y^2 + 6y = (y + 3)^2 - 9$$

По $z$:
$$z^2 - 2z = (z - 1)^2 - 1$$

**Шаг 2.** Подставляем:

$$(x-2)^2 - 4 + (y+3)^2 - 9 + (z-1)^2 - 1 + 5 = 0$$

$$(x-2)^2 + (y+3)^2 + (z-1)^2 = 9$$

**Шаг 3.** Замена: $x' = x - 2$, $y' = y + 3$, $z' = z - 1$:

$$x'^2 + y'^2 + z'^2 = 9$$

**Ответ:** **Сфера** с центром $(2, -3, 1)$ и радиусом $R = 3$.

</details>

## Задание 2

Привести к каноническому виду и определить тип поверхности:

$$4x^2 + y^2 - z^2 + 8x - 2y + 4z - 1 = 0$$

<details>
<summary>Решение</summary>

**Шаг 1.** Произведений $xy, xz, yz$ нет → поворот не нужен. Выделяем полные квадраты:

По $x$:
$$4x^2 + 8x = 4(x^2 + 2x) = 4(x + 1)^2 - 4$$

По $y$:
$$y^2 - 2y = (y - 1)^2 - 1$$

По $z$:
$$-z^2 + 4z = -(z^2 - 4z) = -(z - 2)^2 + 4$$

**Шаг 2.** Подставляем:

$$4(x+1)^2 - 4 + (y-1)^2 - 1 - (z-2)^2 + 4 - 1 = 0$$

$$4(x+1)^2 + (y-1)^2 - (z-2)^2 = 2$$

Делим на 2:

$$\frac{(x+1)^2}{1/2} + \frac{(y-1)^2}{2} - \frac{(z-2)^2}{2} = 1$$

**Шаг 3.** Замена: $x' = x + 1$, $y' = y - 1$, $z' = z - 2$:

$$\frac{x'^2}{1/2} + \frac{y'^2}{2} - \frac{z'^2}{2} = 1$$

Или:

$$2x'^2 + \frac{y'^2}{2} - \frac{z'^2}{2} = 1$$

**Ответ:** **Однополостный гиперболоид** с центром $(-1, 1, 2)$ и полуосями $a = \frac{1}{\sqrt{2}}$, $b = \sqrt{2}$, $c = \sqrt{2}$.

Каноническая форма: $\dfrac{x'^2}{a^2} + \dfrac{y'^2}{b^2} - \dfrac{z'^2}{c^2} = 1$ — два «$+$» и один «$-$», правая часть $= 1$ → однополостный гиперболоид.

</details>

---

## Как определить тип по каноническому виду

После приведения смотрим на знаки и правую часть:

| Знаки при квадратах | Правая часть | Тип |
|---------------------|-------------|-----|
| $+, +, +$ | $= 1$ | Эллипсоид |
| $+, +, +$ | $= 0$ | Точка (вырожденный) |
| $+, +, -$ | $= 1$ | Однополостный гиперболоид |
| $+, +, -$ | $= -1$ | Двуполостный гиперболоид |
| $+, +, -$ | $= 0$ | Конус |
| $+, +$ и линейный $z$ | $= 2z$ | Эллиптический параболоид |
| $+, -$ и линейный $z$ | $= 2z$ | Гиперболический параболоид |
| $+, +$ (нет третьей) | $= 1$ | Эллиптический цилиндр |
| $+, -$ (нет третьей) | $= 1$ | Гиперболический цилиндр |
| $+$ и линейный (нет третьей) | $= 2py$ | Параболический цилиндр |
