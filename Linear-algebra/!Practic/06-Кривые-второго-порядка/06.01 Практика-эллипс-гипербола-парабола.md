# Практика 06.01 — эллипс, гипербола, парабола

**Коротко**

## Эллипс

**Каноническое уравнение**:
$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1, \quad a > b > 0$$

**Элементы**:
- $a$ — большая полуось (по $Ox$)
- $b$ — малая полуось (по $Oy$)
- $c = \sqrt{a^2 - b^2}$ — фокусное расстояние
- Фокусы: $F_1(-c, 0)$, $F_2(c, 0)$
- Эксцентриситет: $\varepsilon = \frac{c}{a} < 1$
- Директрисы: $x = \pm \frac{a}{\varepsilon} = \pm \frac{a^2}{c}$

**Определение**: $|MF_1| + |MF_2| = 2a$ для любой точки $M$ на эллипсе.

## Гипербола

**Каноническое уравнение**:
$$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$$

**Элементы**:
- $a$ — действительная полуось
- $b$ — мнимая полуось
- $c = \sqrt{a^2 + b^2}$ — фокусное расстояние
- Фокусы: $F_1(-c, 0)$, $F_2(c, 0)$
- Эксцентриситет: $\varepsilon = \frac{c}{a} > 1$
- Директрисы: $x = \pm \frac{a^2}{c}$
- Асимптоты: $y = \pm \frac{b}{a}x$

**Определение**: $||MF_1| - |MF_2|| = 2a$ для любой точки $M$ на гиперболе.

## Парабола

**Каноническое уравнение**:
$$y^2 = 2px, \quad p > 0$$

**Элементы**:
- $p$ — параметр параболы
- Фокус: $F(\frac{p}{2}, 0)$
- Директриса: $x = -\frac{p}{2}$
- Вершина: $O(0, 0)$

**Определение**: $|MF| = \rho(M, \text{директриса})$ для любой точки $M$ на параболе.

**Эксцентриситет**: $\varepsilon = 1$

---

**Примеры с разбором**

Пример 1 (Эллипс).
Найти фокусы, эксцентриситет и уравнения директрис эллипса $\frac{x^2}{25} + \frac{y^2}{9} = 1$.

Решение:
1. $a^2 = 25$ ⟹ $a = 5$
2. $b^2 = 9$ ⟹ $b = 3$
3. $c = \sqrt{a^2 - b^2} = \sqrt{25 - 9} = \sqrt{16} = 4$
4. Фокусы: $F_1(-4, 0)$, $F_2(4, 0)$
5. $\varepsilon = \frac{c}{a} = \frac{4}{5} = 0.8$
6. Директрисы: $x = \pm \frac{a^2}{c} = \pm \frac{25}{4} = \pm 6.25$

Пример 2 (Гипербола).
Найти асимптоты гиперболы $\frac{x^2}{16} - \frac{y^2}{9} = 1$.

Решение:
1. $a^2 = 16$ ⟹ $a = 4$
2. $b^2 = 9$ ⟹ $b = 3$
3. Асимптоты: $y = \pm \frac{b}{a}x = \pm \frac{3}{4}x$

Пример 3 (Парабола).
Найти фокус и директрису параболы $y^2 = 8x$.

Решение:
1. $2p = 8$ ⟹ $p = 4$
2. Фокус: $F(\frac{p}{2}, 0) = F(2, 0)$
3. Директриса: $x = -\frac{p}{2} = -2$

Пример 4.
Привести к каноническому виду уравнение $4x^2 + 9y^2 = 36$.

Решение:
Разделим на 36:
$$\frac{4x^2}{36} + \frac{9y^2}{36} = 1$$
$$\frac{x^2}{9} + \frac{y^2}{4} = 1$$

Это эллипс с $a = 3$, $b = 2$.

Пример 5.
Определить тип кривой $x^2 - y^2 = 1$.

Решение:
Это гипербола с $a = b = 1$.

Асимптоты: $y = \pm x$

Фокусы: $c = \sqrt{1 + 1} = \sqrt{2}$, т.е. $F_1(-\sqrt{2}, 0)$, $F_2(\sqrt{2}, 0)$
