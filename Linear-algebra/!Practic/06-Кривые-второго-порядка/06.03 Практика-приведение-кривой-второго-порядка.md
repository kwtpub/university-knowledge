# Практика 06.03 — Приведение общего уравнения кривой второго порядка к простейшему виду

---

## Алгоритм (напоминание)

1. Есть $xy$? → поворот (собственные значения или $\tan 2\varphi = \frac{2a_{12}}{a_{11} - a_{22}}$)
2. Выделить полные квадраты (сдвиг)
3. Определить тип кривой

**Быстрая проверка** — инварианты:
- $I_2 > 0$ → эллипс, $I_2 < 0$ → гипербола, $I_2 = 0$ → парабола

---

## Задание 1

Привести к каноническому виду и определить тип кривой:

$$9x^2 + 4y^2 - 18x + 16y - 11 = 0$$

- [ ] Решено

<details>
<summary>Решение</summary>

**Шаг 1.** Нет $xy$ → поворот не нужен.

**Шаг 2.** Выделяем полные квадраты:

По $x$:
$$9x^2 - 18x = 9(x^2 - 2x) = 9(x - 1)^2 - 9$$

По $y$:
$$4y^2 + 16y = 4(y^2 + 4y) = 4(y + 2)^2 - 16$$

**Шаг 3.** Подставляем:

$$9(x-1)^2 - 9 + 4(y+2)^2 - 16 - 11 = 0$$

$$9(x-1)^2 + 4(y+2)^2 = 36$$

Делим на 36:

$$\frac{(x-1)^2}{4} + \frac{(y+2)^2}{9} = 1$$

**Ответ:** **Эллипс** с центром $(1, -2)$, $a = 3$ (по $y$), $b = 2$ (по $x$).

</details>

## Задание 2

Привести к каноническому виду и определить тип кривой:

$$x^2 - 4y^2 + 6x + 8y - 3 = 0$$

- [ ] Решено

<details>
<summary>Решение</summary>

**Шаг 1.** Нет $xy$ → поворот не нужен.

**Шаг 2.** Выделяем полные квадраты:

По $x$:
$$x^2 + 6x = (x + 3)^2 - 9$$

По $y$:
$$-4y^2 + 8y = -4(y^2 - 2y) = -4(y - 1)^2 + 4$$

**Шаг 3.** Подставляем:

$$(x+3)^2 - 9 - 4(y-1)^2 + 4 - 3 = 0$$

$$(x+3)^2 - 4(y-1)^2 = 8$$

Делим на 8:

$$\frac{(x+3)^2}{8} - \frac{(y-1)^2}{2} = 1$$

**Ответ:** **Гипербола** с центром $(-3, 1)$, $a = 2\sqrt{2}$, $b = \sqrt{2}$.

</details>

## Задание 3

Привести к каноническому виду и определить тип кривой:

$$y^2 + 4x - 6y + 1 = 0$$

- [ ] Решено

<details>
<summary>Решение</summary>

**Шаг 1.** Нет $xy$, нет $x^2$ → поворот не нужен. Квадрат только у $y$ → будет парабола.

**Шаг 2.** Выделяем полный квадрат по $y$:

$$y^2 - 6y = (y - 3)^2 - 9$$

**Шаг 3.** Подставляем:

$$(y - 3)^2 - 9 + 4x + 1 = 0$$

$$(y - 3)^2 = -4x + 8$$

$$(y - 3)^2 = -4(x - 2)$$

**Шаг 4.** Замена: $x' = x - 2$, $y' = y - 3$:

$$y'^2 = -4x'$$

Это $y'^2 = 2px'$ с $2p = -4$, т.е. $p = 2$ и знак минус.

**Ответ:** **Парабола**, раскрывается **влево**, вершина $(2, 3)$, $p = 2$.

Фокус: $(2 - 1, 3) = (1, 3)$. Директриса: $x = 3$.

</details>

## Задание 4 (с поворотом)

Привести к каноническому виду и определить тип кривой:

$$5x^2 + 4xy + 8y^2 - 32 = 0$$

- [ ] Решено

<details>
<summary>Решение</summary>

**Шаг 1.** Есть $4xy$ → нужен поворот.

Матрица квадратичной части:

$$A = \begin{pmatrix} 5 & 2 \\ 2 & 8 \end{pmatrix}$$

**Шаг 2.** Собственные значения:

$$\det(A - \lambda E) = (5 - \lambda)(8 - \lambda) - 4 = \lambda^2 - 13\lambda + 36 = 0$$

$$\lambda = \frac{13 \pm \sqrt{169 - 144}}{2} = \frac{13 \pm 5}{2}$$

$\lambda_1 = 4$, $\lambda_2 = 9$.

**Шаг 3.** После поворота квадратичная часть становится:

$$4x'^2 + 9y'^2$$

Линейных членов нет, поэтому сдвиг не нужен. Уравнение:

$$4x'^2 + 9y'^2 = 32$$

Делим на 32:

$$\frac{x'^2}{8} + \frac{y'^2}{32/9} = 1$$

**Ответ:** **Эллипс** с $a = 2\sqrt{2}$, $b = \frac{4\sqrt{2}}{3}$.

**Проверка инвариантами:** $I_2 = \det A = 40 - 4 = 36 > 0$ → эллипс ✓

</details>

---

## Как определить тип — шпаргалка

| Признак | Тип |
|---------|-----|
| $I_2 > 0$, оба квадрата с одинаковым знаком | Эллипс |
| $I_2 < 0$, квадраты с разными знаками | Гипербола |
| $I_2 = 0$, квадрат только у одной переменной | Парабола |
| Равенство $= 0$ вместо $= 1$ | Вырожденный случай (точка, прямые, пустое множество) |
