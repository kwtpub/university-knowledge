# Лекция 1: Строки (12.02.2026)

## 1. Базовые определения

Строка — это упорядоченная последовательность символов (байтов памяти).

В C/C++ рассмотрены два подхода к работе со строками:

1. C-строки (массив `char` + маркер конца `'\0'`)
2. `string` (строковый тип C++)

## 2. C-строки

- Строка хранится в массиве символов, каждый символ занимает отдельный элемент массива.
- Индексация начинается с `0`.
- Символ `'\0'` — маркер конца строки (нулевой байт).
- Пустая строка содержит только `'\0'` (`NULL STRING`).
- При объявлении буфера под C-строку нужно оставлять место под терминатор (`+1` к полезной длине).

## 3. Особенности и ограничения

- Массив `char[]` имеет фиксированный размер.
- Если буфер больше текста, часть памяти не используется.
- Если буфер меньше текста, возникают ошибки/переполнения.
- Большинство функций C-строк обрабатывают символы до тех пор, пока не встретят `'\0'`.

## 4. Пример: не строка и строка

```cpp
char Dog[4] = {'m', 'e', 'a', 't'};       // не строка (нет '\0')
char Cat[5] = {'f', 'i', 's', 'h', '\0'}; // строка

cout << "NOT String Dog: " << Dog << endl; // вывод до случайного '\0'
cout << "String Cat: " << Cat << endl;     // fish
```

## 5. Строковые литералы и символьные константы

```cpp
char City[] = "МОСКВА";
```

Строковые литералы автоматически содержат завершающий `'\0'`, его нужно учитывать в размере.

Разница:

- `'f'` — символьная константа, длина 1
- `"f"` — строковая константа, длина 2 (`'f' + '\0'`)

```cpp
char simb = 'f';  // OK
char simb = "f";  // ошибка
```

## 6. Пример с изменением строки

```cpp
char myName[] = "C++ programmer"; // 14 + 1

cout << "My Name is " << myName << "\n";
cout << "sizeof(myName) = " << sizeof(myName) << "\n\n";
cout << "Your initial is: " << myName[0] << "\n\n";

myName[3] = '\0';
cout << "My short Name is: " << myName << endl; // C++

myName[3] = ' ';
cout << "My reset Name is: " << myName << endl; // C++ programmer

myName[0] = '\0';
cout << "My Empty Name is " << myName << ".\n";

myName[0] = 'C';
cout << "My Old Name is " << myName << "\n";    // C++ programmer
```

## 7. Ввод строк

```cpp
cin >> Name;
cin >> Fpm;
```

Такой ввод режет строку по пробелу (например, ФИО целиком не прочитается).

Для чтения строки с пробелами:

```cpp
cin.getline(Name, 20); // максимум 19 символов + '\0'
```

## 8. Ручной подсчет длины строки

```cpp
const int MAXLEN = 100;
char str[MAXLEN];
cin.getline(str, MAXLEN);

int i = 0;
while (str[i] != '\0') {
    i++;
}

```
