# Решение задания 3 (вариант 4)

## Постановка
Нужно обработать символ и две строки слов из файла:
1. вывести слова, которые встречаются в обеих строках;
2. определить, в какой строке заданный символ встречается чаще.

## Формат входных данных
Входной файл содержит:
1. `symbol` - один символ;
2. `line1` - первая строка слов;
3. `line2` - вторая строка слов.

Пример:
```text
a
cat dog sky cat
dog sun cat
```

## Формат выходных данных
Программа выводит:
1. считанные данные;
2. список общих слов;
3. результат сравнения частоты символа в строках.

## Идея решения
Решение выполнено на основе работы со строками из лекции (`char[]`, `getline`, посимвольный разбор, поиск `\0`).

Процедура `PrintCommonWords(firstLine, secondLine)`:
1. разбирает первую строку на слова;
2. проверяет наличие каждого слова во второй строке;
3. выводит только уникальные общие слова в терминал (`std::cout`).

Процедура `PrintLineWithMoreSymbol(firstLine, secondLine, symbol)`:
1. считает число вхождений символа в каждой строке;
2. выводит в терминал, где символ встречается чаще (или что количества равны).

Вспомогательные процедуры:
1. `ReadNextWord` - извлекает следующее слово из `char[]`;
2. `WordExistsInLine` - проверяет наличие слова в строке;
3. `CountSymbolInLine` - подсчитывает вхождения символа.

## Ограничения и реализация
1. Глобальные переменные не используются.
2. Обмен данными с подпрограммами идет только через параметры.
3. Для чтения применяются средства C++ (`ifstream`, `getline`).
4. Каждый набор исходных данных хранится в отдельном файле (`lab-1/tests/input*.txt`).

## Что использовано не из лекции
1. Проверка корректности открытия и чтения файла (обработка ошибок ввода).
2. Исключение повторного вывода одинаковых общих слов через отдельный массив уже выведенных слов.
3. Практика разбиения задачи на несколько небольших процедур (декомпозиция алгоритма).
