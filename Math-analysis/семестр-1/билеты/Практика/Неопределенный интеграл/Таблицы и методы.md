# Таблицы и методы по теме «Неопределенный интеграл»

## 1. Таблица простейших интегралов

Нужно знать наизусть:

- $\int x^\alpha\,dx = \dfrac{x^{\alpha+1}}{\alpha+1}+C,\quad \alpha\ne -1$
- $\int \dfrac{dx}{x} = \ln|x|+C$
- $\int e^x\,dx = e^x+C$
- $\int a^x\,dx = \dfrac{a^x}{\ln a}+C$
- $\int \sin x\,dx = -\cos x+C$
- $\int \cos x\,dx = \sin x+C$
- $\int \dfrac{dx}{\cos^2 x} = tg x+C$
- $\int \dfrac{dx}{\sin^2 x} = -ctg x+C$
- $\int \tan x\,dx = -\ln|\cos x|+C$
- $\int \cot x\,dx = \ln|\sin x|+C$
- $\int \dfrac{dx}{1+x^2} = \arctan x+C$
- $\int \dfrac{dx}{\sqrt{1-x^2}} = \arcsin x+C$
- $\int \dfrac{dx}{a^2+x^2} = \dfrac{1}{a}\arctan\dfrac{x}{a}+C$
- $\int \dfrac{dx}{x^2-a^2} = \dfrac{1}{2a}\ln\left|\dfrac{x+a}{x-a}\right|+C$
- $\int \dfrac{dx}{\sqrt{x^2\pm a^2}} = \ln\left|x+\sqrt{x^2\pm a^2}\right|+C$

## 2. Замена переменной и по частям

Формула замены:

$$
\int f(\varphi(t))\varphi'(t)\,dt = \int f(u)\,du.
$$

Формула по частям:

$$
\int u\,dv = uv-\int v\,du.
$$

Что помнить:

- в замене ищем внутреннюю функцию и её производную;
- по частям выбираем $u$ так, чтобы оно упрощалось при дифференцировании;
- $dv$ берём то, что легко интегрируется.

## 3. Простейшие дроби

Шаблоны разложения:

- $\dfrac{A}{x-a}$
- $\dfrac{A}{(x-a)^k}$
- $\dfrac{Ax+B}{x^2+px+q}$
- $\dfrac{Ax+B}{(x^2+px+q)^k}$

Алгоритм:

1. Разложить знаменатель.
2. Написать шаблон.
3. Умножить на знаменатель.
4. Найти коэффициенты.
5. Интегрировать по таблице.

## 4. Иррациональные интегралы

### Билет 45

Если под корнем стоит дробь вида

$$
\sqrt[n]{\frac{ax+b}{cx+d}},
$$

то делаем замену

$$
t=\sqrt[n]{\frac{ax+b}{cx+d}}.
$$

### Билет 46. Подстановки Эйлера

- если $a>0$, то $\sqrt{ax^2+bx+c}=t\pm x\sqrt a$
- если $a<0,\ c>0$, то $\sqrt{ax^2+bx+c}=tx\pm \sqrt c$
- если есть два действительных корня, то $\sqrt{ax^2+bx+c}=t(x-x_1)$

### Билет 47. Тригонометрические подстановки

- $\sqrt{\alpha^2-u^2}$: $u=\alpha\sin t$
- $\sqrt{\alpha^2+u^2}$: $u=\alpha\tan t$
- $\sqrt{u^2-\alpha^2}$: $u=\dfrac{\alpha}{\cos t}$

### Билет 48. Биномиальный дифференциал

Для

$$
\int x^m(a+bx^n)^p\,dx
$$

случаи Чебышёва:

- $p$ целое
- $\dfrac{m+1}{n}$ целое
- $\dfrac{m+1}{n}+p$ целое

Если выполняется один из них, интеграл выражается через элементарные функции.

## 5. Тригонометрические интегралы

### Билет 49. Универсальная подстановка

$$
t=\tan\frac{x}{2}
$$

Тогда:

- $\sin x = \dfrac{2t}{1+t^2}$
- $\cos x = \dfrac{1-t^2}{1+t^2}$
- $dx = \dfrac{2\,dt}{1+t^2}$

### Билет 50. Частные случаи

- нечётная по $\cos x$ -> $t=\sin x$
- нечётная по $\sin x$ -> $t=\cos x$
- чётная по обоим -> $t=\tan x$

### Билет 51. Полезные формулы

Понижение степени:

- $\sin^2 x = \dfrac{1-\cos 2x}{2}$
- $\cos^2 x = \dfrac{1+\cos 2x}{2}$
- $\sin x\cos x = \dfrac{\sin 2x}{2}$

Формулы произведения:

- $\sin\alpha\cos\beta = \dfrac{1}{2}\bigl[\sin(\alpha+\beta)+\sin(\alpha-\beta)\bigr]$
- $\sin\alpha\sin\beta = \dfrac{1}{2}\bigl[\cos(\alpha-\beta)-\cos(\alpha+\beta)\bigr]$
- $\cos\alpha\cos\beta = \dfrac{1}{2}\bigl[\cos(\alpha-\beta)+\cos(\alpha+\beta)\bigr]$
