## 1. Найти предел

$$
\lim_{n\to\infty}\frac{3^{n-2}-2^n}{3^n+2^{n+10}}.
$$

**Решение:**

Выносим константы из степеней: $3^{n-2} = \dfrac{3^n}{9}$, $\quad 2^{n+10} = 1024 \cdot 2^n$

$$\frac{\dfrac{3^n}{9} - 2^n}{3^n + 1024\cdot 2^n}$$

Делим числитель и знаменатель на $3^n$:

$$\frac{\dfrac{1}{9} - \left(\dfrac{2}{3}\right)^n}{1 + 1024\cdot\left(\dfrac{2}{3}\right)^n}$$

При $n\to\infty$: $\left(\dfrac{2}{3}\right)^n \to 0$, так как $\dfrac{2}{3} < 1$.

$$= \frac{\dfrac{1}{9} - 0}{1 + 0} = \boxed{\dfrac{1}{9}}$$

## 2. Найти пределы функции

а)

$$
\lim_{x\to 0}\frac{1-\sqrt{2-\cos x}}{2-2\cos x}.
$$

**Решение:**

При $x\to 0$ получаем $\dfrac{0}{0}$. Умножаем на сопряжённое $(1 + \sqrt{2-\cos x})$:

$$\frac{(1-\sqrt{2-\cos x})(1+\sqrt{2-\cos x})}{(2-2\cos x)(1+\sqrt{2-\cos x})}$$

Числитель по формуле разности квадратов:

$$1-(2-\cos x) = \cos x - 1 = -(1-\cos x)$$

Получаем:

$$\frac{-(1-\cos x)}{2(1-\cos x)(1+\sqrt{2-\cos x})}$$

Сокращаем $(1-\cos x)$:

$$\frac{-1}{2(1+\sqrt{2-\cos x})}$$

При $x\to 0$: $\cos x \to 1$, значит $\sqrt{2-\cos x} \to 1$:

$$\frac{-1}{2(1+1)} = \boxed{-\dfrac{1}{4}}$$

б)

$$
\lim_{x\to 0}\frac{x\cdot \arcsin 2x}{1-\cos 4x}.
$$

**Решение:**

Применяем эквивалентные бесконечно малые при $x\to 0$:
- $\arcsin 2x \sim 2x$
- $1 - \cos 4x \sim \dfrac{(4x)^2}{2} = 8x^2$

$$\frac{x \cdot 2x}{8x^2} = \frac{2x^2}{8x^2} = \boxed{\dfrac{1}{4}}$$

## 3. Найти производные

а)

$$
y=\frac12\,arctg\frac{e^x-3}{2}.
$$

**Решение:**

Сложная функция: $(arctg u)' = \dfrac{1}{1+u^2} \cdot u'$, где $u = \dfrac{e^x-3}{2}$, $\quad u' = \dfrac{e^x}{2}$

$$y' = \frac{1}{2} \cdot \frac{1}{1+\left(\dfrac{e^x-3}{2}\right)^2} \cdot \frac{e^x}{2}$$

Упрощаем знаменатель:

$$1 + \frac{(e^x-3)^2}{4} = \frac{4+(e^x-3)^2}{4}$$

$$y' = \frac{1}{2} \cdot \frac{4}{4+(e^x-3)^2} \cdot \frac{e^x}{2} = \boxed{\dfrac{e^x}{4+(e^x-3)^2}}$$

б)

$$
x\cos y+y\cos x=0.
$$

**Решение:**

Неявная функция — дифференцируем обе части по $x$, считая $y = y(x)$:

$$(x\cos y)' = \cos y - x\sin y \cdot y'$$

$$(y\cos x)' = y'\cos x - y\sin x$$

Сумма равна нулю:

$$\cos y - x\sin y \cdot y' + y'\cos x - y\sin x = 0$$

Собираем $y'$:

$$y'(\cos x - x\sin y) = y\sin x - \cos y$$

$$\boxed{y' = \dfrac{y\sin x - \cos y}{\cos x - x\sin y}}$$

## 4. Найти $y^{(n)}$ функции

$$
y=\frac{x}{6(x+1)}.
$$

**Решение:**

Упрощаем дробь:

$$y = \frac{1}{6}\cdot\frac{x}{x+1} = \frac{1}{6}\cdot\frac{(x+1)-1}{x+1} = \frac{1}{6}\left(1 - \frac{1}{x+1}\right)$$

Единица при дифференцировании пропадает. Дифференцируем $\dfrac{1}{x+1}$ несколько раз и ищем закономерность:

$$y' = \frac{1}{6}\cdot\frac{1}{(x+1)^2}, \quad y'' = \frac{1}{6}\cdot\frac{-2}{(x+1)^3}, \quad y''' = \frac{1}{6}\cdot\frac{6}{(x+1)^4}, \quad y'''' = \frac{1}{6}\cdot\frac{-24}{(x+1)^5}$$

Закономерность:
- числитель: $1, 2, 6, 24, \ldots = n!$
- знак чередуется: $+, -, +, - \ldots = (-1)^{n+1}$
- знаменатель: $(x+1)^{n+1}$

$$\boxed{y^{(n)} = \dfrac{(-1)^{n+1}\cdot n!}{6(x+1)^{n+1}}}$$

## 5. Исследовать функции и построить графики

а)

$$
y=e^{-x^2}.
$$

**Решение:**

**1. Область определения:** $x \in (-\infty, +\infty)$

**2. Чётность:** $f(-x) = e^{-(-x)^2} = e^{-x^2} = f(x)$ — функция **чётная**, симметрична относительно оси $Oy$

**3. Периодичность:** непериодическая

**4. Пересечения с осями:**
- С осью $Oy$: $f(0) = e^0 = 1$, точка $(0,\ 1)$
- С осью $Ox$: $e^{-x^2} > 0$ всегда — нулей нет

**5. Точки разрыва:** нет, функция непрерывна везде

**6. Экстремумы и монотонность:**

Берём производную по цепному правилу: внешняя $e^u$ даёт $e^u$, внутренняя $-x^2$ даёт $-2x$:

$$y' = -2x\cdot e^{-x^2}$$

Приравниваем к нулю: $e^{-x^2} \neq 0$ никогда, значит только $-2x = 0 \Rightarrow x = 0$

| Промежуток | Знак $y'$ | Монотонность |
|---|---|---|
| $(-\infty,\ 0)$ | $+$ | возрастает |
| $(0,\ +\infty)$ | $-$ | убывает |

$x = 0$ — **максимум**, $y(0) = 1$

**7. Перегибы и выпуклость:**

$$y'' = -2e^{-x^2} + 4x^2 e^{-x^2} = e^{-x^2}(4x^2 - 2)$$

$y'' = 0$ при $x = \pm\dfrac{1}{\sqrt{2}} \approx \pm 0.707$

| Промежуток | Знак $y''$ | Выпуклость |
|---|---|---|
| $\left(-\infty,\ -\dfrac{1}{\sqrt{2}}\right)$ | $+$ | вогнутая |
| $\left(-\dfrac{1}{\sqrt{2}},\ \dfrac{1}{\sqrt{2}}\right)$ | $-$ | выпуклая |
| $\left(\dfrac{1}{\sqrt{2}},\ +\infty\right)$ | $+$ | вогнутая |

Точки перегиба: $x = \pm\dfrac{1}{\sqrt{2}}$, $y = e^{-1/2} \approx 0.607$

**8. Асимптоты:**
- Вертикальных нет
- Горизонтальная: $\lim_{x\to\pm\infty} e^{-x^2} = 0$, то есть $y = 0$
- Наклонных нет

**9. График:** колокол с максимумом в $(0,\ 1)$, симметричный относительно $Oy$, прижимается к оси $Ox$ на бесконечности.

б)

$$
y=\frac{4-x^3}{x^2}.
$$

**Решение:**

**1. Область определения:** $x \neq 0$, то есть $x \in (-\infty, 0) \cup (0, +\infty)$

**2. Чётность:** $f(-x) = \dfrac{4+x^3}{x^2} \neq f(x)$ и $\neq -f(x)$ — ни чётная ни нечётная

**3. Периодичность:** непериодическая

**4. Пересечения с осями:**
- С осью $Oy$: $x=0$ не входит в область — пересечения нет
- С осью $Ox$: $4 - x^3 = 0 \Rightarrow x = \sqrt[3]{4} \approx 1.587$, точка $(\sqrt[3]{4},\ 0)$

**5. Точки разрыва:** $x = 0$ — разрыв второго рода, $\lim_{x\to 0} y = \infty$

**6. Экстремумы и монотонность:**

Упрощаем: $y = \dfrac{4}{x^2} - x = 4x^{-2} - x$

$$y' = -\frac{8}{x^3} - 1 = \frac{-8 - x^3}{x^3}$$

$y' = 0 \Rightarrow x^3 = -8 \Rightarrow x = -2$

| Промежуток | Знак $y'$ | Монотонность |
|---|---|---|
| $(-\infty,\ -2)$ | $-$ | убывает |
| $(-2,\ 0)$ | $+$ | возрастает |
| $(0,\ +\infty)$ | $-$ | убывает |

$x = -2$ — **минимум**, $y(-2) = \dfrac{4-(-8)}{4} = 3$, точка $(-2,\ 3)$

**7. Перегибы и выпуклость:**

$$y'' = \frac{24}{x^4} > 0 \text{ всегда}$$

Функция везде **вогнутая**, точек перегиба нет

**8. Асимптоты:**
- Вертикальная: $x = 0$
- Горизонтальной нет
- Наклонная: $k = \lim_{x\to\infty}\dfrac{y}{x} = \lim_{x\to\infty}\dfrac{4-x^3}{x^3} = -1$

$$b = \lim_{x\to\infty}(y - kx) = \lim_{x\to\infty}\left(\frac{4-x^3}{x^2} + x\right) = \lim_{x\to\infty}\frac{4}{x^2} = 0$$

Наклонная асимптота: $y = -x$

**9. График:** вертикальная асимптота $x=0$, наклонная $y=-x$, минимум $(-2,\ 3)$, ноль в $x = \sqrt[3]{4}$

## 6. Вычислить интегралы

а)

$$
\int \frac{1+\ln(x-1)}{x-1}\,dx.
$$

**Решение:**

Замена: $t = \ln(x-1)$, тогда $dt = \dfrac{dx}{x-1}$

$$\int (1+t)\,dt = t + \frac{t^2}{2} + C = \boxed{\ln(x-1) + \frac{\ln^2(x-1)}{2} + C}$$

---

б)

$$
\int \frac{x-4}{(x+1)(x-2)^2}\,dx.
$$

**Решение:**

Разложение на простые дроби:

$$\frac{x-4}{(x+1)(x-2)^2} = \frac{A}{x+1} + \frac{B}{x-2} + \frac{C}{(x-2)^2}$$

Умножаем на $(x+1)(x-2)^2$:

$$x-4 = A(x-2)^2 + B(x+1)(x-2) + C(x+1)$$

- $x = -1$: $-5 = 9A \Rightarrow A = -\dfrac{5}{9}$
- $x = 2$: $-2 = 3C \Rightarrow C = -\dfrac{2}{3}$
- $x = 0$: $-4 = 4A - 2B + C \Rightarrow B = \dfrac{5}{9}$

$$\boxed{-\frac{5}{9}\ln|x+1| + \frac{5}{9}\ln|x-2| + \frac{2}{3(x-2)} + C}$$

---

в)

$$
\int \frac{x\,dx}{\sqrt{x-1}}.
$$

**Решение:**

Замена: $t = \sqrt{x-1}$, тогда $x = t^2+1$, $dx = 2t\,dt$

$$\int \frac{(t^2+1)\cdot 2t\,dt}{t} = 2\int(t^2+1)\,dt = 2\left(\frac{t^3}{3} + t\right) + C$$

$$\boxed{\frac{2}{3}(x-1)^{3/2} + 2\sqrt{x-1} + C}$$
