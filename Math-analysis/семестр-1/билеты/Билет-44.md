## 44. Неопределённый интеграл от рациональной функции. Разложение правильных дробей на простейшие.

Для того, чтобы проинтегрировать рациональную дробь необходимо разложить её на элементарные дроби.

**Теорема:** Если $R(x) = \dfrac{Q(x)}{P(x)}$ — правильная рациональная дробь, знаменатель $P(x)$ которой представлен в виде произведения линейных и квадратичных множителей (отметим, что любой многочлен с действительными коэффициентами может быть представлен в таком виде):

$$P(x) = (x-a_1)^{\alpha_1} \cdot \ldots \cdot (x-a_k)^{\alpha_k} \cdot (x^2+p_1 x+q_1)^{\beta_1} \cdot \ldots \cdot (x^2+p_r x+q_r)^{\beta_r},$$

то эта дробь может быть разложена на элементарные по следующей схеме:

$$\frac{Q(x)}{P(x)} = \sum_{i=1}^{k}\sum_{j=1}^{\alpha_i} \frac{A_{ij}}{(x-a_i)^j} + \sum_{i=1}^{r}\sum_{j=1}^{\beta_i} \frac{M_{ij}x + N_{ij}}{(x^2+p_i x+q_i)^j},$$

где $A_{ij}$, $M_{ij}$, $N_{ij}$ — некоторые постоянные величины.

При интегрировании рациональных дробей прибегают к разложению исходной дроби на элементарные. Для нахождения величин $A_{ij}$, $M_{ij}$, $N_{ij}$ применяют так называемый **метод неопределённых коэффициентов**, суть которого состоит в том, что для того, чтобы два многочлена были тождественно равны, необходимо и достаточно, чтобы были равны коэффициенты при одинаковых степенях $x$.

---

## Объяснение на пальцах

**Правильная дробь** — это дробь где степень числителя меньше степени знаменателя. Например $\dfrac{1}{x^2-1}$ — правильная, а $\dfrac{x^3}{x^2-1}$ — нет.

**Идея метода:** любую сложную правильную дробь можно разбить на сумму простых дробей (типов I–IV из билета 43), каждую из которых уже легко интегрировать.

**Алгоритм:**
1. Разложить знаменатель на множители.
2. Написать шаблон разложения с неизвестными коэффициентами $A$, $B$, $C$...
3. Умножить обе части на знаменатель.
4. Найти коэффициенты — либо подстановкой удобных значений $x$, либо приравниванием коэффициентов при одинаковых степенях.
5. Интегрировать каждую простую дробь отдельно.

**Шаблоны разложения в зависимости от множителей знаменателя:**

| Множитель в знаменателе | Слагаемое в разложении |
|---|---|
| $(x - a)$ | $\dfrac{A}{x-a}$ |
| $(x - a)^2$ | $\dfrac{A}{x-a} + \dfrac{B}{(x-a)^2}$ |
| $(x - a)^3$ | $\dfrac{A}{x-a} + \dfrac{B}{(x-a)^2} + \dfrac{C}{(x-a)^3}$ |
| $(x^2+px+q)$ | $\dfrac{Ax+B}{x^2+px+q}$ |
| $(x^2+px+q)^2$ | $\dfrac{Ax+B}{x^2+px+q} + \dfrac{Cx+D}{(x^2+px+q)^2}$ |

## Примеры

### Пример 1 — простые различные корни

**Найти** $\displaystyle\int \frac{dx}{x^2-1}$.

**Шаг 1.** Раскладываем знаменатель на множители:

$$x^2 - 1 = (x-1)(x+1).$$

**Шаг 2.** Пишем шаблон разложения — на каждый простой множитель по одной дроби:

$$\frac{1}{(x-1)(x+1)} = \frac{A}{x-1} + \frac{B}{x+1}.$$

**Шаг 3.** Умножаем обе части на $(x-1)(x+1)$:

$$1 = A(x+1) + B(x-1).$$

**Шаг 4.** Находим коэффициенты подстановкой удобных значений $x$.

После умножения на знаменатель получили тождество $1 = A(x+1) + B(x-1)$, которое верно для **любого** $x$. Значит, можно подставить любое удобное значение. Удобнее всего подставлять корни знаменателя — тогда все неизвестные кроме одной сразу зануляются.

- Подставляем $x = 1$ — корень множителя $(x-1)$, слагаемое с $B$ обнуляется:

$$1 = A(1+1) + B(1-1) = 2A + 0 \Rightarrow A = \frac{1}{2}.$$

- Подставляем $x = -1$ — корень множителя $(x+1)$, слагаемое с $A$ обнуляется:

$$1 = A(-1+1) + B(-1-1) = 0 + (-2B) \Rightarrow B = -\frac{1}{2}.$$

**Шаг 5.** Подставляем найденные коэффициенты:

$$\frac{1}{x^2-1} = \frac{1/2}{x-1} - \frac{1/2}{x+1} = \frac{1}{2}\cdot\frac{1}{x-1} - \frac{1}{2}\cdot\frac{1}{x+1}.$$

**Шаг 6.** Интегрируем каждую дробь по типу I:

$$\int \frac{dx}{x^2-1} = \frac{1}{2}\ln|x-1| - \frac{1}{2}\ln|x+1| + C = \frac{1}{2}\ln\left|\frac{x-1}{x+1}\right| + C.$$

---

### Пример 2 — кратный корень

**Найти** $\displaystyle\int \frac{x+2}{x^2(x-1)}\,dx$.

**Шаг 1.** Знаменатель уже разложен: $x^2(x-1)$. Множитель $x$ входит дважды (кратный корень), $x-1$ входит один раз.

**Шаг 2.** Шаблон — на кратный корень $x^2$ пишем две дроби:

$$\frac{x+2}{x^2(x-1)} = \frac{A}{x} + \frac{B}{x^2} + \frac{C}{x-1}.$$

**Шаг 3.** Умножаем обе части на $x^2(x-1)$:

$$x+2 = Ax(x-1) + B(x-1) + Cx^2.$$

**Шаг 4.** Находим коэффициенты:

- Подставляем $x = 0$: $\quad 2 = 0 + B(-1) + 0 \Rightarrow B = -2$.
- Подставляем $x = 1$: $\quad 3 = 0 + 0 + C \Rightarrow C = 3$.
- Приравниваем коэффициенты при $x^2$: $\quad 0 = A + C \Rightarrow A = -3$.

**Шаг 5.** Разложение:

$$\frac{x+2}{x^2(x-1)} = -\frac{3}{x} - \frac{2}{x^2} + \frac{3}{x-1}.$$

**Шаг 6.** Интегрируем:

$$\int \frac{x+2}{x^2(x-1)}\,dx = -3\ln|x| + \frac{2}{x} + 3\ln|x-1| + C = 3\ln\left|\frac{x-1}{x}\right| + \frac{2}{x} + C.$$

---

### Пример 3 — линейный множитель и квадратный трёхчлен

**Найти** $\displaystyle\int \frac{x^2+1}{(x-1)(x^2+x+1)}\,dx$.

**Шаг 1.** Знаменатель: $(x-1)$ — линейный, $(x^2+x+1)$ — квадратный (дискриминант $1-4 = -3 < 0$, корней нет).

**Шаг 2.** Шаблон:

$$\frac{x^2+1}{(x-1)(x^2+x+1)} = \frac{A}{x-1} + \frac{Bx+C}{x^2+x+1}.$$

**Шаг 3.** Умножаем на $(x-1)(x^2+x+1)$:

$$x^2+1 = A(x^2+x+1) + (Bx+C)(x-1).$$

**Шаг 4.** Находим коэффициенты:

- Подставляем $x=1$: $\quad 2 = A \cdot 3 \Rightarrow A = \dfrac{2}{3}$.
- Раскрываем правую часть и приравниваем коэффициенты:

$$x^2+1 = Ax^2 + Ax + A + Bx^2 - Bx + Cx - C.$$
$$x^2+1 = (A+B)x^2 + (A - B + C)x + (A - C).$$

- При $x^2$: $A + B = 1 \Rightarrow B = 1 - \dfrac{2}{3} = \dfrac{1}{3}$.
- При $x^0$: $A - C = 1 \Rightarrow C = A - 1 = -\dfrac{1}{3}$.

**Шаг 5.** Разложение:

$$\frac{x^2+1}{(x-1)(x^2+x+1)} = \frac{2/3}{x-1} + \frac{\frac{1}{3}x - \frac{1}{3}}{x^2+x+1}.$$

**Шаг 6.** Интегрируем каждую часть по типу I и III (из билета 43):

$$\int \frac{x^2+1}{(x-1)(x^2+x+1)}\,dx = \frac{2}{3}\ln|x-1| + \frac{1}{3}\int\frac{x-1}{x^2+x+1}\,dx.$$

Второй интеграл считается по методу типа III (разбивка числителя через производную знаменателя).

## Практика

### Задача 1 — два простых корня

$$\int \frac{dx}{(x-2)(x+3)}$$

<details>
<summary>Решение</summary>

**Шаблон:** $\dfrac{1}{(x-2)(x+3)} = \dfrac{A}{x-2} + \dfrac{B}{x+3}$

**Умножаем на знаменатель:** $1 = A(x+3) + B(x-2)$

**Подставляем корни:**

- $x = 2$: $\quad 1 = 5A \Rightarrow A = \dfrac{1}{5}$
- $x = -3$: $\quad 1 = -5B \Rightarrow B = -\dfrac{1}{5}$

**Интегрируем:** $\displaystyle\int \frac{dx}{(x-2)(x+3)} = \frac{1}{5}\ln|x-2| - \frac{1}{5}\ln|x+3| + C = \frac{1}{5}\ln\left|\frac{x-2}{x+3}\right| + C$

</details>

---

### Задача 2 — числитель не единица

$$\int \frac{2x+1}{(x+1)(x-3)}\,dx$$

<details>
<summary>Решение</summary>

**Шаблон:** $\dfrac{2x+1}{(x+1)(x-3)} = \dfrac{A}{x+1} + \dfrac{B}{x-3}$

**Умножаем на знаменатель:** $2x+1 = A(x-3) + B(x+1)$

**Подставляем корни:**

- $x = -1$: $\quad -1 = A(-4) \Rightarrow A = \dfrac{1}{4}$
- $x = 3$: $\quad 7 = B \cdot 4 \Rightarrow B = \dfrac{7}{4}$

**Интегрируем:** $\displaystyle\int \frac{2x+1}{(x+1)(x-3)}\,dx = \frac{1}{4}\ln|x+1| + \frac{7}{4}\ln|x-3| + C$

</details>

---

### Задача 3 — кратный корень

$$\int \frac{3x}{x^2(x-1)}\,dx$$

<details>
<summary>Решение</summary>

**Шаблон** — на $x^2$ пишем два слагаемых: $\dfrac{3x}{x^2(x-1)} = \dfrac{A}{x} + \dfrac{B}{x^2} + \dfrac{C}{x-1}$

**Умножаем на $x^2(x-1)$:** $3x = Ax(x-1) + B(x-1) + Cx^2$

**Подставляем корни:**

- $x = 0$: $\quad 0 = B(-1) \Rightarrow B = 0$
- $x = 1$: $\quad 3 = C \Rightarrow C = 3$

**Приравниваем коэффициенты при $x^2$:** $0 = A + C \Rightarrow A = -3$

**Интегрируем:** $\displaystyle\int \frac{3x}{x^2(x-1)}\,dx = -3\ln|x| + 3\ln|x-1| + C = 3\ln\left|\frac{x-1}{x}\right| + C$

</details>

---

### Задача 4 — квадратный трёхчлен в знаменателе

$$\int \frac{x+3}{(x-1)(x^2+1)}\,dx$$

<details>
<summary>Решение</summary>

**Шаблон** — на квадратный трёхчлен пишем $\dfrac{Bx+C}{\cdots}$:

$\dfrac{x+3}{(x-1)(x^2+1)} = \dfrac{A}{x-1} + \dfrac{Bx+C}{x^2+1}$

**Умножаем на $(x-1)(x^2+1)$:** $x+3 = A(x^2+1) + (Bx+C)(x-1)$

**Подставляем $x = 1$:** $4 = 2A \Rightarrow A = 2$

**Раскрываем:** $x+3 = (2+B)x^2 + (-B+C)x + (2-C)$

- При $x^2$: $\quad 0 = 2 + B \Rightarrow B = -2$
- При $x^0$: $\quad 3 = 2 - C \Rightarrow C = -1$

**Разложение:** $\dfrac{x+3}{(x-1)(x^2+1)} = \dfrac{2}{x-1} + \dfrac{-2x-1}{x^2+1}$

**Интегрируем:** $\displaystyle\int \frac{x+3}{(x-1)(x^2+1)}\,dx = 2\ln|x-1| - \ln(x^2+1) - \arctan x + C$

</details>

---

### Задача 5 — неправильная дробь (нужно выделить целую часть)

$$\int \frac{x^3}{x^2-1}\,dx$$

<details>
<summary>Решение</summary>

Степень числителя ($3$) больше степени знаменателя ($2$) — дробь **неправильная**. Сначала делим в столбик:

$\dfrac{x^3}{x^2-1} = x + \dfrac{x}{x^2-1}$

Раскладываем оставшуюся правильную дробь: $\dfrac{x}{(x-1)(x+1)} = \dfrac{A}{x-1} + \dfrac{B}{x+1}$

Умножаем: $x = A(x+1) + B(x-1)$

- $x = 1$: $\quad 1 = 2A \Rightarrow A = \dfrac{1}{2}$
- $x = -1$: $\quad -1 = -2B \Rightarrow B = \dfrac{1}{2}$

**Интегрируем:** $\displaystyle\int \frac{x^3}{x^2-1}\,dx = \frac{x^2}{2} + \frac{1}{2}\ln|x-1| + \frac{1}{2}\ln|x+1| + C = \frac{x^2}{2} + \frac{1}{2}\ln|x^2-1| + C$

</details>
