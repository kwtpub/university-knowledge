## Определения пределов

**Предел последовательности** ($\lim_{n\to\infty} x_n = A$):

$$\forall \varepsilon > 0 \ \exists N \in \mathbb{N} : \forall n > N \Rightarrow |x_n - A| < \varepsilon$$

**Предел функции по Коши** ($\lim_{x\to a} f(x) = A$):

$$\forall \varepsilon > 0 \ \exists \delta > 0 : \forall x \ \ 0 < |x - a| < \delta \Rightarrow |f(x) - A| < \varepsilon$$

**Предел функции по Гейне** ($\lim_{x\to a} f(x) = A$):

$$\forall \{x_n\} : x_n \to a,\ x_n \neq a \Rightarrow f(x_n) \to A$$

> На пальцах: для любой последовательности $x_n$ стремящейся к $a$ (но не равной $a$), значения $f(x_n)$ стремятся к $A$.

## Эквивалентные бесконечно малые при $x \to 0$

| Функция | Эквивалентна |
|---|---|
| $\sin x$ | $x$ |
| $\tan x$ | $x$ |
| $\arcsin x$ | $x$ |
| $\arctan x$ | $x$ |
| $1 - \cos x$ | $\dfrac{x^2}{2}$ |
| $\ln(1+x)$ | $x$ |
| $e^x - 1$ | $x$ |
| $(1+x)^\alpha - 1$ | $\alpha x$ |

> **Правило:** если внутри стоит не $x$, а $f(x)$, заменяем всё целиком. Например $\arcsin 2x \sim 2x$, $1 - \cos 4x \sim \dfrac{(4x)^2}{2} = 8x^2$
