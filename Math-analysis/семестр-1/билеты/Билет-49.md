## 49. Интегрирование тригонометрических выражений вида $R(\sin x, \cos x)$. Универсальная тригонометрическая подстановка.

$$\int \sin u\,du = -\cos u + C$$

$$\int \cos u\,du = \sin u + C$$

Здесь $R$ — обозначение некоторой рациональной функции от переменных $\sin x$ и $\cos x$. Интегралы этого вида вычисляются с помощью подстановки $t = \operatorname{tg}\dfrac{x}{2}$. Эта подстановка позволяет преобразовать тригонометрическую функцию в рациональную:

$$\sin x = \frac{2\operatorname{tg}\frac{x}{2}}{1 + \operatorname{tg}^2\frac{x}{2}} = \frac{2t}{1+t^2}, \quad \cos x = \frac{1 - \operatorname{tg}^2\frac{x}{2}}{1 + \operatorname{tg}^2\frac{x}{2}} = \frac{1-t^2}{1+t^2}, \quad x = 2\operatorname{arctg} t; \quad dx = \frac{2\,dt}{1+t^2}.$$

Таким образом:

$$\int R(\sin x, \cos x)\,dx = \int R\!\left(\frac{2t}{1+t^2},\, \frac{1-t^2}{1+t^2}\right)\frac{2}{1+t^2}\,dt = \int r(t)\,dt.$$

Описанное выше преобразование называется **универсальной тригонометрической подстановкой**.

---

## Объяснение на пальцах

Есть интеграл где под знаком стоит какая-то комбинация sin x и cos x — например $\dfrac{1}{1+\sin x}$ или $\dfrac{\sin x}{1+\cos x}$. Напрямую такие интегралы не берутся.

Идея: делаем замену $t = \text{tg}(x/2)$ — и sin x, cos x, dx все выражаются через $t$ рационально. Корней нет, только дроби. Получается рациональный интеграл который считаем по билету 44.

Три формулы которые надо знать наизусть:

$$\sin x = \frac{2t}{1+t^2}, \quad \cos x = \frac{1-t^2}{1+t^2}, \quad dx = \frac{2\,dt}{1+t^2}$$

где $t = \text{tg}(x/2)$.

Откуда они берутся — из формул двойного угла. Например для sin x:

$\sin x = 2\sin\dfrac{x}{2}\cos\dfrac{x}{2}$. Делим числитель и знаменатель на $\cos^2\dfrac{x}{2}$:

$\sin x = \dfrac{2\text{tg}\frac{x}{2}}{1+\text{tg}^2\frac{x}{2}} = \dfrac{2t}{1+t^2}$

Для cos x используем $\cos x = \cos^2\dfrac{x}{2} - \sin^2\dfrac{x}{2}$, аналогично делим на $\cos^2\dfrac{x}{2}$:

$\cos x = \dfrac{1-\text{tg}^2\frac{x}{2}}{1+\text{tg}^2\frac{x}{2}} = \dfrac{1-t^2}{1+t^2}$

Для dx: из $t = \text{tg}(x/2)$ берём производную: $dt = \dfrac{1}{2\cos^2(x/2)}\,dx$, откуда $dx = \dfrac{2\,dt}{1+t^2}$.

## Примеры

### Пример 1

Найти $\displaystyle\int \frac{dx}{1+\sin x}$

**Шаг 1.** Замена $t = \text{tg}(x/2)$. Подставляем формулы:

$$\sin x = \frac{2t}{1+t^2}, \quad dx = \frac{2\,dt}{1+t^2}$$

**Шаг 2.** Подставляем в интеграл:

$$\int \frac{1}{1+\frac{2t}{1+t^2}} \cdot \frac{2\,dt}{1+t^2}$$

**Шаг 3.** Упрощаем знаменатель $1 + \dfrac{2t}{1+t^2}$:

$$1 + \frac{2t}{1+t^2} = \frac{1+t^2+2t}{1+t^2} = \frac{(1+t)^2}{1+t^2}$$

**Шаг 4.** Подставляем обратно:

$$\int \frac{1+t^2}{(1+t)^2} \cdot \frac{2\,dt}{1+t^2} = \int \frac{2\,dt}{(1+t)^2} = 2\int(1+t)^{-2}\,dt$$

**Шаг 5.** Берём интеграл — замена $u = 1+t$:

$$2 \cdot \frac{(1+t)^{-1}}{-1} + C = -\frac{2}{1+t} + C$$

**Шаг 6.** Обратная замена $t = \text{tg}(x/2)$:

$$\boxed{-\frac{2}{1+\text{tg}(x/2)} + C}$$

---

### Пример 2

Найти $\displaystyle\int \frac{dx}{2+\cos x}$

**Шаг 1.** Замена $t = \text{tg}(x/2)$:

$$\cos x = \frac{1-t^2}{1+t^2}, \quad dx = \frac{2\,dt}{1+t^2}$$

**Шаг 2.** Подставляем:

$$\int \frac{1}{2+\frac{1-t^2}{1+t^2}} \cdot \frac{2\,dt}{1+t^2}$$

**Шаг 3.** Упрощаем знаменатель:

$$2 + \frac{1-t^2}{1+t^2} = \frac{2(1+t^2)+1-t^2}{1+t^2} = \frac{t^2+3}{1+t^2}$$

**Шаг 4.** Подставляем:

$$\int \frac{1+t^2}{t^2+3} \cdot \frac{2\,dt}{1+t^2} = 2\int \frac{dt}{t^2+3}$$

**Шаг 5.** Табличный интеграл $\displaystyle\int \frac{dt}{t^2+a^2} = \frac{1}{a}\arctg\frac{t}{a}$, здесь $a = \sqrt{3}$:

$$2 \cdot \frac{1}{\sqrt{3}}\arctg\frac{t}{\sqrt{3}} + C = \frac{2}{\sqrt{3}}\arctg\frac{t}{\sqrt{3}} + C$$

**Шаг 6.** Обратная замена $t = \text{tg}(x/2)$:

$$\boxed{\frac{2}{\sqrt{3}}\arctg\frac{\text{tg}(x/2)}{\sqrt{3}} + C}$$

## Практика

### Задача 1

$$\int \frac{dx}{1+\cos x}$$

Подсказка: замена $t = \text{tg}(x/2)$, подставь $\cos x$ и $dx$.

<details>
<summary>Ответ</summary>

$$\int \frac{2\,dt/(1+t^2)}{1+(1-t^2)/(1+t^2)} = \int \frac{2\,dt}{2} = \int dt = t + C = \text{tg}\frac{x}{2} + C$$

</details>

---

### Задача 2

$$\int \frac{dx}{1-\sin x}$$

Подсказка: замена $t = \text{tg}(x/2)$, подставь $\sin x$ и $dx$.

<details>
<summary>Ответ</summary>

Знаменатель: $1 - \dfrac{2t}{1+t^2} = \dfrac{(1-t)^2}{1+t^2}$

$$\int \frac{2\,dt}{(1-t)^2} = \frac{2}{1-t} + C = \frac{2}{1-\text{tg}(x/2)} + C$$

</details>

---

### Задача 3

$$\int \frac{dx}{3+2\sin x}$$

Подсказка: подставь $\sin x = \dfrac{2t}{1+t^2}$, упрости знаменатель и получишь интеграл типа $\int \dfrac{dt}{t^2+at+b}$.

<details>
<summary>Ответ</summary>

Знаменатель: $3 + \dfrac{4t}{1+t^2} = \dfrac{3t^2+4t+3}{1+t^2}$

$$2\int \frac{dt}{3t^2+4t+3}$$

Выделяем полный квадрат: $3t^2+4t+3 = 3\left(t+\dfrac{2}{3}\right)^2 + \dfrac{5}{3}$

$$\frac{2}{\sqrt{5}}\arctg\frac{3t+2}{\sqrt{5}} + C = \frac{2}{\sqrt{5}}\arctg\frac{3\text{tg}(x/2)+2}{\sqrt{5}} + C$$

</details>
