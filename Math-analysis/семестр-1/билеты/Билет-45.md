## 45. Неопределённый интеграл от иррациональной функции. Интегрирование выражений вида $R\!\left(x,\, \sqrt[n]{\dfrac{ax+b}{cx+d}}\right)$.

**Интеграл вида** $\displaystyle\int R\!\left(x,\, \sqrt[n]{\frac{ax+b}{cx+d}}\right)dx$, где $n$ — натуральное число, $R$ — рациональная функция от своих аргументов.

---

### Идея метода: рационализация подстановкой

Чтобы избавиться от корня, вводим замену:

$$t = \sqrt[n]{\frac{ax+b}{cx+d}}$$

Тогда выражение под корнем:

$$\frac{ax+b}{cx+d} = t^n$$

Из этого уравнения выражаем $x$ через $t$ (решаем линейное уравнение относительно $x$):

$$ax + b = t^n(cx + d)$$
$$ax - ct^n x = dt^n - b$$
$$x(a - ct^n) = dt^n - b$$
$$x = \frac{dt^n - b}{a - ct^n}$$

Дифференцируем, чтобы найти $dx$:

$$dx = \left(\frac{dt^n - b}{a - ct^n}\right)' dt$$

Производная считается по $t$; это дробь, дифференцируется по правилу частного.

---

### Результат подстановки

После замены интеграл становится **рациональным**:

$$\int R\!\left(x,\, \sqrt[n]{\frac{ax+b}{cx+d}}\right)dx = \int R\!\left(\frac{dt^n-b}{a-ct^n},\, t\right)\left(\frac{dt^n-b}{a-ct^n}\right)'dt = \int r(t)\,dt$$

где $r(t)$ — уже рациональная функция от $t$, которую берут стандартными методами (разложение на простейшие дроби и т.д.).

---

### Ключевые моменты

| Что | Как |
|---|---|
| Подстановка | $t = \sqrt[n]{\dfrac{ax+b}{cx+d}}$ |
| Выражаем $x$ | Решаем линейное уравнение из $\dfrac{ax+b}{cx+d} = t^n$ |
| Находим $dx$ | Дифференцируем $x(t)$ по $t$, умножаем на $dt$ |
| Результат | Рациональный интеграл $\int r(t)\,dt$ |

После перехода к рациональной функции используют метод **разложения на простейшие дроби**.

---

## Объяснение на пальцах

Идея простая: под интегралом стоит корень — он мешает считать. Мы вводим замену t = этот корень, выражаем x через t, находим dx через t, и подставляем всё в интеграл. Корень исчезает — остаётся рациональная функция от t, которую уже умеем считать по билету 44.

Важно: после того как посчитал интеграл по t — не забыть сделать обратную замену и вернуться к x.

---

## Пример

Найти $\displaystyle\int \frac{dx}{\sqrt{x+1} + 1}$.

Это частный случай нашего типа: под корнем стоит $\sqrt{x+1}$, то есть $n=2$, $a=1$, $b=1$, $c=0$, $d=1$.

**Шаг 1. Вводим замену.**

$$t = \sqrt{x+1}.$$

**Шаг 2. Выражаем x через t.**

$$t^2 = x+1 \Rightarrow x = t^2 - 1.$$

**Шаг 3. Находим dx.**

$$dx = 2t\,dt.$$

**Шаг 4. Подставляем в интеграл — корень заменяется на t, dx на 2t dt.**

$$\int \frac{dx}{\sqrt{x+1}+1} = \int \frac{2t\,dt}{t+1}.$$

**Шаг 5. Считаем рациональный интеграл.**

Числитель $2t$ — степень равна степени знаменателя, значит дробь неправильная. Делим:

$$\frac{2t}{t+1} = 2 - \frac{2}{t+1}.$$

Интегрируем:

$$\int \left(2 - \frac{2}{t+1}\right)dt = 2t - 2\ln|t+1| + C.$$

**Шаг 6. Обратная замена $t = \sqrt{x+1}$.**

$$\boxed{2\sqrt{x+1} - 2\ln(\sqrt{x+1}+1) + C.}$$

## Практика

### Задача 1 — простой корень в знаменателе

$$\int \frac{dx}{\sqrt{x+3} + 2}$$

Замена: $t = \sqrt{x+3}$

<details>
<summary>Решение</summary>

$t = \sqrt{x+3} \Rightarrow t^2 = x+3 \Rightarrow x = t^2-3$

$dx = 2t\,dt$

$$\int \frac{2t\,dt}{t+2}$$

Делим $2t$ на $t+2$. Степень числителя равна степени знаменателя — дробь неправильная, делим в столбик:

- Смотрим на старшие члены: $2t$ делим на $t$ — получаем $2$.
- Умножаем $2$ на весь знаменатель $t+2$: получаем $2t+4$.
- Вычитаем из числителя: $2t - (2t+4) = -4$.
- Остаток $-4$ — степень меньше знаменателя, дальше не делим.

Итого: $\dfrac{2t}{t+2} = 2 - \dfrac{4}{t+2}$

$$\int \left(2 - \frac{4}{t+2}\right)dt = 2t - 4\ln|t+2| + C$$

Обратная замена $t = \sqrt{x+3}$:

$$\boxed{2\sqrt{x+3} - 4\ln(\sqrt{x+3}+2) + C}$$

</details>

### Задача 2 — корень из дроби, n=2

$$\int \sqrt{\frac{x+1}{x-1}}\,dx$$

Замена: $t = \sqrt{\dfrac{x+1}{x-1}}$, то есть $t^2 = \dfrac{x+1}{x-1}$

Выразить x через t: умножить обе части на $(x-1)$, раскрыть и выразить $x$.

<details>
<summary>Решение</summary>

$t^2(x-1) = x+1$

$t^2 x - t^2 = x+1$

$t^2 x - x = 1 + t^2$

$x(t^2-1) = 1+t^2$

$x = \dfrac{1+t^2}{t^2-1}$

Дифференцируем по правилу частного:

$$dx = \frac{2t(t^2-1) - (1+t^2)\cdot 2t}{(t^2-1)^2}\,dt = \frac{2t(t^2-1-1-t^2)}{(t^2-1)^2}\,dt = \frac{-4t}{(t^2-1)^2}\,dt$$

Подставляем ($\sqrt{\frac{x+1}{x-1}} = t$):

$$\int t \cdot \frac{-4t}{(t^2-1)^2}\,dt = -4\int \frac{t^2}{(t^2-1)^2}\,dt$$

Это рациональный интеграл — $t^2-1 = (t-1)(t+1)$, раскладываем на простейшие дроби по билету 44.

Имеем $-4\displaystyle\int \frac{t^2}{(t-1)^2(t+1)^2}\,dt$. Шаблон разложения — каждый множитель кратный (степень 2), поэтому на каждый по две дроби:

$$\frac{t^2}{(t-1)^2(t+1)^2} = \frac{A}{t-1} + \frac{B}{(t-1)^2} + \frac{C}{t+1} + \frac{D}{(t+1)^2}$$

Умножаем обе части на $(t-1)^2(t+1)^2$:

$$t^2 = A(t-1)(t+1)^2 + B(t+1)^2 + C(t+1)(t-1)^2 + D(t-1)^2$$

Подставляем удобные значения:

- $t=1$: $\quad 1 = 0 + B\cdot4 + 0 + 0 \Rightarrow B = \dfrac{1}{4}$
- $t=-1$: $\quad 1 = 0 + 0 + 0 + D\cdot4 \Rightarrow D = \dfrac{1}{4}$

Приравниваем коэффициенты при $t^3$: $\quad 0 = A + C$, значит $C = -A$.

Приравниваем коэффициенты при $t^0$ (свободный член): $\quad 0 = -A\cdot1\cdot1 + B\cdot1 + C\cdot1\cdot1 + D\cdot1 = -A + \tfrac{1}{4} + C + \tfrac{1}{4}$

Подставляем $C = -A$: $\quad 0 = -A + \tfrac{1}{4} - A + \tfrac{1}{4} = -2A + \tfrac{1}{2} \Rightarrow A = \dfrac{1}{4}$, $C = -\dfrac{1}{4}$.

Разложение:

$$\frac{t^2}{(t-1)^2(t+1)^2} = \frac{1/4}{t-1} + \frac{1/4}{(t-1)^2} - \frac{1/4}{t+1} + \frac{1/4}{(t+1)^2}$$

Интегрируем:

$$-4\int\left(\frac{1/4}{t-1} + \frac{1/4}{(t-1)^2} - \frac{1/4}{t+1} + \frac{1/4}{(t+1)^2}\right)dt$$

$$= -4\left(\frac{1}{4}\ln|t-1| - \frac{1}{4(t-1)} - \frac{1}{4}\ln|t+1| - \frac{1}{4(t+1)}\right) + C$$

$$= -\ln|t-1| + \frac{1}{t-1} + \ln|t+1| + \frac{1}{t+1} + C$$

$$= \ln\left|\frac{t+1}{t-1}\right| + \frac{1}{t-1} + \frac{1}{t+1} + C$$

Обратная замена $t = \sqrt{\dfrac{x+1}{x-1}}$:

$$\boxed{\ln\left|\frac{\sqrt{\frac{x+1}{x-1}}+1}{\sqrt{\frac{x+1}{x-1}}-1}\right| + \frac{1}{\sqrt{\frac{x+1}{x-1}}-1} + \frac{1}{\sqrt{\frac{x+1}{x-1}}+1} + C}$$

</details>

### Задача 3 — корень третьей степени

$$\int \frac{dx}{1 + \sqrt[3]{x+2}}$$

Замена: $t = \sqrt[3]{x+2}$, то есть $t^3 = x+2$

<details>
<summary>Решение</summary>

$t^3 = x+2 \Rightarrow x = t^3-2$

$dx = 3t^2\,dt$

$$\int \frac{3t^2\,dt}{1+t}$$

Делим $3t^2$ на $1+t$ в столбик:

$$\frac{3t^2}{1+t} = 3t - 3 + \frac{3}{1+t}$$

$$\int \left(3t - 3 + \frac{3}{1+t}\right)dt = \frac{3t^2}{2} - 3t + 3\ln|1+t| + C$$

Обратная замена $t = \sqrt[3]{x+2}$:

$$\boxed{\frac{3\sqrt[3]{(x+2)^2}}{2} - 3\sqrt[3]{x+2} + 3\ln|1+\sqrt[3]{x+2}| + C}$$

</details>
