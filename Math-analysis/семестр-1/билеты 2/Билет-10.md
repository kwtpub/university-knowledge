## Билет 10. Предел функции в точке по Коши и по Гейне, их эквивалентность. Предел функции на бесконечности. Односторонние пределы.

---

## Предел функции по Коши (epsilon-delta)

Число $A$ называется пределом $f(x)$ при $x \to a$, если:

$$\forall \varepsilon > 0 \ \exists \delta > 0 : \ 0 < |x - a| < \delta \Rightarrow |f(x) - A| < \varepsilon$$

![Определение предела функции по Коши](https://mathprofi.ru/n/predely_po_koshi_clip_image016.jpg)

На картинке:
- По оси $X$: коридор шириной $\delta$ вокруг точки $x_0$ — это $(x_0 - \delta,\ x_0 + \delta)$, красные стрелки
- По оси $Y$: коридор шириной $\varepsilon$ вокруг предела $A$ — это $(A - \varepsilon,\ A + \varepsilon)$, синие стрелки
- Белый кружок в точке $x_0$ — функция там **не определена** (или её значение не важно)
- Смысл: все $x$ из $\delta$-коридора попадают в $\varepsilon$-коридор по $y$

> **На пальцах:** как бы мало мы ни взяли $\varepsilon$ (коридор по $y$), найдётся такой $\delta$ (коридор по $x$), что все $x$ из этого коридора дают $f(x)$ в коридоре по $y$.

> **Важно:** $0 < |x - a|$ означает что $x \neq a$. Предел — это про поведение функции **вблизи** точки, не в самой точке.

---

## Предел функции по Гейне (через последовательности)

Число $A$ называется пределом $f(x)$ при $x \to a$, если:

$$\forall \{x_n\} : \ x_n \neq a, \ \lim_{n \to \infty} x_n = a \Rightarrow \lim_{n \to \infty} f(x_n) = A$$

То есть для **любой** последовательности $x_n$, стремящейся к $a$, последовательность значений $f(x_n)$ стремится к $A$.

> **На пальцах:** подходи к точке $a$ как угодно — хоть слева, хоть справа, хоть зигзагом — $f(x)$ всегда будет стремиться к $A$.

---

## Сравнение двух определений

| | По Коши | По Гейне |
|---|---|---|
| Язык | $\varepsilon$-$\delta$ | Последовательности |
| Удобен для | Точных доказательств | Проверки через примеры |
| Суть | Коридор по $x$ → коридор по $y$ | Любая последовательность → одно значение |

**Теорема:** оба определения **эквивалентны** — дают одно и то же понятие предела.

---

## Предел функции на бесконечности

> Определяется по **Коши** — язык $\varepsilon$-$\delta$. По Гейне здесь менее удобно, так как бесконечность — не точка, к которой стремится последовательность в обычном смысле.

При $x \to +\infty$:

$$\forall \varepsilon > 0 \ \exists \delta > 0 : \ x > \delta \Rightarrow |f(x) - A| < \varepsilon$$

При $x \to -\infty$:

$$\forall \varepsilon > 0 \ \exists \delta > 0 : \ x < -\delta \Rightarrow |f(x) - A| < \varepsilon$$

> **На пальцах:** как бы мало ни был $\varepsilon$, начиная с какого-то $\delta$ все значения $f(x)$ будут в коридоре $(A-\varepsilon, A+\varepsilon)$.

**Правило для дробей** $\dfrac{h(x)}{g(x)}$ при $x \to \infty$: предел равен отношению **старших членов** числителя и знаменателя.

Пример:
$$\lim_{x \to \infty} \frac{3x^2 + x + 1}{x^2 - 5} = \lim_{x \to \infty} \frac{3x^2}{x^2} = 3$$

## Односторонние пределы

> Определяются по **Коши** — просто добавляем ограничение на направление подхода к точке.

**Предел справа** ($x \to a^+$, подходим только с правой стороны, $x > a$):

$$\forall \varepsilon > 0 \ \exists \delta > 0 : \ 0 < x - a < \delta \Rightarrow |f(x) - A^+| < \varepsilon$$

**Предел слева** ($x \to a^-$, подходим только с левой стороны, $x < a$):

$$\forall \varepsilon > 0 \ \exists \delta > 0 : \ 0 < a - x < \delta \Rightarrow |f(x) - A^-| < \varepsilon$$

**Связь с двусторонним пределом:**

$$\lim_{x \to a} f(x) = A \iff \lim_{x \to a^+} f(x) = \lim_{x \to a^-} f(x) = A$$

Предел существует тогда и только тогда, когда оба односторонних предела существуют и **равны**.

> Пример где односторонние пределы не равны: $f(x) = \text{sign}(x)$ в точке $x = 0$.
> - Слева: $\lim_{x \to 0^-} = -1$
> - Справа: $\lim_{x \to 0^+} = 1$
> - Двусторонний предел **не существует**.

> Пример где односторонние пределы не равны: $f(x) = \text{sign}(x)$ в точке $x = 0$.
> Слева: $\lim_{x \to 0^-} = -1$. Справа: $\lim_{x \to 0^+} = 1$. Двусторонний предел не существует.

## Практика

**Задание 1.** Найти $\lim_{x \to \infty} \dfrac{5x^3 - x}{2x^3 + 7}$

> [!note]- Ответ
> Берём старшие члены:
>
> $$\lim_{x \to \infty} \frac{5x^3}{2x^3} = \frac{5}{2}$$

---

**Задание 2.** Существует ли $\lim_{x \to 0} \dfrac{|x|}{x}$?

> [!note]- Ответ
> Предел справа: $\lim_{x \to 0^+} \dfrac{x}{x} = 1$
>
> Предел слева: $\lim_{x \to 0^-} \dfrac{-x}{x} = -1$
>
> Пределы не равны — двусторонний предел **не существует**.

---

**Задание 3.** Чем отличается определение по Коши от определения по Гейне?

> [!note]- Ответ
> По Коши — язык $\varepsilon$-$\delta$: для любого коридора по $y$ найдётся коридор по $x$.
>
> По Гейне — язык последовательностей: любая последовательность $x_n \to a$ даёт $f(x_n) \to A$.
>
> Оба определения эквивалентны.
