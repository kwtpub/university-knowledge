## 60. Приложение интегрального исчисления к геометрии. Площадь поверхности вращения.

**Определение.** Площадью поверхности вращения кривой $AB$ вокруг заданной оси называется предел, к которому стремятся площади поверхностей вращения ломаных, вписанных в кривую $AB$, при стремлении к нулю наибольшей из длин звеньев этих ломаных.

Разобьём дугу $AB$ на $n$ частей точками $M_0 = A, M_1, \ldots, M_{n-1}, M_n = B$. Координаты вершин полученной ломаной имеют координаты $x_i$ и $y_i$. При вращении ломаной вокруг оси получим поверхность, состоящую из боковых поверхностей усечённых конусов, площадь которых равна $\Delta S_i$. Эта площадь может быть найдена по формуле:

$$\Delta S_i = 2\pi\frac{y_{i-1}+y_i}{2}\Delta l_i = \pi(y_{i-1}+y_i)\Delta l_i.$$

Здесь $\Delta l_i$ — длина каждой хорды:

$$\Delta l_i = \sqrt{\Delta x_i^2 + \Delta y_i^2} = \sqrt{1 + \left(\frac{\Delta y_i}{\Delta x_i}\right)^2}\,\Delta x_i.$$

Применяя теорему Лагранжа к отношению $\dfrac{\Delta y_i}{\Delta x_i}$, получаем:

$$\frac{\Delta y_i}{\Delta x_i} = \frac{f(x_i) - f(x_{i-1})}{x_i - x_{i-1}} = f'(\varepsilon_i), \quad x_{i-1} < \varepsilon_i < x_i.$$

Тогда:

$$\Delta l_i = \sqrt{1 + f'^2(\varepsilon_i)}\,\Delta x_i \implies \Delta S_i = \pi(y_{i-1}+y_i)\sqrt{1+f'^2(\varepsilon_i)}\,\Delta x_i.$$

Площадь поверхности, описанной ломаной равна:

$$S_n = \pi\sum_{i=1}^n [f(x_{i-1}) + f(x_i)]\sqrt{1+f'^2(\varepsilon_i)}\,\Delta x_i.$$

Эта сумма не является интегральной, но можно показать, что:

$$S = \lim_{\max \Delta x_i \to 0} \pi\sum_{i=1}^n [f(x_{i-1})+f(x_i)]\sqrt{1+f'^2(\varepsilon_i)}\,\Delta x_i = \lim_{\max \Delta x_i \to 0} 2\pi\sum_{i=1}^n f(\varepsilon_i)\sqrt{1+f'^2(\varepsilon_i)}\,\Delta x_i.$$

Тогда **площадь поверхности тела вращения** находится по формуле:

$$S = 2\pi\int_a^b f(x)\sqrt{1+f'^2(x)}\,dx.$$
