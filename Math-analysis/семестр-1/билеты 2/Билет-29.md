## 29. Производная обратной функции и функции, заданной параметрически.

---

## Производная обратной функции

Если функция $y = f(x)$ имеет обратную $x = g(y)$, и $g'(y) \neq 0$, то:

$$y'_x = \frac{1}{x'_y}$$

Словами: производная обратной функции — это единица, делённая на производную прямой функции. Смотреть надо внимательно: $x'_y$ — это производная $x$ по $y$, а не наоборот.

**Пример 1.** Найти $(\arcsin x)'$.

Пусть $y = \arcsin x$, тогда обратная функция $x = \sin y$.

$x'_y = (\sin y)' = \cos y = \sqrt{1 - \sin^2 y} = \sqrt{1 - x^2}$

$$(\arcsin x)' = \frac{1}{x'_y} = \frac{1}{\sqrt{1-x^2}}$$

**Пример 2.** Найти $(\text{arctg}\, x)'$.

Пусть $y = \text{arctg}\, x$, тогда обратная функция $x = \text{tg}\, y$.

$x'_y = (\text{tg}\, y)' = \dfrac{1}{\cos^2 y} = 1 + \text{tg}^2 y = 1 + x^2$

$$(\text{arctg}\, x)' = \frac{1}{x'_y} = \frac{1}{1+x^2}$$

## Производная функции, заданной параметрически

Иногда функцию задают не явно $y = f(x)$, а через параметр $t$:

$$\begin{cases} x = g(t) \\ y = h(t) \end{cases}$$

Тогда производная вычисляется по формуле:

$$y'_x = \frac{h'(t)}{g'(t)}$$

Словами: берём производную $y$ по $t$ и делим на производную $x$ по $t$.

**Пример 1.** Найти $y'_x$, если $\begin{cases} x = t^2 \\ y = t^3 \end{cases}$.

$x'_t = 2t$, $\quad y'_t = 3t^2$

$$y'_x = \frac{3t^2}{2t} = \frac{3t}{2}$$

**Пример 2.** Найти $y'_x$ для эллипса $\begin{cases} x = a\cos t \\ y = b\sin t \end{cases}$.

$x'_t = -a\sin t$, $\quad y'_t = b\cos t$

$$y'_x = \frac{b\cos t}{-a\sin t} = -\frac{b}{a}\cdot\frac{\cos t}{\sin t} = -\frac{b}{a}\,\text{ctg}\, t$$

**Пример 3.** Найти $y'_x$, если $\begin{cases} x = e^t \\ y = e^{2t} \end{cases}$.

$x'_t = e^t$, $\quad y'_t = 2e^{2t}$

$$y'_x = \frac{2e^{2t}}{e^t} = 2e^t = 2x$$

Можно проверить: из параметрического задания $y = e^{2t} = (e^t)^2 = x^2$, значит $y' = 2x$. Совпадает.
