## 50. Интегрирование рациональных тригонометрических выражений $R(\sin x, \cos x)$ в случае, когда подынтегральная функция нечётна относительно $\sin x$ (или $\cos x$) или чётна относительно $\sin x$ и $\cos x$.

### Интеграл вида $\int R(\sin x, \cos x)\,dx$ если функция $R$ является нечётной относительно косинуса

Несмотря на возможность вычисления такого интеграла с помощью универсальной тригонометрической подстановки, рациональнее применить подстановку $t = \sin x$ если функция $R$ является нечётной относительно косинуса.

$$\int R(\sin x, \cos x)\,dx = \int \frac{R(\sin x, \cos x)}{\cos x}\cos x\,dx.$$

Функция $\dfrac{R(\sin x, \cos x)}{\cos x}$ может содержать $\cos x$ только в чётных степенях, а следовательно, может быть преобразована в рациональную функцию относительно $\sin x$:

$$\int R(\sin x, \cos x)\,dx = \int r(\sin x)\cos x\,dx = \int r(t)\,dt.$$

### Интеграл вида $\int R(\sin x, \cos x)\,dx$ если функция $R$ является нечётной относительно синуса

По аналогии с рассмотренным выше случаем делается подстановка $t = \cos x$. Тогда:

$$\int R(\sin x, \cos x)\,dx = \int r(\cos x)\sin x\,dx = -\int r(t)\,dt.$$

### Интеграл вида $\int R(\sin x, \cos x)\,dx$ если функция $R$ чётная относительно $\sin x$ и $\cos x$

Для преобразования подынтегральной функции в рациональную используется подстановка $t = \operatorname{tg} x$. Тогда:

$$\int R(\sin x, \cos x)\,dx = \int r(t)\,dt.$$

---

## Объяснение на пальцах

Билет 49 — универсальная подстановка $t = \text{tg}(x/2)$, работает всегда, но после неё получается громоздкий рациональный интеграл.

Билет 50 — три частных случая когда можно использовать более простую замену и считать быстрее.

Главное что надо понять — что значит нечётная/чётная функция относительно sin или cos.

**Нечётная относительно cos x** — если заменить cos x на -cos x, функция меняет знак. Например $\sin x \cdot \cos x$ — нечётная по cos, потому что $\sin x \cdot (-\cos x) = -\sin x \cos x$.

**Нечётная относительно sin x** — если заменить sin x на -sin x, функция меняет знак. Например $\sin x \cdot \cos^2 x$ — нечётная по sin, потому что $(-\sin x)\cos^2 x = -\sin x\cos^2 x$.

**Чётная относительно обоих** — если заменить и sin x на -sin x и cos x на -cos x, функция не меняется. Например $\sin^2 x$, $\cos^2 x$, $\text{tg}^2 x$.

---

**Три случая и замены:**

| Случай | Признак | Замена |
|---|---|---|
| Нечётная по cos x | $R(\sin x, -\cos x) = -R(\sin x, \cos x)$ | $t = \sin x$, $dt = \cos x\,dx$ |
| Нечётная по sin x | $R(-\sin x, \cos x) = -R(\sin x, \cos x)$ | $t = \cos x$, $dt = -\sin x\,dx$ |
| Чётная по обоим | $R(-\sin x, -\cos x) = R(\sin x, \cos x)$ | $t = \text{tg}\,x$, $dt = \dfrac{dx}{\cos^2 x}$ |

---

## Как проверить чётность/нечётность на практике

Берёшь подынтегральную функцию и заменяешь cos x на -cos x (или sin x на -sin x). Если знак изменился — нечётная по этой переменной.

Пример: $f(x) = \sin^2 x \cdot \cos x$

Заменяем cos x на -cos x: $\sin^2 x \cdot (-\cos x) = -\sin^2 x \cos x = -f(x)$

Знак изменился — нечётная по cos x. Применяем замену $t = \sin x$.

---

## Примеры

### Пример 1 — нечётная по cos x, замена $t = \sin x$

Найти $\displaystyle\int \sin^2 x \cos x\,dx$

Проверяем: заменяем $\cos x$ на $-\cos x$: $\sin^2 x \cdot (-\cos x) = -f(x)$ — нечётная по cos x.

Замена $t = \sin x$, $dt = \cos x\,dx$:

$$\int \sin^2 x \cos x\,dx = \int t^2\,dt = \frac{t^3}{3} + C$$

Обратная замена:

$$\boxed{\frac{\sin^3 x}{3} + C}$$

---

### Пример 2 — нечётная по sin x, замена $t = \cos x$

Найти $\displaystyle\int \sin x \cos^2 x\,dx$

Проверяем: заменяем $\sin x$ на $-\sin x$: $(-\sin x)\cos^2 x = -f(x)$ — нечётная по sin x.

Замена $t = \cos x$, $dt = -\sin x\,dx$, значит $\sin x\,dx = -dt$:

$$\int \sin x \cos^2 x\,dx = \int t^2 \cdot (-dt) = -\frac{t^3}{3} + C$$

Обратная замена:

$$\boxed{-\frac{\cos^3 x}{3} + C}$$

---

### Пример 3 — чётная по обоим, замена $t = \text{tg}\,x$

Найти $\displaystyle\int \frac{dx}{\sin^2 x \cos^2 x}$

Проверяем: $\dfrac{1}{(-\sin x)^2(-\cos x)^2} = \dfrac{1}{\sin^2 x\cos^2 x} = f(x)$ — чётная по обоим.

Замена $t = \text{tg}\,x$. Используем $\sin^2 x \cos^2 x = \dfrac{\text{tg}^2 x}{\sec^4 x} \cdot \cos^4 x$...

Проще использовать тождество: $\sin^2 x \cos^2 x = \dfrac{\sin^2 2x}{4} = \dfrac{1}{4}\cdot\dfrac{4\text{tg}^2 x}{(1+\text{tg}^2 x)^2}$

Или ещё проще — разобьём дробь:

$$\frac{1}{\sin^2 x \cos^2 x} = \frac{\sin^2 x + \cos^2 x}{\sin^2 x \cos^2 x} = \frac{1}{\cos^2 x} + \frac{1}{\sin^2 x}$$

$$\int \left(\frac{1}{\cos^2 x} + \frac{1}{\sin^2 x}\right)dx = \text{tg}\,x - \text{ctg}\,x + C$$

$$\boxed{\text{tg}\,x - \text{ctg}\,x + C}$$

## Практика

### Задача 1

$$\int \cos^3 x\,dx$$

Подсказка: $\cos^3 x = \cos^2 x \cdot \cos x = (1-\sin^2 x)\cos x$ — нечётная по cos x, замена $t = \sin x$.

<details>
<summary>Ответ</summary>

$t = \sin x$, $dt = \cos x\,dx$:

$$\int(1-t^2)\,dt = t - \frac{t^3}{3} + C = \sin x - \frac{\sin^3 x}{3} + C$$

</details>

---

### Задача 2

$$\int \sin^3 x\,dx$$

Подсказка: нечётная по sin x, замена $t = \cos x$.

<details>
<summary>Ответ</summary>

$\sin^3 x = \sin^2 x \cdot \sin x = (1-\cos^2 x)\sin x$

$t = \cos x$, $dt = -\sin x\,dx$:

$$-\int(1-t^2)\,dt = -t + \frac{t^3}{3} + C = -\cos x + \frac{\cos^3 x}{3} + C$$

</details>

---

### Задача 3

$$\int \frac{\sin^2 x}{\cos^2 x}\,dx = \int \text{tg}^2 x\,dx$$

Подсказка: чётная по обоим, используй $\text{tg}^2 x = \dfrac{1}{\cos^2 x} - 1$.

<details>
<summary>Ответ</summary>

$$\int\left(\frac{1}{\cos^2 x} - 1\right)dx = \text{tg}\,x - x + C$$

</details>
