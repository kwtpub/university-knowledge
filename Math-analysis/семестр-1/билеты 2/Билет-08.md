## Билет 8. Определение монотонной последовательности. Теорема Вейерштрасса о монотонной последовательности.

---

## Монотонная последовательность

Последовательность $\{x_n\}$ называется **монотонной**, если она только возрастает или только убывает.

Четыре вида:

| Вид | Условие | Пример |
|---|---|---|
| Возрастающая | $x_n < x_{n+1}$ | $1, 2, 3, 4, \ldots$ |
| Неубывающая | $x_n \leq x_{n+1}$ | $1, 1, 2, 2, 3, \ldots$ |
| Убывающая | $x_n > x_{n+1}$ | $4, 3, 2, 1, \ldots$ |
| Невозрастающая | $x_n \geq x_{n+1}$ | $3, 3, 2, 2, 1, \ldots$ |

> **Отличие возрастающей от неубывающей:** неубывающая допускает равенство соседних членов, возрастающая — нет.

---

## Теорема Вейерштрасса

### Формулировка

**Всякая монотонная ограниченная последовательность имеет конечный предел.**

Причём:
- Неубывающая ограниченная → $\lim x_n = \sup\{x_n\}$
- Невозрастающая ограниченная → $\lim x_n = \inf\{x_n\}$

**Бонус:** монотонная **неограниченная** → предел бесконечный:
- Неубывающая неограниченная → $\lim x_n = +\infty$
- Невозрастающая неограниченная → $\lim x_n = -\infty$

> **На пальцах:** если последовательность только растёт и при этом не уходит в бесконечность (ограничена сверху) — она обязана к чему-то прийти. Куда именно — к своей точной верхней грани.

### Пример 1

$x_n = 1 - \dfrac{1}{n} = 0, \dfrac{1}{2}, \dfrac{2}{3}, \dfrac{3}{4}, \ldots$

- Возрастающая ✓
- Ограничена сверху числом $1$ ✓
- По теореме Вейерштрасса предел существует: $\lim x_n = \sup\{x_n\} = 1$

### Пример 2

$x_n = n = 1, 2, 3, 4, \ldots$

- Возрастающая ✓
- Не ограничена ✓
- По теореме: $\lim x_n = +\infty$

## Практика

**Задание 1.** Является ли последовательность $x_n = \dfrac{n}{n+1}$ монотонной? Ограниченной? Найди предел.

> [!note]- Ответ
> **Монотонность:** $x_{n+1} - x_n = \dfrac{n+1}{n+2} - \dfrac{n}{n+1} = \dfrac{1}{(n+1)(n+2)} > 0$
>
> Значит возрастающая ✓
>
> **Ограниченность:** $0 < x_n < 1$ для всех $n$ ✓
>
> **Предел:** по теореме Вейерштрасса предел существует.
> $\lim_{n\to\infty} \dfrac{n}{n+1} = \lim_{n\to\infty} \dfrac{1}{1 + \frac{1}{n}} = 1$

---

**Задание 2.** Является ли $x_n = (-1)^n$ монотонной?

> [!note]- Ответ
> $x_1 = -1,\ x_2 = 1,\ x_3 = -1,\ x_4 = 1, \ldots$
>
> Не возрастает и не убывает — **не монотонная**.
>
> Поэтому теорема Вейерштрасса не применима, и предела нет.

---

**Задание 3.** $x_n = 2^n$ — монотонная? Ограниченная? Какой предел?

> [!note]- Ответ
> Возрастающая ✓
>
> Не ограничена (растёт до бесконечности) ✓
>
> $\lim_{n\to\infty} 2^n = +\infty$
