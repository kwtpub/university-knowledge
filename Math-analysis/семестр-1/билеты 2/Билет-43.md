[[]]## 43. Неопределённый интеграл от рациональной функции. Интегрирование простейших дробей.

Среди правильных рациональных дробей выделяют четыре типа, которые относят к простейшим рациональным дробям:

$$\frac{A}{x-a}, \quad \frac{A}{(x-a)^k}, \quad \frac{Ax+B}{x^2+px+q}, \quad \frac{Ax+B}{(x^2+px+q)^k},$$

где $k > 1$ — целое и $p^2 - 4q < 0$, т.е. квадратные уравнения не имеют действительных корней.

**Определение.** Элементарными называются дроби следующих четырёх типов:

- I. $\dfrac{1}{ax+b}$
- II. $\dfrac{1}{(ax+b)^m}$
- III. $\dfrac{Mx+N}{ax^2+bx+c}$
- IV. $\dfrac{Mx+N}{(ax^2+bx+c)^n}$

где $m, n$ — натуральные числа ($m \geq 2$, $n \geq 2$) и $b^2 - 4ac < 0$.

Первые два типа интегралов от элементарных дробей довольно просто приводятся к табличным подстановкой $t = ax + b$:

**I.**
$$\int \frac{dx}{ax+b} = \frac{1}{a}\int \frac{dt}{t} = \frac{1}{a}\ln|t| + C = \frac{1}{a}\ln|ax+b| + C.$$

**II.**
$$\int \frac{dx}{(ax+b)^m} = \frac{1}{a}\int \frac{dt}{t^m} = -\frac{1}{a(m-1)t^{m-1}} + C = -\frac{1}{a(m-1)(ax+b)^{m-1}} + C.$$

Интеграл дроби вида III может быть представлен в виде:

$$\int \frac{Ax+B}{x^2+px+q}\,dx = \int \frac{\dfrac{A}{2}(2x+p) + \left(B - \dfrac{Ap}{2}\right)}{x^2+px+q}\,dx = \frac{A}{2}\int \frac{2x+p}{x^2+px+q}\,dx + \left(B - \frac{Ap}{2}\right)\int \frac{dx}{x^2+px+q}$$

$$= \frac{A}{2}\ln|x^2+px+q| + \left(B - \frac{Ap}{2}\right)\int \frac{dx}{(x+p/2)^2 + (q - p^2/4)}$$

$$= \frac{A}{2}\ln|x^2+px+q| + \frac{2B - Ap}{\sqrt{4q-p^2}}\operatorname{arctg}\frac{2x+p}{\sqrt{4q-p^2}} + C.$$

**Пример для типа III.** $\displaystyle\int \frac{4x+1}{x^2+2x+5}\,dx$

**Идея:** числитель типа III всегда разбивается на две части — одна содержит производную знаменателя (даёт логарифм), вторая — просто константу (даёт арктангенс).

---

**Шаг 1. Находим производную знаменателя.**

Знаменатель: $x^2+2x+5$. Берём производную:

$$(x^2+2x+5)' = 2x+2.$$

Именно $2x+2$ нам нужно получить в числителе.

---

**Шаг 2. Разбиваем числитель $4x+1$ через $2x+2$.**

Хотим написать $4x+1$ в виде $\alpha \cdot (2x+2) + \beta$. Подбираем $\alpha$ и $\beta$:

$$\alpha \cdot (2x+2) + \beta = 2\alpha x + 2\alpha + \beta.$$

Приравниваем коэффициенты:
- при $x$: $2\alpha = 4 \Rightarrow \alpha = 2$,
- свободный член: $2\alpha + \beta = 1 \Rightarrow 4 + \beta = 1 \Rightarrow \beta = -3$.

Итого: $4x+1 = 2(2x+2) - 3$. Проверка: $2(2x+2)-3 = 4x+4-3 = 4x+1$ — верно.

---

**Шаг 3. Разбиваем интеграл на два.**

Подставляем $4x+1 = 2(2x+2)-3$ в числитель:

$$\int \frac{4x+1}{x^2+2x+5}\,dx = \int \frac{2(2x+2)-3}{x^2+2x+5}\,dx.$$

Одну дробь разбиваем на сумму двух дробей с тем же знаменателем:

$$= \int \left(\frac{2(2x+2)}{x^2+2x+5} - \frac{3}{x^2+2x+5}\right)dx.$$

По свойству интеграла сумму/разность можно разбить на два отдельных интеграла:

$$= \int \frac{2(2x+2)}{x^2+2x+5}\,dx - \int \frac{3}{x^2+2x+5}\,dx.$$

Константы $2$ и $3$ выносим за знак интеграла:

$$= 2\int \frac{2x+2}{x^2+2x+5}\,dx - 3\int \frac{dx}{x^2+2x+5}.$$

---

**Шаг 4. Берём первый интеграл.**

В числителе стоит производная знаменателя, поэтому используем формулу $\int \frac{f'}{f}\,dx = \ln|f| + C$:

$$2\int \frac{2x+2}{x^2+2x+5}\,dx = 2\ln|x^2+2x+5| + C.$$

---

**Шаг 5. Берём второй интеграл — выделяем полный квадрат.**

В знаменателе $x^2+2x+5$ нет действительных корней, поэтому выделяем полный квадрат:

$$x^2+2x+5 = (x^2+2x+1) + 4 = (x+1)^2 + 4.$$

Теперь используем табличный интеграл $\displaystyle\int \frac{dx}{u^2+a^2} = \frac{1}{a}\arctg\frac{u}{a} + C$, где $u = x+1$, $a = 2$:

$$-3\int \frac{dx}{(x+1)^2+4} = -3 \cdot \frac{1}{2}\arctg\frac{x+1}{2} + C = -\frac{3}{2}\arctg\frac{x+1}{2} + C.$$

---

**Шаг 6. Складываем результаты.**

$$\boxed{2\ln|x^2+2x+5| - \frac{3}{2}\arctg\frac{x+1}{2} + C.}$$

---

**Пример для типа IV.** $\displaystyle\int \frac{2x+3}{(x^2+2x+5)^2}\,dx$

**Идея та же что в типе III** — разбиваем числитель так, чтобы первая часть содержала производную знаменателя. Только теперь знаменатель в квадрате, поэтому первый интеграл даёт не логарифм, а степень, а второй считается через полный квадрат.

**Шаг 1.** Производная знаменателя: $(x^2+2x+5)' = 2x+2$.

**Шаг 2.** Разбиваем числитель $2x+3$ через $2x+2$:

$$2x+3 = 1\cdot(2x+2) + 1, \quad \text{то есть } \alpha=1,\ \beta=1.$$

**Шаг 3.** Разбиваем на два интеграла:

$$\int \frac{2x+3}{(x^2+2x+5)^2}\,dx = \int \frac{2x+2}{(x^2+2x+5)^2}\,dx + \int \frac{dx}{(x^2+2x+5)^2}.$$

**Шаг 4.** Первый интеграл — замена $t = x^2+2x+5$, $dt = (2x+2)\,dx$:

$$\int \frac{2x+2}{(x^2+2x+5)^2}\,dx = \int \frac{dt}{t^2} = \int t^{-2}\,dt = -\frac{1}{t} + C = -\frac{1}{x^2+2x+5} + C.$$

**Шаг 5.** Второй интеграл — выделяем полный квадрат $x^2+2x+5 = (x+1)^2+4$ и используем табличную формулу для степени $n=2$:

$$\int \frac{dx}{((x+1)^2+4)^2} = \frac{x+1}{2\cdot4\cdot((x+1)^2+4)} + \frac{1}{2\cdot4}\cdot\frac{1}{2}\arctg\frac{x+1}{2} + C = \frac{x+1}{8(x^2+2x+5)} + \frac{1}{16}\arctg\frac{x+1}{2} + C.$$

**Шаг 6.** Складываем:

$$\boxed{-\frac{1}{x^2+2x+5} + \frac{x+1}{8(x^2+2x+5)} + \frac{1}{16}\arctg\frac{x+1}{2} + C.}$$

> **На пальцах:** тип IV — это тип III, но знаменатель в степени $n$. Первая часть всегда даёт $-\frac{1}{(n-1)\cdot f^{n-1}}$, вторая считается рекуррентно или по формуле. Главное — уметь разбить числитель так же как в типе III.

---

**Пример для типа II.** $\displaystyle\int \frac{dx}{(3x+2)^4}$

Замена $t = 3x+2$, $dt = 3\,dx$, $dx = \tfrac{dt}{3}$:

$$\int \frac{dx}{(3x+2)^4} = \frac{1}{3}\int \frac{dt}{t^4} = \frac{1}{3}\int t^{-4}\,dt = \frac{1}{3} \cdot \frac{t^{-3}}{-3} + C = -\frac{1}{9t^3} + C = -\frac{1}{9(3x+2)^3} + C.$$

### Тип I — $\dfrac{1}{ax+b}$

**Задача 1.** $\displaystyle\int \frac{dx}{x+3}$

Тип I, $a=1$, $b=3$:

$$\int \frac{dx}{x+3} = \frac{1}{1}\ln|x+3| + C = \ln|x+3| + C.$$

**Задача 2.** $\displaystyle\int \frac{dx}{2x-5}$

Тип I, $a=2$, $b=-5$. Замена $t = 2x-5$, $dt = 2\,dx$, $dx = \tfrac{dt}{2}$:

$$\int \frac{dx}{2x-5} = \frac{1}{2}\int \frac{dt}{t} = \frac{1}{2}\ln|2x-5| + C.$$

### Тип III — $\dfrac{Ax+B}{x^2+px+q}$

**Задача 6.** $\displaystyle\int \frac{2x+1}{x^2+1}\,dx$

**Шаг 1.** Производная знаменателя: $(x^2+1)' = 2x$.

**Шаг 2.** Разбиваем числитель $2x+1 = 1\cdot(2x) + 1$, то есть $\alpha=1$, $\beta=1$.

**Шаг 3.** Разбиваем на два интеграла:

$$\int \frac{2x+1}{x^2+1}\,dx = \int \frac{2x}{x^2+1}\,dx + \int \frac{dx}{x^2+1}.$$

**Шаг 4.** Первый интеграл — формула $\int \frac{f'}{f}\,dx = \ln|f|$:

$$\int \frac{2x}{x^2+1}\,dx = \ln(x^2+1) + C \quad \text{(модуль не нужен, } x^2+1 > 0 \text{ всегда)}.$$

**Шаг 5.** Второй интеграл — табличный:

$$\int \frac{dx}{x^2+1} = \arctg x + C.$$

**Ответ:**

$$\boxed{\ln(x^2+1) + \arctg x + C.}$$
