# Практика 05.02 — замена переменной и по частям

**Коротко: замена переменной**
- Формула: $\int f(x)\,dx=\int f(\varphi(t))\varphi'(t)\,dt$.
- Выбирай $u=\varphi(x)$ так, чтобы $du$ «жил» рядом с подынтегральной функцией.
- Для тригонометрических подстановок полезно помнить: $x=a\tan t$, $x=a\sin t$, $x=a\sec t$.

**Примеры с разбором (замена)**
Пример 1. $\int 2x\cos(x^2)\,dx$. $u=x^2$, $du=2x\,dx$. Ответ: $\sin(x^2)+C$.

Пример 2. $\int \frac{2x+1}{x^2+x+1}\,dx$. $u=x^2+x+1$, $du=(2x+1)dx$. Ответ: $\ln(x^2+x+1)+C$.

Пример 3. $\int \sqrt{a^2-x^2}\,dx$. Подстановка $x=a\sin t$, $dx=a\cos t\,dt$. Ответ: $\frac{a^2}{2}\arcsin\frac{x}{a}+\frac{x}{2}\sqrt{a^2-x^2}+C$.

Пример 4. $\int \frac{x^2+a^2}{x^2}\,dx$. Можно взять $x=a\tan t$, а можно сразу упростить: $\int (1+a^2/x^2)\,dx=x-\frac{a^2}{x}+C$.

**Коротко: по частям**
- Формула: $\int u\,dv = uv-\int v\,du$.
- Идея: в правой части должен получаться более простой интеграл.
- Типичные пары: $u=\ln x$, $u=\arctan x$, $u=x^k$ вместе с $dv=\sin(ax)dx$, $e^{ax}dx$, $\cos(ax)dx$.

**Примеры с разбором (по частям)**
Пример 5. $\int x e^{-2x}\,dx$. Берем $u=x$, $dv=e^{-2x}dx$. Ответ: $-\frac12 x e^{-2x}-\frac14 e^{-2x}+C$.

Пример 6. $\int (x^2+2x)\cos 2x\,dx$. Два раза по частям. Ответ: $\frac12(x^2+2x)\sin 2x+\frac12(x+1)\cos 2x-\frac14\sin 2x+C$.

Пример 7. $\int x\arctan x\,dx$. $u=\arctan x$, $dv=x\,dx$. Ответ: $\frac{x^2}{2}\arctan x-\frac{x}{2}+\frac12\arctan x+C$.
