# Практика 05.03 — интегрирование по частям

**Цель:** отработать выбор $u, dv$ и уверенное применение формулы по частям.

**Главная формула**
$$
\int u\,dv = uv-\int v\,du
$$

**Как выбирать $u$ и $dv$ (кратко)**
1. Обычно в $u$ берем то, что упрощается при дифференцировании: $\ln x$, $\arctan x$, $\arcsin x$, многочлен.
2. В $dv$ берем то, что легко интегрируется: $e^{ax}dx$, $\sin(ax)dx$, $\cos(ax)dx$, $dx$.
3. Если после одного шага стало сложнее, поменяй выбор $u,dv$.

**Блок A. Один шаг по частям (база)**
1. [x] $\int x e^x\,dx$
2. [x] $\int x\sin x\,dx$
3. [x] $\int x\cos x\,dx$
4. [ ] $\int x\ln x\,dx$
5. [ ] $\int \ln x\,dx$
6. [ ] $\int x\arctan x\,dx$
7. [ ] $\int x e^{-2x}\,dx$
8. [ ] $\int x\sin(3x)\,dx$
9. [ ] $\int x\cos(4x)\,dx$
10. [ ] $\int x^2\ln x\,dx$

Ответы (Блок A, 1-3):
1. $\int x e^x\,dx = x e^x - e^x + C = e^x(x-1)+C$
2. $\int x\sin x\,dx = -x\cos x + \sin x + C$
3. $\int x\cos x\,dx = x\sin x + \cos x + C$

**Мини-алгоритм для каждого примера**
1. Явно запиши выбор: $u=\ldots,\ dv=\ldots$.
2. Отдельно посчитай $du$ и $v$.
3. Подставь в формулу по частям.
4. Проверь производной полученный ответ.
