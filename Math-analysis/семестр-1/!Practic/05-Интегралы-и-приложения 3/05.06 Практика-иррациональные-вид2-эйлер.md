# Практика 05.06 — иррациональные (вид 2, подстановки Эйлера)

**Коротко**
- Интегралы вида $\int R(x,\sqrt{ax^2+bx+c})\,dx$ сводятся к рациональным с помощью подстановок Эйлера.
- Варианты подстановки:
- Если $a>0$: $\sqrt{ax^2+bx+c}=\sqrt{a}\,x+t$, тогда $x=\frac{t^2-c}{b-2\sqrt{a}\,t}$.
- Если $c>0$: $\sqrt{ax^2+bx+c}=t x+\sqrt{c}$, тогда $x=\frac{2t\sqrt{c}-b}{a-t^2}$.
- Если $D=b^2-4ac>0$ и $x_1,x_2$ — корни: $\sqrt{ax^2+bx+c}=t(x-x_1)$, тогда $x=\frac{a x_2-t^2 x_1}{a-t^2}$.
- После подстановки интеграл становится рациональным по $t$.

**Примеры с разбором (схема)**
Пример 1. $\int \frac{dx}{\sqrt{x^2+4x+5}}$.
Сдвиг $x=u-2$ дает $\int \frac{du}{\sqrt{u^2+1}}$, далее можно применить Эйлера или стандартную формулу.

Пример 2. $\int \frac{x\,dx}{\sqrt{2x^2-3x+1}}$.
Берем подстановку Эйлера для $a>0$: $\sqrt{2x^2-3x+1}=\sqrt{2}x+t$ и сводим к рациональной функции.

Пример 3. $\int \sqrt{x^2-4x-5}\,dx$.
Используем подстановку Эйлера с корнем квадратного трехчлена.
