# Практика 05.01 — типы тригонометрических сложностей

**Ограничение этого листа:** без замены переменной и без интегрирования по частям.

**Карта выбора (без замены)**
1. Сначала упрости выражение:
- вынеси константы за знак интеграла;
- раскрывай степени и дроби тождествами.
2. Табличный вид с линейным аргументом:
- $\sin(ax+b),\ \cos(ax+b),\ \dfrac1{\cos^2(ax+b)},\ \dfrac1{\sin^2(ax+b)},\ \dfrac1{\sin(ax+b)},\ \dfrac1{\cos(ax+b)}$;
- сразу интегрируй по таблице, учитывая множитель $\dfrac1a$.
3. Степени $\sin x,\cos x$ (в этом листе берем четные степени):
- понижение степени:
  $\sin^2x=\dfrac{1-\cos2x}{2},\ \cos^2x=\dfrac{1+\cos2x}{2}$.
4. Степени $\tan x,\cot x$:
- $\tan^2x=\dfrac1{\cos^2x}-1$;
- $\cot^2x=\dfrac1{\sin^2x}-1$;
- своди задачу к табличным интегралам.
5. Произведения разных частот:
- $\sin \alpha x\sin \beta x,\ \cos \alpha x\cos \beta x,\ \sin \alpha x\cos \beta x$;
- переходи к сумме/разности формулами произведения в сумму.
6. Смешанные выражения:
- сначала приведи к виду из пунктов 2-5 (через тождества), потом интегрируй.

**Тип A. База (линейный аргумент)**
1. [x] $\int \sin(4x-1)\,dx$
2. [x] $\int \cos(7x+3)\,dx$
3. [x] $\int \dfrac{dx}{\cos^2(5x)}$
4. [x] $\int \dfrac{dx}{\sin^2(2x-1)}$
5. [ ] $\int \dfrac{dx}{\sin(3x)}$
6. [ ] $\int \dfrac{dx}{\cos(2x+\pi/6)}$

Ответы (Тип A, 1-4):
1. $\int \sin(4x-1)\,dx=-\dfrac14\cos(4x-1)+C$
2. $\int \cos(7x+3)\,dx=\dfrac17\sin(7x+3)+C$
3. $\int \dfrac{dx}{\cos^2(5x)}=\dfrac15\tan(5x)+C$
4. $\int \dfrac{dx}{\sin^2(2x-1)}=-\dfrac12\cot(2x-1)+C$

**Тип B. Четные степени $\sin/\cos$**
1. [x] $\int \sin^2 x\,dx$
2. [x] $\int \cos^2 x\,dx$
3. [ ] $\int \sin^2 x\cos^2 x\,dx$
4. [ ] $\int \sin^4 x\,dx$
5. [ ] $\int \cos^4 x\,dx$
6. [ ] $\int \sin^2(2x)\,dx$
7. [ ] $\int \cos^2(3x)\,dx$
8. [ ] $\int \sin^4(2x)\,dx$

Разбор B.1:
Используемые формулы:
1. Понижение степени: $\sin^2x=\dfrac{1-\cos2x}{2}$.
2. Понижение степени: $\cos^2x=\dfrac{1+\cos2x}{2}$.
3. Линейность: $\int(\alpha f+\beta g)\,dx=\alpha\int f\,dx+\beta\int g\,dx$.
4. Табличный интеграл с линейным аргументом: $\int \cos(ax+b)\,dx=\dfrac1a\sin(ax+b)+C$.

$$
\int \sin^2 x\,dx=\int \frac{1-\cos 2x}{2}\,dx
=\frac12\int dx-\frac12\int \cos 2x\,dx
=\frac x2-\frac{\sin 2x}{4}+C
$$
