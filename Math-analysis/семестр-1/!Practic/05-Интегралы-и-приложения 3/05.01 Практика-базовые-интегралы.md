# Практика 05.01 — базовые интегралы

**Коротко**
- Если $F'(x)=f(x)$, то $\int f(x)\,dx = F(x)+C$.
- Линейность: $\int( f\pm g)dx=\int fdx \pm \int gdx$, $\int a f dx = a\int f dx$.
- Линейный аргумент: если $\int f(u)du=F(u)$, то $\int f(ax+b)dx=\frac1a F(ax+b)+C$.
- Подведение под знак дифференциала: $\int \frac{f'(x)}{f(x)}dx=\ln|f(x)|+C$.

**Табличные интегралы (минимум, который нужен постоянно)**
- $\int u^\alpha du=\frac{u^{\alpha+1}}{\alpha+1}+C$ $(\alpha\ne-1)$.
- $\int \frac{du}{u}=\ln|u|+C$.
- $\int a^u du=\frac{a^u}{\ln a}+C$ $(a>0, a\ne1)$.
- $\int e^u du=e^u+C$.
- $\int \sin u\,du=-\cos u+C$, $\int \cos u\,du=\sin u+C$.
- $\int \frac{du}{a^2+u^2}=\frac1a\arctan\frac{u}{a}+C$ $(a\ne0)$.
- $\int \frac{du}{a^2-u^2}=\frac1{2a}\ln\left|\frac{a+u}{a-u}\right|+C$.
- $\int \frac{du}{u^2-a^2}=\frac1{2a}\ln\left|\frac{u-a}{u+a}\right|+C$.
- $\int \frac{du}{\sqrt{a^2-u^2}}=\arcsin\frac{u}{a}+C$.
- $\int \frac{du}{\cos^2 u}=\tan u+C$, $\int \frac{du}{\sin^2 u}=-\cot u+C$.
- $\int \frac{du}{\sin u}=\ln\left|\tan\frac{u}{2}\right|+C$.
- $\int \frac{du}{\cos u}=\ln\left|\tan\left(\frac{u}{2}+\frac\pi4\right)\right|+C$.

**Примеры с разбором**
Пример 1. $\int (4x^3-2x^2+2x^{-3}+1)\,dx$.
Решение: по линейности $=x^4-\frac23x^3-\frac1{x^2}+x+C$.

Пример 2. $\int 3^x e^{2x}\,dx$.
Решение: $\int (3e^2)^x\,dx=\frac{(3e^2)^x}{\ln(3e^2)}+C$.

Пример 3. $\int \cos(7x-3)\,dx$.
Решение: $\frac1{7}\sin(7x-3)+C$.

Пример 4. $\int \frac{2x}{x^2+5}\,dx$.
Решение: $\ln(x^2+5)+C$.
