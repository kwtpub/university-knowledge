# Практика 05.07 — тригонометрические подстановки

**Коротко**
- $\sqrt{a^2-x^2}$: $x=a\sin t$, $dx=a\cos t\,dt$.
- $\sqrt{a^2+x^2}$: $x=a\tan t$, $dx=a\sec^2 t\,dt$.
- $\sqrt{x^2-a^2}$: $x=a\sec t$, $dx=a\sec t\tan t\,dt$.
- После подстановки упрощай, используй $1+\tan^2 t=\sec^2 t$, $1-\sin^2 t=\cos^2 t$.

**Примеры с разбором**
Пример 1. $\int \sqrt{a^2-x^2}\,dx$. Подстановка $x=a\sin t$.
Ответ: $\frac{a^2}{2}\arcsin\frac{x}{a}+\frac{x}{2}\sqrt{a^2-x^2}+C$.

Пример 2. $\int \frac{x^2+a^2}{x^2}\,dx$. Подстановка $x=a\tan t$ приводит к рациональному интегралу; ответ: $x-\frac{a^2}{x}+C$.

Пример 3. $\int \frac{dx}{\sqrt{a^2+x^2}}$. Подстановка $x=a\tan t$ дает $\ln|x+\sqrt{a^2+x^2}|+C$.

Пример 4. $\int \frac{dx}{\sqrt{x^2-a^2}}$. Подстановка $x=a\sec t$ дает $\ln|x+\sqrt{x^2-a^2}|+C$.
