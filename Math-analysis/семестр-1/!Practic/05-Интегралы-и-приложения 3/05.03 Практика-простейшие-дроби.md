# Практика 05.03 — простейшие дроби

**Коротко**
- Простейшие дроби (по методичке):
- $\frac{A}{x-a}$.
- $\frac{A}{(x-a)^k}$, $k\ge2$.
- $\frac{Mx+N}{x^2+px+q}$ при $p^2-4q<0$.
- $\frac{Mx+N}{(x^2+px+q)^k}$, $k\ge2$, $p^2-4q<0$.

**Базовые формулы интегрирования**
- $\int \frac{A}{x-a}\,dx=A\ln|x-a|+C$.
- $\int \frac{A}{(x-a)^k}\,dx=\frac{A}{1-k}(x-a)^{1-k}+C$.
- Для $\int \frac{Mx+N}{x^2+px+q}\,dx$ представь $Mx+N=A(2x+p)+B$.
- Тогда $\int \frac{Mx+N}{x^2+px+q}\,dx=\frac{A}{2}\ln(x^2+px+q)+B\int \frac{dx}{x^2+px+q}$.
- Для последнего интеграла доведи квадрат: $x^2+px+q=(x+\frac p2)^2+\left(q-\frac{p^2}{4}\right)$.

**Примеры с разбором**
Пример 1. $\int \frac{3}{x-2}\,dx=3\ln|x-2|+C$.

Пример 2. $\int \frac{5}{(x+1)^2}\,dx=-\frac{5}{x+1}+C$.

Пример 3. $\int \frac{2x+1}{x^2+4x+5}\,dx$. Представим $2x+1=(2x+4)-3$. Ответ: $\ln(x^2+4x+5)-3\int \frac{dx}{(x+2)^2+1}$.

Пример 4. $\int \frac{x-1}{x^2+1}\,dx=\frac12\ln(x^2+1)-\arctan x+C$.
