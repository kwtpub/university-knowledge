# Практика 05.04 — разложение на дроби

**Коротко: алгоритм**
- Если дробь неправильная, сначала раздели многочлены: $\frac{Q_m}{P_n}=M_{m-n}+\frac{Q_l}{P_n}$.
- Разложи знаменатель $P_n$ на линейные и неприводимые квадратные множители.
- Запиши разложение по шаблону:
- Для $(x-a)^k$: $\frac{A_1}{x-a}+\frac{A_2}{(x-a)^2}+\dots+\frac{A_k}{(x-a)^k}$.
- Для $(x^2+px+q)^t$: $\frac{M_1x+N_1}{x^2+px+q}+\dots+\frac{M_tx+N_t}{(x^2+px+q)^t}$.
- Найди коэффициенты методом неопределенных коэффициентов или подстановкой частных значений.
- Проинтегрируй каждую простейшую дробь (см. 06.03).

**Примеры с разбором**
Пример 1. $\int \frac{2x-3}{x(x-1)(x-2)}\,dx$. Разложение: $\frac{A}{x}+\frac{B}{x-1}+\frac{C}{x-2}$, далее интегрируй каждую дробь.

Пример 2. $\int \frac{x^2+1}{(x-1)^2(x+2)}\,dx$. Разложение: $\frac{A}{x-1}+\frac{B}{(x-1)^2}+\frac{C}{x+2}$.

Пример 3. $\int \frac{3x+5}{(x^2+2x+5)^2}\,dx$. Разложение: $\frac{Mx+N}{x^2+2x+5}+\frac{Px+Q}{(x^2+2x+5)^2}$.
