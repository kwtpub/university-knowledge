# Практика 05.08 — биномиальные дифференциалы

**Коротко (подстановки Чебышева)**
Рассматривается интеграл $\int x^m (a+bx^n)^p\,dx$.
- Случай 1: $p\in\mathbb Z$. Тогда раскрываем скобки или делаем стандартные замены.
- Случай 2: $(m+1)/n\in\mathbb Z$. Подстановка $a+bx^n=u^s$, где $p=r/s$.
- Случай 3: $(m+1)/n+p\in\mathbb Z$. Подстановка $a+bx^n=u^s x^n$.
После подстановки получаем рациональный интеграл по $u$.

**Пример с разбором**
Пример. $\int \frac{dx}{x^7\sqrt{1+x^4}}$.
$m=-7$, $n=4$, $p=-1/2$, поэтому $(m+1)/n+p=-2$ — целое число. Берем $1+x^4=u^2 x^4$ и получаем рациональный интеграл по $u$.
