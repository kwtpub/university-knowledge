# Практика 03.06 — обратная и параметрическая функции

**Коротко**
- Для взаимно обратных функций:
  $$\Large (f^{-1})'(y_0)=\frac1{f'(x_0)},\quad y_0=f(x_0),\ f'(x_0)\ne0$$
- Если $x=x(t),\ y=y(t)$, то
  $$\Large \frac{dy}{dx}=\frac{dy/dt}{dx/dt},\quad \frac{dx}{dt}\ne0$$
- Вторая производная для параметрического задания:
  $$\Large \frac{d^2y}{dx^2}=\frac{\frac{d}{dt}(dy/dx)}{dx/dt}$$

**Примеры с разбором**
Пример 1.
$$\Large y=\arcsin x\Rightarrow y'=\frac1{\sqrt{1-x^2}}$$
(как производная обратной к $x=\sin y$).

Пример 2.
$$\Large x=t^2+1,\ y=t^3-2t$$
$$\Large \frac{dy}{dx}=\frac{3t^2-2}{2t}$$

Пример 3.
$$\Large x=\cos t,\ y=\sin t$$
$$\Large \frac{dy}{dx}=\frac{\cos t}{-\sin t}=-\cot t$$
