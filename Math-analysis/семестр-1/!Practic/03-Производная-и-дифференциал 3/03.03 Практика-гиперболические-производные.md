# Практика 03.03 — гиперболические производные (sinh, cosh, tgh, cth)

**Цель:** закрепить табличные производные гиперболических функций и цепное правило.

**Мини-таблица (что использовать в задачах)**
1. $(\sinh x)'=\cosh x$
2. $(\cosh x)'=\sinh x$
3. $(\tanh x)'=\dfrac{1}{\cosh^2 x}$
4. $(\coth x)'=-\dfrac{1}{\sinh^2 x}$
5. Для линейного аргумента $u=ax+b$:
$$
(\sinh u)'=\cosh u\cdot u',\quad
(\cosh u)'=\sinh u\cdot u',\quad
(\tanh u)'=\dfrac{u'}{\cosh^2 u},\quad
(\coth u)'=-\dfrac{u'}{\sinh^2 u}.
$$

> Обозначения: $\tanh x = \operatorname{tgh}x$, $\coth x = \operatorname{cth}x$.

**Дополнительные примеры (база + линейный аргумент)**
1. [x] $(\sinh(2x-1))'$
2. [x] $(\cosh(3x+2))'$
3. [x] $(\tanh(4x))'$
4. [x] $(\coth(5x-1))'$
5. [ ] $(\sinh(x/2))'$
6. [ ] $(\cosh(7-2x))'$
7. [ ] $(\tanh(6x+3))'$
8. [ ] $(\coth(9x))'$

**Ответы к отмеченным [x]**
1. $(\sinh(2x-1))'=2\cosh(2x-1)$
2. $(\cosh(3x+2))'=3\sinh(3x+2)$
3. $(\tanh(4x))'=\dfrac{4}{\cosh^2(4x)}$
4. $(\coth(5x-1))'=-\dfrac{5}{\sinh^2(5x-1)}$

**Разбор 4**
$$
f(x)=\coth(5x-1),\quad u=5x-1,\quad f=\coth u
$$
$$
f'(x)=\left(-\frac{1}{\sinh^2 u}\right)\cdot u'
=-\frac{1}{\sinh^2(5x-1)}\cdot5
=-\frac{5}{\sinh^2(5x-1)}.
$$
