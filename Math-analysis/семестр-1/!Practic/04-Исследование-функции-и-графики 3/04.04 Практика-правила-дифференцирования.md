# Практика 04.04 — правила дифференцирования

**Коротко**
- $(u\pm v)'=u'\pm v'$
- $(uv)'=u'v+uv'$
- $\left(\frac{u}{v}\right)'=\frac{u'v-uv'}{v^2}$, $v\ne0$
- Для композиции: $(f(g(x)))'=f'(g(x))\,g'(x)$

**Примеры с разбором**
Пример 1.
$$\Large y=(x^2+1)(x^3-2x)$$
$$\Large y'=2x(x^3-2x)+(x^2+1)(3x^2-2)$$

Пример 2.
$$\Large y=\frac{x^2+3x-1}{x-2}$$
$$\Large y'=\frac{(2x+3)(x-2)-(x^2+3x-1)}{(x-2)^2}$$

Пример 3.
$$\Large y=\sqrt{1+x^2}$$
$$\Large y'=\frac{x}{\sqrt{1+x^2}}$$
