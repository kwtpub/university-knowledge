# Практика 02.05 — пределы на бесконечности

**Коротко**
- Для рациональных функций сравниваем старшие степени числителя и знаменателя.
- Если степени равны, предел — отношение старших коэффициентов.
- Если степень числителя меньше — предел 0, если больше — бесконечность (с учетом знаков).
- Для корней используем вынос старшей степени или умножение на сопряжённое.

**Примеры с разбором**
Пример 1 /Large.
$$\Large \lim\limits_{x\to +\infty}\frac{3x^2-5x+1}{2x^2+7}$$
Ответ: $\\frac{3}{2}$.

Пример 2 /Large.
$$\Large \lim\limits_{x\to +\infty}\left(\sqrt{x^2+3x}-x\right)$$
Домножаем на сопряжённое: $\frac{3x}{\sqrt{x^2+3x}+x}\to\frac{3}{2}$.

Пример 3 /Large.
$$\Large \lim\limits_{x\to +\infty}\frac{\sqrt{x^2+4x+1}}{x}$$
Ответ: $1$.
