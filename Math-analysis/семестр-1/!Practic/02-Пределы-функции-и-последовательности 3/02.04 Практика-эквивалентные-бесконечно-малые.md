# Практика 02.04 — эквивалентные бесконечно малые

**Коротко (таблица из методички)**
При $u\to 0$ эквивалентны:
- $\sin u\sim u$
- $\arcsin u\sim u$
- $\tan u\sim u$
- $\arctan u\sim u$
- $\ln(1+u)\sim u$
- $e^u-1\sim u$
- $(1+u)^m-1\sim mu$

**Полезное следствие**
- $1-\cos u=2\sin^2\frac{u}{2}\sim \frac{u^2}{2}$.

**Примеры с разбором**
Пример 1 /Large.
$$\Large \lim\limits_{x\to 0}\frac{\ln(1+3x)}{\sin(2x)}$$
Ответ: $\\frac32$.

Пример 2 /Large.
$$\Large \lim\limits_{x\to 0}\frac{e^{5x}-1}{\tan(2x)}$$
Ответ: $\\frac52$.

Пример 3 /Large.
$$\Large \lim\limits_{x\to 0}\frac{1-\cos(4x)}{x^2}$$
Ответ: $8$.
