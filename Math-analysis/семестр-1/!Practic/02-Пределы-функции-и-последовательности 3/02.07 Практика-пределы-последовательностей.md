# Практика 02.07 — пределы последовательностей

**Коротко**
- Если $|q|<1$, то $q^n\to0$ при $n\to +\infty$.
- Для рациональных выражений в $n$ сравниваем старшие степени.
- Если числитель растет медленнее знаменателя, предел 0.

**Примеры с разбором**
Пример 1 /Large.
$$\Large \lim\limits_{n\to +\infty}\frac{2n^3-3n}{5n^3+1}$$
Ответ: $\\frac25$.

Пример 2 /Large.
$$\Large \lim\limits_{n\to +\infty}\left(1+\frac{1}{n}\right)^n$$
Ответ: $e$.

Пример 3 /Large.
$$\Large \lim\limits_{n\to +\infty}\frac{(-1)^n}{n}$$
Ответ: $0$.
