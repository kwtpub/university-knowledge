# Практика 02.02 — алгебраические неопределенности

**Коротко**
- Типичные неопределенности: $0/0$, $\infty/\infty$, $\infty-\infty$, $0\cdot\infty$.
- При $0/0$: разложение на множители, сокращение, общий знаменатель.
- При $\infty-\infty$: приводим к дроби, рационализируем.
- При корнях: умножаем на сопряжённое.

**Примеры с разбором**
Пример 1 /Large.
$$\Large \lim\limits_{x\to 1}\frac{x^2-1}{x-1}$$
Сокращаем: $\frac{(x-1)(x+1)}{x-1}\\to 2$.

Пример 2 /Large.
$$\Large \lim\limits_{x\to 0}\frac{\sqrt{1+x}-1}{x}$$
Умножаем на сопряжённое: $\frac{\sqrt{1+x}-1}{x}\cdot\frac{\sqrt{1+x}+1}{\sqrt{1+x}+1}=\frac{1}{\sqrt{1+x}+1}\to\frac12$.

Пример 3 /Large.
$$\Large \lim\limits_{x\to \infty}\frac{2x^3-3x}{4x^3+1}$$
Ответ: $\\frac12$.
