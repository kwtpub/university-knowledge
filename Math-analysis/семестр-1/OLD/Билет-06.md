# Билет 6. Предел последовательности. Свойства сходящихся последовательностей. Ограниченные последовательности. Теорема о связи между сходящимися и ограниченными последовательностями

# Определения

- Предел последовательности: ^seq-limit
  - Последовательность $\{x_n\}$ имеет предел $a$, если для любого $\varepsilon > 0$ существует $N$, что для всех $n \ge N$ выполняется $|x_n - a| < \varepsilon$.
  - Обозначения: $x_n \to a$ при $n \to \infty$, $\lim_{n\to\infty} x_n = a$.
  - Пример: $x_n = 1/n \to 0$.

- Сходящаяся последовательность: ^seq-convergent
  - Последовательность называется сходящейся, если существует конечный предел $a$.
  - Тогда говорят, что $x_n$ сходится к $a$.
  - Пример: $x_n = 2 + 1/n$ сходится к $2$.

- Расходящаяся последовательность: ^seq-divergent
  - Последовательность называется расходящейся, если конечного предела не существует.
  - Пример: $x_n = (-1)^n$ расходится.

- Ограниченная последовательность: ^seq-bounded
  - Последовательность $\{x_n\}$ ограничена, если существует $M > 0$ такое, что $|x_n| \le M$ для всех $n$.
  - Ограниченность сверху: $\exists M: x_n \le M$ для всех $n$.
  - Ограниченность снизу: $\exists m: x_n \ge m$ для всех $n$.
  - Пример: $x_n = \sin n$ ограничена, так как $|x_n| \le 1$.

# Теоремы и свойства

## Свойство 1: Единственность предела ^seq-unique-limit

**Теорема.** Если последовательность $\{x_n\}$ сходится, то её предел единственен.

**Доказательство:**
Предположим противное: пусть $x_n \to a$ и $x_n \to b$, причем $a \ne b$.

Возьмем $\varepsilon = \frac{|a - b|}{2} > 0$.

По определению предела:
- Существует $N_1$: при $n > N_1$ выполнено $|x_n - a| < \varepsilon$
- Существует $N_2$: при $n > N_2$ выполнено $|x_n - b| < \varepsilon$

Пусть $N = \max\{N_1, N_2\}$. При $n > N$:
$$|a - b| = |a - x_n + x_n - b| \le |a - x_n| + |x_n - b| < \varepsilon + \varepsilon = 2\varepsilon = |a - b|$$

Получили противоречие: $|a - b| < |a - b|$. Значит, предел единственен. ∎

---

## Свойство 2: Ограниченность сходящейся последовательности ^convergent-implies-bounded

**Теорема.** Если последовательность $\{x_n\}$ сходится, то она ограничена.

**Доказательство:**
Пусть $\lim_{n \to \infty} x_n = a$. Возьмем $\varepsilon = 1$.

По определению предела существует $N$ такое, что при $n > N$ выполнено:
$$|x_n - a| < 1$$

Это означает:
$$a - 1 < x_n < a + 1 \quad \text{для всех } n > N$$

Следовательно:
$$|x_n| < |a| + 1 \quad \text{для всех } n > N$$

Для остальных членов (при $n \le N$) последовательность конечна:
$$\{x_1, x_2, \ldots, x_N\}$$

Положим:
$$C = \max\{|x_1|, |x_2|, \ldots, |x_N|, |a| + 1\}$$

Тогда $|x_n| \le C$ для всех $n \in \mathbb{N}$, т.е. последовательность ограничена. ∎

**Важно:** Обратное неверно! Ограниченная последовательность может расходиться.
- Пример: $x_n = (-1)^n$ ограничена ($|x_n| = 1$), но не имеет предела.

## Свойство 3: Сохранение знака ^seq-sign-preservation

**Теорема.** Если $\lim_{n \to \infty} x_n = a$ и $a > 0$ (или $a < 0$), то существует номер $N$ такой, что для всех $n > N$ выполнено $x_n > 0$ (соответственно $x_n < 0$).

**Доказательство:**
Рассмотрим случай $a > 0$. Возьмем $\varepsilon = \frac{a}{2} > 0$.

По определению предела существует $N$ такое, что при $n > N$:
$$|x_n - a| < \frac{a}{2}$$

Это неравенство эквивалентно:
$$-\frac{a}{2} < x_n - a < \frac{a}{2}$$

Прибавляя $a$ ко всем частям:
$$\frac{a}{2} < x_n < \frac{3a}{2}$$

Значит, $x_n > \frac{a}{2} > 0$ при $n > N$. ∎

**Следствие.** Если $\lim_{n \to \infty} x_n = a \ne 0$, то начиная с некоторого номера все члены последовательности имеют тот же знак, что и $a$.

## Свойство 4: Арифметические операции с пределами ^seq-arithmetic-operations

**Теорема.** Если $\lim_{n \to \infty} x_n = a$ и $\lim_{n \to \infty} y_n = b$, то:

1. **Сумма:** $\lim_{n \to \infty} (x_n + y_n) = a + b$
2. **Разность:** $\lim_{n \to \infty} (x_n - y_n) = a - b$
3. **Произведение:** $\lim_{n \to \infty} (x_n \cdot y_n) = a \cdot b$
4. **Частное:** $\lim_{n \to \infty} \frac{x_n}{y_n} = \frac{a}{b}$ (при $b \ne 0$)
5. **Умножение на константу:** $\lim_{n \to \infty} (C \cdot x_n) = C \cdot a$

### Доказательство суммы:

Возьмем произвольное $\varepsilon > 0$. По определению предела:
- Существует $N_1$: при $n > N_1$ выполнено $|x_n - a| < \frac{\varepsilon}{2}$
- Существует $N_2$: при $n > N_2$ выполнено $|y_n - b| < \frac{\varepsilon}{2}$

Пусть $N = \max\{N_1, N_2\}$. Тогда при $n > N$:
$$|(x_n + y_n) - (a + b)| = |(x_n - a) + (y_n - b)| \le |x_n - a| + |y_n - b| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon$$

Следовательно, $\lim_{n \to \infty} (x_n + y_n) = a + b$. ∎

### Доказательство произведения:

Рассмотрим разность:
$$|x_n y_n - ab| = |x_n y_n - x_n b + x_n b - ab| = |x_n(y_n - b) + b(x_n - a)|$$

По неравенству треугольника:
$$|x_n y_n - ab| \le |x_n| \cdot |y_n - b| + |b| \cdot |x_n - a|$$

Так как $\{x_n\}$ сходится, она ограничена: $|x_n| \le M$ для некоторого $M > 0$.

Возьмем произвольное $\varepsilon > 0$. Существуют $N_1$ и $N_2$ такие, что:
- При $n > N_1$: $|x_n - a| < \frac{\varepsilon}{2(|b| + 1)}$
- При $n > N_2$: $|y_n - b| < \frac{\varepsilon}{2M}$

Пусть $N = \max\{N_1, N_2\}$. При $n > N$:
$$|x_n y_n - ab| \le M \cdot \frac{\varepsilon}{2M} + |b| \cdot \frac{\varepsilon}{2(|b| + 1)} < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon$$

Следовательно, $\lim_{n \to \infty} (x_n \cdot y_n) = a \cdot b$. ∎

### Доказательство частного:

Достаточно доказать, что если $\lim_{n \to \infty} y_n = b \ne 0$, то $\lim_{n \to \infty} \frac{1}{y_n} = \frac{1}{b}$.

По теореме о сохранении знака, существует $N_0$: при $n > N_0$ имеем $|y_n| > \frac{|b|}{2}$.

Оценим разность:
$$\left|\frac{1}{y_n} - \frac{1}{b}\right| = \left|\frac{b - y_n}{y_n \cdot b}\right| = \frac{|y_n - b|}{|y_n| \cdot |b|} < \frac{|y_n - b|}{\frac{|b|}{2} \cdot |b|} = \frac{2|y_n - b|}{b^2}$$

Для $\varepsilon > 0$ существует $N_1$: при $n > N_1$ выполнено $|y_n - b| < \frac{b^2 \varepsilon}{2}$.

Пусть $N = \max\{N_0, N_1\}$. При $n > N$:
$$\left|\frac{1}{y_n} - \frac{1}{b}\right| < \frac{2}{b^2} \cdot \frac{b^2 \varepsilon}{2} = \varepsilon$$

Следовательно, $\lim_{n \to \infty} \frac{1}{y_n} = \frac{1}{b}$. По теореме о произведении:
$$\lim_{n \to \infty} \frac{x_n}{y_n} = \lim_{n \to \infty} \left(x_n \cdot \frac{1}{y_n}\right) = a \cdot \frac{1}{b} = \frac{a}{b}$$ ∎

## Примеры применения

**Пример 1.** $x_n = \frac{1}{n}$, $y_n = 1 + \frac{1}{n}$
- Сумма: $x_n + y_n = 1 + \frac{2}{n} \to 1 + 0 = 1$
- Произведение: $x_n y_n = \frac{1}{n}(1 + \frac{1}{n}) \to 0 \cdot 1 = 0$
- Частное: $\frac{x_n}{y_n} = \frac{1/n}{1 + 1/n} = \frac{1}{n+1} \to 0$

**Пример 2.** $x_n = -2 + \frac{1}{n} \to -2 < 0$
- По теореме о сохранении знака, начиная с некоторого $N$, все $x_n < 0$
