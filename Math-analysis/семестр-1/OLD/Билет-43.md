# Билет 43. Неопределенный интеграл от рациональной функции. Интегрирование простейших дробей

# Определения

- Рациональная функция: ^rational
  - $f(x)=\frac{P(x)}{Q(x)}$, где $P,Q$ — многочлены, $Q\ne 0$.

- Простейшие дроби: ^simple-fractions
  - $\frac{A}{x-a}$, $\frac{A}{(x-a)^k}$.
  - $\frac{Ax+B}{x^2+px+q}$, $\frac{Ax+B}{(x^2+px+q)^k}$ при неприводимом квадратном знаменателе.

---

# Теоремы и свойства

- Интеграл рациональной функции сводится к сумме интегралов простейших дробей после разложения на простейшие дроби.

---

# Формулы и правила

- $\int \frac{A}{x-a}\,dx=A\ln|x-a|+C$.
- $\int \frac{A}{(x-a)^k}\,dx=\frac{A}{1-k}(x-a)^{1-k}+C$, $k\ne 1$.
- $\int \frac{Ax+B}{x^2+px+q}\,dx=\frac{A}{2}\ln(x^2+px+q)+\text{(арктангенс)}$.

---

# Методы и алгоритмы

- Привести рациональную дробь к правильной.
- Разложить на простейшие дроби.
- Интегрировать каждую простейшую дробь отдельно.

---

# Доказательства (краткие идеи)

- Использовать разложение рациональной функции на простейшие дроби.

---

# Примеры

## Случай 1: $\dfrac{A}{x-a}$

$$\int \frac{3}{x-2}\,dx = 3\ln|x-2|+C$$

## Случай 2: $\dfrac{A}{(x-a)^k}$, $k \geq 2$

$$\int \frac{5}{(x-1)^3}\,dx = \frac{5}{1-3}(x-1)^{1-3}+C = -\frac{5}{2(x-1)^2}+C$$

## Случай 3: $\dfrac{Ax+B}{x^2+px+q}$ (неприводимый знаменатель)

Пример: $\displaystyle\int \frac{2x+3}{x^2+2x+5}\,dx$

Знаменатель $x^2+2x+5=(x+1)^2+4$ — неприводим ($D=4-20<0$).

Разбиваем числитель: $2x+3 = A(2x+2) + B$, откуда $A=1,\ B=1$.

$$\int \frac{2x+3}{x^2+2x+5}\,dx = \int \frac{2x+2}{x^2+2x+5}\,dx + \int \frac{1}{(x+1)^2+4}\,dx$$

$$= \ln(x^2+2x+5) + \frac{1}{2}\arctan\frac{x+1}{2}+C$$

## Случай 4: $\dfrac{Ax+B}{(x^2+px+q)^k}$, $k \geq 2$

Пример: $\displaystyle\int \frac{x+1}{(x^2+1)^2}\,dx$

Разбиваем: $x+1 = \frac{1}{2}(2x) + 1$.

$$\int \frac{x+1}{(x^2+1)^2}\,dx = \frac{1}{2}\int \frac{2x}{(x^2+1)^2}\,dx + \int \frac{1}{(x^2+1)^2}\,dx$$

Первый интеграл — подстановка $t=x^2+1$:

$$\frac{1}{2}\int \frac{dt}{t^2} = -\frac{1}{2t} = -\frac{1}{2(x^2+1)}$$

Второй — рекуррентная формула (или подстановка $x=\tan\theta$):

$$\int \frac{dx}{(x^2+1)^2} = \frac{x}{2(x^2+1)}+\frac{1}{2}\arctan x+C$$

**Итог:**

$$\int \frac{x+1}{(x^2+1)^2}\,dx = -\frac{1}{2(x^2+1)} + \frac{x}{2(x^2+1)}+\frac{1}{2}\arctan x+C$$

---

# Практика

- Найди $\int \frac{3}{x+1}\,dx$.
- Найди $\int \frac{2}{(x-3)^3}\,dx$.
