# Билет 59. Приложение интегрального исчисления к геометрии. Объем фигуры. Объем фигур вращения

# Определения

- Объем тела: ^volume
  - Объем вычисляется как предел сумм элементарных объёмов.

- Тело вращения: ^solid-of-revolution
  - Получается вращением плоской фигуры вокруг оси.

---

# Теоремы и свойства

- Метод дисков/шайб:
  - Вращение вокруг оси `Ox`: `V = π ∫_a^b (R(x)^2 − r(x)^2) dx`.

- Метод цилиндрических оболочек:
  - Вращение вокруг `Oy`: `V = 2π ∫_a^b x f(x) dx` (для области под `f`).

---

# Формулы и правила

- Диски/шайбы: `V = π ∫ (R^2 − r^2) dx`.
- Оболочки: `V = 2π ∫ x f(x) dx`.

---

# Методы и алгоритмы

- Определи ось вращения.
- Выбери метод (диски/шайбы или оболочки).
- Определи радиусы и пределы интегрирования.

---

# Доказательства (краткие идеи)

- Выводится из сумм Римана для объёмов дисков/колец.

---

# Примеры

- Объём конуса радиуса `R` и высоты `h`:
  - Вращение `y = (R/h)x`, `x∈[0,h]` вокруг `Ox`:
  - `V = π ∫_0^h (R x / h)^2 dx = (1/3)πR^2h`.

---

# Практика

- Найди объём тела вращения:
  - области под `y=x` на `[0,1]` вокруг `Ox`.

## Ответы
- **Ответ 1.** `V = π ∫_0^1 x^2 dx = π/3`. ^answer-01
