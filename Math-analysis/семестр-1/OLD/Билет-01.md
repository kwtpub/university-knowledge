# Билет 1. Множества. Основные операции над множествами. Метрические и арифметические пространства

**Множество и базовые понятия**
Множество — совокупность объектов (элементов), рассматриваемых как единое целое.  
Принадлежность: `a ∈ A`, не принадлежит: `a ∉ A`.  
Подмножество: `A ⊆ B`, если каждый элемент `A` принадлежит `B`.  
Собственное подмножество: `A ⊂ B`, если `A ⊆ B` и `A ≠ B`.  
Равенство множеств: `A = B` ⇔ `A ⊆ B` и `B ⊆ A`.  
Пустое множество: `∅`. Универсальное множество: `U` (если фиксировано).

**Декартово произведение**
`A × B = { (a, b) | a ∈ A, b ∈ B }`.

**Основные операции над множествами**
Объединение: `A ∪ B = { x | x ∈ A или x ∈ B }`.  
Пересечение: `A ∩ B = { x | x ∈ A и x ∈ B }`.  
Разность: `A \ B = { x | x ∈ A и x ∉ B }`.  
Дополнение относительно `U`: `A^c = U \ A`.  
Симметричная разность: `A △ B = (A \ B) ∪ (B \ A)`.

**Свойства операций**
Коммутативность: `A ∪ B = B ∪ A`, `A ∩ B = B ∩ A`.  
Ассоциативность: `(A ∪ B) ∪ C = A ∪ (B ∪ C)`, `(A ∩ B) ∩ C = A ∩ (B ∩ C)`.  
Дистрибутивность: `A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)`, `A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)`.  
Идемпотентность: `A ∪ A = A`, `A ∩ A = A`.  
Поглощение: `A ∪ (A ∩ B) = A`, `A ∩ (A ∪ B) = A`.  
Нейтральные/поглощающие элементы: `A ∪ ∅ = A`, `A ∩ ∅ = ∅`, `A ∪ U = U`, `A ∩ U = A`.  
Законы де Моргана: `(A ∪ B)^c = A^c ∩ B^c`, `(A ∩ B)^c = A^c ∪ B^c`.  
Разность через дополнение: `A \ B = A ∩ B^c`.

**Метрические пространства**
Метрическое пространство — пара `(X, ρ)`, где `X` — множество, `ρ: X × X → R` — метрика (расстояние), удовлетворяющая:
1. `ρ(x, y) ≥ 0` (неотрицательность).
2. `ρ(x, y) = 0 ⇔ x = y` (тождество неразличимых).
3. `ρ(x, y) = ρ(y, x)` (симметричность).
4. `ρ(x, y) ≤ ρ(x, z) + ρ(z, y)` (неравенство треугольника).

Графически: ε‑окрестность (открытый шар) точки `x0` — все точки на расстоянии `< ε` от `x0`.

**Арифметические пространства**
Арифметическое пространство обычно понимают как `R^n` — множество упорядоченных `n`‑ок вещественных чисел.  
Операции задаются покоординатно:
`(x1,...,xn) + (y1,...,yn) = (x1 + y1,...,xn + yn)`,  
`α(x1,...,xn) = (αx1,...,αxn)`.

**Числовые множества**
`N` — натуральные числа, `Z` — целые, `Q` — рациональные, `R` — действительные, `I` — иррациональные.  
`Q = { p/q | p ∈ Z, q ∈ Z, q ≠ 0 }`, `I = R \ Q`.  
Связи: `R = Q ∪ I`, `Q ∩ I = ∅`.
