# Билет 23. Обратные тригонометрические и гиперболические функции

# Определения

- Обратные тригонометрические функции: ^inverse-trig
  - `arcsin x`: область `[-1,1]`, значения `[-\pi/2, \pi/2]`.
  - `arccos x`: область `[-1,1]`, значения `[0, \pi]`.
  - `arctan x`: область `R`, значения `(-\pi/2, \pi/2)`.

- Обратные гиперболические функции: ^inverse-hyperbolic
  - `arsinh x = \ln(x+\sqrt{x^2+1})`.
  - `arcosh x = \ln(x+\sqrt{x^2-1})`, `x≥1`.
  - `artanh x = (1/2)\ln((1+x)/(1-x))`, `|x|<1`.

---

# Теоремы и свойства

- Основные тождества: ^inverse-trig-identities
  - `sin(arcsin x)=x`, `cos(arccos x)=x`.
  - `arcsin(sin x)=x` для `x∈[-\pi/2,\pi/2]` (аналогично для `arccos`).

- Свойства и монотонность: ^inverse-trig-properties
  - `arcsin` и `arctan` возрастают, `arccos` убывает на своих областях.
  - Все обратные тригонометрические функции непрерывны на своих областях.

---

# Формулы и правила

- `\arcsin x + \arccos x = \pi/2`
- `\tanh^{-1} x = (1/2)\ln\frac{1+x}{1-x}`

---

# Методы и алгоритмы

- Работа с областями значений:
  - Всегда проверять, что аргумент лежит в области определения обратной функции.

---

# Доказательства (краткие идеи)

- Свойства монотонности:
  - Следуют из монотонности исходных функций на выбранных промежутках.

---

# Примеры

- arcsin(1/2):
  - Равно `\pi/6`.

- arccos(0):
  - Равно `\pi/2`.

- arsinh(0):
  - Равно `0`.

---

# Практика

- Вычисли `arctan(1)` и `arcsin(-1)`.
- Определи область определения `arcosh x` и `artanh x`.
