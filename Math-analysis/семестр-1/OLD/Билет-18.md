# Билет 18. Определение сложной функции. Предел сложной функции. Непрерывность сложной функции. Гиперболические функции

# Определения

- Сложная функция (композиция): ^composition
  - Если `g: X→Y` и `f: Y→Z`, то `(f∘g)(x)=f(g(x))`.

- Гиперболические функции: ^hyperbolic-def
  - `sinh x = (e^x - e^{-x})/2`, `cosh x = (e^x + e^{-x})/2`.
  - `tanh x = sinh x / cosh x`.

---

# Теоремы и свойства

- Предел сложной функции: ^limit-composition
  - Если `g(x)→b` и `lim_{y→b} f(y)=L`, то `lim f(g(x))=L`.

- Непрерывность композиции: ^cont-composition
  - Если `g` непрерывна в `a` и `f` непрерывна в `g(a)`, то `f∘g` непрерывна в `a`.

- Свойства гиперболических функций: ^hyperbolic-properties
  - `cosh^2 x - sinh^2 x = 1`.
  - `cosh` — чётная, `sinh` — нечётная, обе непрерывны на `R`.

---

# Формулы и правила

- `\sinh x = (e^x - e^{-x})/2`
- `\cosh x = (e^x + e^{-x})/2`
- `\cosh^2 x - \sinh^2 x = 1`

---

# Методы и алгоритмы

- Композиция пределов:
  - Сначала найти предел внутренней функции `g(x)`.
  - Затем применить предел/непрерывность внешней `f`.

- Проверка области определения:
  - Убедиться, что `g(x)` попадает в область определения `f`.

---

# Доказательства (краткие идеи)

- Предел композиции:
  - Использовать определение предела и заменить `y=g(x)`.

---

# Примеры

- sin(g(x)):
  - Если `g(x)→0`, то `lim sin(g(x))/g(x)=1`.

- cosh(0):
  - `cosh(0)=1`, `sinh(0)=0`.

---

# Практика

- Найди `lim_{x→0} sin(\sqrt{x})/\sqrt{x}`.
- Проверь непрерывность `f(g(x))`, где `g(x)=x^2`, `f(y)=\ln(1+y)`.
