# Билет 41. Определение первообразной. Основные свойства первообразной. Таблица простейших интегралов

# Определения

- Первообразная: ^antiderivative
  - Функция $F(x)$ называется **первообразной** функции $f(x)$ на промежутке $I$, если для всех $x \in I$:
    $$F'(x) = f(x).$$
  - Словами: первообразная — это такая функция, производная которой равна подынтегральной функции.
  - Пример: $F(x) = x^2$ — первообразная для $f(x) = 2x$, потому что $(x^2)' = 2x$.

- Неопределённый интеграл: ^indef-integral
  - **Неопределённый интеграл** функции $f(x)$ — это семейство всех первообразных:
    $$\int f(x)\,dx = F(x) + C,$$
    где $F(x)$ — любая конкретная первообразная, $C \in \mathbb{R}$ — произвольная константа.
  - Словами: интеграл это не одна функция, а целое семейство — все они отличаются на константу.

---

# Теоремы и свойства

- Единственность с точностью до константы: ^antiderivative-uniqueness
  - Если $F$ и $G$ — две первообразные одной и той же функции $f$ на промежутке, то:
    $$G(x) = F(x) + C$$
    для некоторой константы $C$.
  - Доказательство: $(G - F)' = f - f = 0$, значит $G - F = \text{const}$.

- Линейность: ^antiderivative-linearity
  - $$\int \bigl(\alpha f(x) + \beta g(x)\bigr)\,dx = \alpha \int f(x)\,dx + \beta \int g(x)\,dx$$
  - Следует из линейности производной: $(\alpha F + \beta G)' = \alpha f + \beta g$.

- Связь дифференцирования и интегрирования: ^diff-integral-connection
  - $$\left(\int f(x)\,dx\right)' = f(x)$$
  - $$\int F'(x)\,dx = F(x) + C$$
  - Словами: дифференцирование и интегрирование — обратные операции (с точностью до константы).

- Инвариантность формы: ^invariance
  - Если $u = \varphi(x)$ — дифференцируемая функция, то:
    $$\int f(u)\,du = F(u) + C$$
    — формула та же, что и для $\int f(x)\,dx$. Это позволяет делать замену переменной.

# Таблица простейших интегралов

$$\int x^n\,dx = \frac{x^{n+1}}{n+1} + C, \quad n \neq -1$$

$$\int \frac{1}{x}\,dx = \ln|x| + C$$

$$\int e^x\,dx = e^x + C$$

$$\int a^x\,dx = \frac{a^x}{\ln a} + C, \quad a > 0,\ a \neq 1$$

$$\int \sin x\,dx = -\cos x + C$$

$$\int \cos x\,dx = \sin x + C$$

$$\int \frac{dx}{\cos^2 x} = \tan x + C$$

$$\int \frac{dx}{\sin^2 x} = -\cot x + C$$

$$\int \frac{dx}{1 + x^2} = \arctan x + C$$

$$\int \frac{dx}{\sqrt{1 - x^2}} = \arcsin x + C$$

$$\int \frac{dx}{\sqrt{x^2 \pm a^2}} = \ln\left|x + \sqrt{x^2 \pm a^2}\right| + C$$

$$\int \frac{dx}{x^2 - a^2} = \frac{1}{2a}\ln\left|\frac{x-a}{x+a}\right| + C$$

$$\int \frac{dx}{x^2 + a^2} = \frac{1}{a}\arctan\frac{x}{a} + C$$

$$\int \frac{dx}{\sqrt{a^2 - x^2}} = \arcsin\frac{x}{a} + C$$

---

# Доказательства (краткие идеи)

- Единственность с точностью до константы:
  - Пусть $F' = G' = f$. Тогда $(G - F)' = 0$ на промежутке. Функция с нулевой производной на промежутке — константа. Значит $G - F = C$.

- Линейность:
  - $(\alpha F + \beta G)' = \alpha F' + \beta G' = \alpha f + \beta g$. Значит $\alpha F + \beta G$ — первообразная для $\alpha f + \beta g$.

# Примеры

- $\displaystyle\int (3x^2 - 2x + 5)\,dx = x^3 - x^2 + 5x + C$

- $\displaystyle\int \frac{1}{\sqrt{x}}\,dx = \int x^{-1/2}\,dx = 2\sqrt{x} + C$

- $\displaystyle\int 5e^x\,dx = 5e^x + C$

- $\displaystyle\int (\sin x + \cos x)\,dx = -\cos x + \sin x + C$

# Практика

- Найди $\displaystyle\int (5x^4 - 3)\,dx$.
- Найди $\displaystyle\int \frac{1}{1+x^2}\,dx$.
- Найди $\displaystyle\int \left(\sqrt{x} + \frac{1}{x}\right)dx$.
- Проверь дифференцированием: $\displaystyle\int \frac{1}{\cos^2 x}\,dx = \tan x + C$.
