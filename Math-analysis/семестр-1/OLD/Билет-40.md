# Билет 40. Схема построения графика функции

# Определения

- Схема исследования функции: ^graphing-scheme
  - Последовательность шагов для полного анализа функции и построения её графика.

---

# Теоремы и свойства

- Результаты исследований из предыдущих билетов используются совместно:
  - Монотонность и экстремумы — по `f'`.
  - Выпуклость и перегибы — по `f''`.
  - Асимптоты — по пределам.

---

# Формулы и правила

- Полезные пределы для асимптот:
  - `k = lim f(x)/x`, `b = lim (f(x) − kx)`.

---

# Методы и алгоритмы

- Схема построения графика:
  1) Область определения.
  2) Симметрии (чётность/нечётность, периодичность).
  3) Нули и пересечения с осями.
  4) Точки разрыва и асимптоты.
  5) Производная `f'`: критические точки, интервалы монотонности, экстремумы.
  6) Вторая производная `f''`: выпуклость/вогнутость, точки перегиба.
  7) Набросок графика с учётом всех данных.

---

# Доказательства (краткие идеи)

- Обычно не требуются: схема — это алгоритм применения известных теорем.

---

# Примеры

- `f(x)=x^3-3x`:
  - `D(f)=R`, функция нечётная.
  - `f'(x)=3x^2-3`, критические точки `x=±1`.
  - `f''(x)=6x`, перегиб в `x=0`.
  - Набросок: локальный максимум при `x=-1`, минимум при `x=1`.

---

# Практика

- Построй схематично график:
  - `f(x) = x^2/(x^2+1)`
  - `g(x) = (x^2-4)/x`

## Ответы
- **Ответ 1.** `D=R`, `f(0)=0`, `0 ≤ f(x) < 1`, горизонтальная асимптота `y=1`, минимум в `x=0`. ^answer-01
- **Ответ 2.** `D=R \\ {0}`, вертикальная асимптота `x=0`, наклонная асимптота `y=x`, нули `x=±2`. ^answer-02
