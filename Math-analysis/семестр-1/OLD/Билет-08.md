# Билет 8. Определение монотонной последовательности. Теорема Вейерштрасса о монотонной последовательности

# Определения

- Монотонная последовательность: ^monotone
  - Последовательность называется монотонной, если она либо неубывает, либо невозрастает.
  - Пример: $x_n = 1/n$ монотонно убывает.

- Неубывающая (монотонно возрастающая): ^monotone-increasing
  - $x_{n+1} \ge x_n$ для всех $n$.
  - Строго возрастающая: $x_{n+1} > x_n$.
  - Пример: $x_n = n$ строго возрастает.

- Невозрастающая (монотонно убывающая): ^monotone-decreasing
  - $x_{n+1} \le x_n$ для всех $n$.
  - Строго убывающая: $x_{n+1} < x_n$.
  - Пример: $x_n = 1/n$ строго убывает.

---

# Теоремы и свойства

- Теорема Вейерштрасса о монотонной последовательности: ^weierstrass-monotone
  - Всякая монотонная и ограниченная последовательность сходится.
  - Если $\{x_n\}$ неубывает и ограничена сверху, то $\lim x_n = \sup\{x_n\}$.
  - Если $\{x_n\}$ невозрастает и ограничена снизу, то $\lim x_n = \inf\{x_n\}$.
  - Пример: $x_n = 1 - 1/n$ неубывает и ограничена сверху $1$, поэтому $x_n \to 1$.

- Монотонная и неограниченная: ^monotone-unbounded
  - Если $\{x_n\}$ неубывает и не ограничена сверху, то $x_n \to +\infty$.
  - Если $\{x_n\}$ невозрастает и не ограничена снизу, то $x_n \to -\infty$.
  - Пример: $x_n = n \to +\infty$, $x_n = -n \to -\infty$.

---

# Формулы и правила

- Если $\{x_n\}$ неубывает и ограничена сверху, то $\lim x_n = \sup\{x_n\}$.
- Если $\{x_n\}$ невозрастает и ограничена снизу, то $\lim x_n = \inf\{x_n\}$.

---

# Методы и алгоритмы

- Проверка сходимости монотонной последовательности:
  - Доказать монотонность: показать, что $x_{n+1} - x_n \ge 0$ (неубывание) или $x_{n+1} - x_n \le 0$ (невозрастание).
  - Пример: для $x_n = (n-1)/n$ имеем $x_{n+1}-x_n = 1/(n(n+1)) \ge 0$.
  - Доказать ограниченность сверху/снизу.
  - Заключить сходимость и найти предел через $\sup$ или $\inf$.

- Поиск предела:
  - Если $x_n$ задана рекуррентно и монотонна, найти $L$ как решение уравнения $L = f(L)$.

---

# Доказательства (краткие идеи)

- Монотонная и ограниченная:
  - Для неубывающей последовательности $\{x_n\}$ верхняя грань $s = \sup\{x_n\}$.
  - По определению супремума для любого $\varepsilon > 0$ найдется $n$, что $s - \varepsilon < x_n \le s$.
  - Из монотонности следует $x_k \ge x_n$ при $k \ge n$, значит $s - \varepsilon < x_k \le s$.
  - Следовательно, $x_n \to s$.

---

# Примеры

- $x_n = 1 - 1/n$:
  - Последовательность неубывает и ограничена сверху числом $1$, значит $x_n \to 1$.

- $x_n = 1/n$:
  - Невозрастает, ограничена снизу нулем, значит $x_n \to 0$.

- $x_n = n$:
  - Неубывает и не ограничена сверху, значит $x_n \to +\infty$.

---

# Практика

- Проверь монотонность и найди предел: $x_n = (n-1)/n$, $x_n = (n+1)/n$.
- Докажи сходимость последовательности $x_n = \sqrt{n+1} - \sqrt{n}$.
- Исследуй $x_{n+1} = \sqrt{2 + x_n}$, $x_1 = 1$ на монотонность и предел.
