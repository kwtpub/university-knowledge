# Билет 42. Интегрирование путем замены переменной и формула интегрирования по частям в неопределенном интеграле

# Определения

- Замена переменной: ^substitution
  - Если $u=g(x)$, то $du=g'(x)dx$ и
    $\int f(g(x))g'(x)\,dx=\int f(u)\,du$.

- Интегрирование по частям: ^parts
  - Основано на формуле $d(uv)=u\,dv+v\,du$.

---

# Теоремы и свойства

- Формула замены переменной: ^substitution-formula
  - $\int f(g(x))g'(x)\,dx=\int f(u)\,du$.

- Формула интегрирования по частям: ^parts-formula
  - $\int u\,dv=u v-\int v\,du$.

---

# Формулы и правила

- $\int f(g(x))g'(x)\,dx=\int f(u)\,du$.
- $\int u\,dv=u v-\int v\,du$.

---

# Методы и алгоритмы

- Замена:
  - Подобрать $u=g(x)$ так, чтобы $g'(x)dx$ присутствовал.
  - Переписать интеграл через $u$.

- По частям:
  - Выбрать $u$ так, чтобы его производная проще.
  - Выбрать $dv$ так, чтобы интеграл $v$ считался.

---

# Доказательства (краткие идеи)

- Замена переменной следует из цепного правила.
- Формула по частям следует из производной произведения.

---

# Примеры

- Замена: $\int 2x\cos(x^2)\,dx=\sin(x^2)+C$.
- По частям: $\int x e^x\,dx=(x-1)e^x+C$.

---

# Практика

- Найди $\int 3x^2\sin(x^3)\,dx$.
- Найди $\int x\ln x\,dx$.

---

# Графическое представление

- Замена переменной соответствует «растяжению» оси интегрирования.
